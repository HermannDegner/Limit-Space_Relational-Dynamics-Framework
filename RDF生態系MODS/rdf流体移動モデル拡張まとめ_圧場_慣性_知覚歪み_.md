# RDF流体移動モデル拡張まとめ（圧場・慣性・知覚歪み）

---

## ■ コア命題

エージェントは意思で動くのではなく、

「複雑かつ動的に変化する素流圧の勾配に従い、
過去の流れ（整合慣性）を引きずりながら流れ続ける構造」

である。

---

## ■ スケール階層とボトムアップ構築

基本式はスケールを問わず同一である。
複雑さの違いは圧の種類・センサー精度・慣性の層の追加として現れる。

| スケール | 追加される要素 | 圧の種類 |
|---|---|---|
| 単細胞 | 圧場への直接反応のみ | 化学勾配・毒素 |
| 多細胞 | 内部分業（センサー専門・移動専門） | 光・接触・温度 |
| 虫 | 簡単な記憶・フェロモン | 社会圧の萌芽 |
| 魚・鳥 | 群れ圧・予測圧の萌芽 | 個体認識 |
| 哺乳類 | 感情（熱の精緻化）・個体認識 | 関係圧 |
| 人間 | 言語・概念・仮想圧場（想像・計画） | 社会圧・意味圧 |
| 社会 | 集合的整合慣性・制度・文化 | 規範圧・物語圧 |

どのスケールで止めても**そのスケールの生命として動く**。
そして友人関係も単細胞の凝集本能も、
同じ構造の異なるスケールのSILNとして読める。

### ■ 実装指針

```
単純エージェント → 圧場反応のみ
↓ センサー追加   → 世界の解像度が上がる
↓ 慣性追加       → 過去の流れが残る
↓ 記憶追加       → M の学習が始まる
↓ 社会圧追加     → 他個体が圧場に入る
↓ 予測圧追加     → 内部目標が生まれる
```

各レイヤーは独立してテスト可能。

## ■ 基本運動モデル

```
V(t+1) = M · V(t) + Σ w_x F_x + ξ
```

- F_x：素流圧（環境要因）
- w_x：重み（基層構造）
- M：整合慣性（方向依存）
- ξ：揺らぎ

---

## ■ 圧場としての世界

世界は固定された空間ではなく、
素流圧の分布による動的勾配場である。

### ■ 基本圧

- 食料：引力圧
- 危険：反発圧
- 地形：コスト圧
- 未知：不確実性圧

---

## ■ 勾配の非定常性

勾配は常に変化する。

### 1. 微小揺らぎ
```
ξ(t)
```

### 2. 連続変化
```
F(t)
```

### 3. 跳躍（構造再編）
```
M → M'
```

---

## ■ センサーと知覚

```
F_agent = Sensor(F_world)
```

センサーは圧場の射影であり、
個体ごとに世界の見え方が異なる。

---

## ■ 知覚歪み（幻覚）

```
F_agent = Sensor_biased(F_world) + F_fake
```

または

```
F_agent = g(H, A, M) ∘ Sensor(F_world)
```

- H：熱（不安・興奮）
- A：attention（注意）

### ■ 本質

幻覚とは「存在しないものを見る」ことではなく、

**圧場の知覚が歪むこと**

---

## ■ attention（注意）の役割

```
F_agent = A(t) ∘ Sensor(F_world)
```

- H上昇 → Aが一点集中
- 他圧の検知低下

→ 視野狭窄・突進・見落とし

---

## ■ 慣性（整合慣性）

```
V(t+1) = M · V(t) + ...
```

### ■ 特徴

- 方向依存
- 状態依存

### ■ 例

- 恐怖 → 直進慣性↑
- 疲労 → 慣性↓
- 習熟 → 特定方向慣性↑

---

## ■ 典型現象

### ■ 透明ガラス衝突

- 障害圧が弱い
- 慣性が勝つ

→ 激突

---

### ■ 低い柵を越えない

- 越境コスト圧
- 不確実性圧
- モード切替コスト

→ 平時では回避

---

### ■ 社会的境界（他人の敷地）

```
F_social + F_risk + F_boundary
```

物理的境界ではなく、

**認識による反発圧場**

---

## ■ 状態依存変化

### ■ 平時

- 社会圧↑
- リスク回避

### ■ 緊急時

- 危険圧↑
- 他圧相対低下

→ 境界突破

---

## ■ 行動の再定義

- 行動 = 圧場 + 慣性 + 揺らぎ
- 意思 = 後付け解釈

---

## ■ 本質まとめ

エージェントは

「揺らぎ、歪み、再編される圧場の中で、
過去の流れを保持しながら整合方向へ流れ続ける構造」

この構造は単細胞から人間・社会まで共通である。
スケールが変わっても基本式は変わらない。
変わるのは圧の種類とセンサーと慣性の層の厚さだけである。

---

## ■ 応用方向

- 群れ圧（他個体圧）
- 社会圧の動的変化
- エネルギー制約
- 記憶（M）の学習
- 予測圧（未来圧）

---

## ■ 最終定義

移動とは

「動的に変化する素流圧場と整合慣性の相互作用の中で、
構造が連続的に更新される過程」

である。
