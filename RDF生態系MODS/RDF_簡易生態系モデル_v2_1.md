## 1. 概要

本ドキュメントは、関係力学言語（RDF）および移動・思考・流れ統合考察に基づき、
生態系NPCエージェントが「関係性勾配場」の中でいかに流動するかを定義するものである。

### v1.0 からの根本的更新

v1.0 の設計思想：
> NPCは熱を管理する存在である

v2.0 の設計思想：
> NPCは勾配場の中を流れる構造である
> 熱はその流動を歪める変数である

この変更は単なる言い換えではない。
v1.0では「熱が閾値を超えたとき行動が変わる」という受動的な記述だった。
v2.0では「勾配が構造を引き・押し・流す」という能動的な場の記述になる。

移動・思考・行動は別のシステムではなく、
**関係性勾配の中を構造が流れる現象の異なる表現**である。

## 2. 設計思想

### 2.1 コア方程式

```
V(t+1) = M · V(t) + Σ w_x F_x + ξ
```

- `V(t)` ：構造の現在状態ベクトル
- `M`    ：整合慣性（SILN）行列 ← **流れが戻っていく土台基準**
- `F_x`  ：各素流圧（食料・危険・音・匂い・群れ等）
- `w_x`  ：各素流圧への感度重み（個体ごとに異なる）
- `ξ`    ：揺らぎ

この式はスケールを問わず成立する。
ウサギが逃げるのも、キツネが追跡するのも、
勾配場の中を構造が流れているという一つの現象である。

### 2.2 Mは土台基準である（v2.1 明確化）

`M` は「構造が戻っていく先」を定義する土台基準であり、
バネの自然長に相当する。

**跳躍で変わるのはMではなく：**

```
w_x   ： どの素流圧にどれだけ感応するか（感度と符号）
θ     ： 次の跳躍の起きやすさ
F_pred： 内部予測勾配のパターン
```

「M が変わる」という旧来の記述は不正確であった。
Mに対する感度と閾値が変わる、が正確な記述である。

### 2.3 Mの時間スケール（v2.1 追記）

Mは不変ではなく、**変わるが非常に遅い**。

| 時間スケール | 変わるもの |
|---|---|
| 瞬時 | F_pred, H, ξ |
| 事象単位（跳躍） | w_x, θ |
| 反復・習慣・道具 | M_local |
| 世代・文化・進化 | M_base |

NPC設計において「跳躍でMが変わる」と記述する場合は、
`w_x` と `θ` の変化を指すものとして扱う。

### 2.4 勾配の型

```
引力勾配 (+) ： 食料・安全・群れ・獲物・交配相手
反発勾配 (-) ： 捕食者・危険地形・未知・高熱個体
コスト勾配   ： 地形・移動距離・エネルギー消費
```

エージェントの移動・行動とは、
**これらの合成勾配に従った流動**として記述される。

### 2.5 熱は勾配を歪める

勾配は固定ではない。

```
∇F_agent = g(H, A, M) ∘ Sensor(F_world)
```

- `H` が低い → 勾配を安定・正確に受け取る
- `H` が中程度 → 揺らぎが増し、探索的流動が起きやすい
- `H` が高い → 勾配が歪み・増幅され、跳躍が起きる

**熱は行動の引き金ではなく、勾配の見え方を変える変数である。**

### 2.6 世界観の前提

NPCが受け取るのは客観的な勾配場ではなく、
センサー・熱・整合慣性によって変形された局所的な勾配場である。

これは限界空間的前提と整合する：
観測は常に作用を介してしか成立しない。

## 3. 環境構造（勾配場としての世界）

世界は座標系ではなく、**引力・反発・コストが重なり合う勾配場**として設計される。

### 3.1 地形（コスト勾配）

地形はエネルギーコストとして勾配に乗る。

| 地形 | 移動コスト | 視線 | 音・匂い伝播 |
|---|---|---|---|
| 岩・崖 | 高 | 遮断 | 変形・吸収 |
| 平地 | 低 | 開放 | 減衰小 |
| 森・長草 | 中 | 遮断 | 乱反射 |
| 水場 | 高（侵入） | 開放 | 増幅 |

構造はコスト勾配の低い方向へ自然に流れる。
**地形設計＝勾配場の地形設計である。**

### 3.2 資源（引力勾配）

資源は引力として機能するが、その強さは個体の内部状態に依存する。

```
F_food の引力強度 ∝ 個体の飢餓状態
飽食状態では F_food はほぼゼロになる
```

- **短い草** ：採餌引力は強い・露出によりリスク反発も同時発生
- **長い草** ：採餌引力は弱い・遮蔽により安心引力が追加される

同じ資源でも、個体の状態と位置によって勾配の合成値が変わる。

### 3.3 風（勾配場のベクトルバイアス）

風は勾配場全体に方向性バイアスをかける。

```
風上 → 匂い・音の引力/反発強度が増す（情報の濃縮）
風下 → 匂い・音の勾配が流失（観測の死角化）
```

風の導入により、勾配場は時々刻々と再構成される動的場になる。
エージェントにとって「風上を読む」こと自体が勾配最適化行動になる。

## 4. 観測システム（勾配知覚の動態）

センサーは環境の勾配場を個体の知覚勾配場に変換する装置である。

```
F_agent = g(H, A, M) ∘ Sensor(F_world)
```

センサーごとに変換特性が異なり、それが行動多様性を生む。

### 4.1 匂い（過去勾配の残響）

```
RDF的定義：過去の整合慣性が空間に残した痕跡
```

- 時間とともに減衰する引力/反発の場
- 「今ここ」の確定情報ではなく、過去の構造の痕跡
- **追跡とは過去勾配の上流を辿る流動である**

```
匂い勾配の方向 → 対象の移動方向の推定
匂い勾配の強さ → 対象がそこにいた「最近さ」の指標
```

### 4.2 音（瞬発的勾配パルス）

```
RDF的定義：素流圧の急激なパルス → 勾配の急激な書き換え
```

- 突発的に反発勾配が最大化される
- 周囲のエージェントに伝播し、群れ全体の勾配場を同期書き換えする

```
音量 → 反発強度
周波数（低）→ 大スケール・遠距離の危険勾配
周波数（高）→ 局所・近距離の危険勾配
増加 → 接近勾配（反発が強まる方向）
減少 → 離脱勾配（危険が遠ざかるシグナル）
```

### 4.3 視覚（高精度勾配確定）

```
RDF的定義：局所的な高解像度の勾配確定装置
```

- 見えているものは勾配が「確定」する
- **見えている捕食者は反発勾配だが、位置が読める分だけ勾配の安定性がある**
- 見えない捕食者は仮想反発勾配が膨張し、最悪ケースへの熱蓄積が起きる

```
視認あり → 勾配確定 → 予測可能な流動
視認なし → 勾配不確定 → 仮想勾配が膨張 → 熱上昇
```

### 4.4 勾配歪みと熱の関係

熱が高いとき、知覚された勾配は歪む：

```
低熱 ： F_agent ≈ Sensor(F_world)  （客観的知覚）
中熱 ： F_agent に揺らぎが乗る     （探索的知覚）
高熱 ： F_agent が増幅・偏向する   （パニック状態）
```

パニック中のウサギが意味のない方向へ走るのは、
勾配歪みによる誤った流動として自然に説明される。

## 5. エージェント構造

### 5.1 基本変数

各個体 $i$ は以下を持つ：

```
V_i(t)      ： 現在状態ベクトル（位置・速度・内部状態）
M_i         ： 整合慣性（SILN）行列 ← 土台基準・原則不変
w_i[x]      ： 各素流圧への感度重み（符号を含む）← 跳躍で更新される
H_i(t)      ： 現在熱
θ_i         ： 跳躍閾値 ← 跳躍で更新される
F_pred_i(t) ： 内部予測勾配（仮想勾配場）← 跳躍で更新される
```

### 5.2 勾配重みの符号と個体差

`w_i[x]` の符号が個体差の本質である。

```
w[捕食者] < 0    ： 反発（全個体共通・基層）
w[食料]   > 0    ： 引力（飢餓状態に依存）
w[未知]         ： 個体によって符号が異なる　← ここが性格を生む
```

- ウサギ：`w[未知] < 0`（未知は反発）
- キツネ：`w[未知] > 0`（未知は誤差解決の引力＝ドーパミン効果）

### 5.3 内部予測勾配（仮想勾配）

エージェントは現実の勾配だけでなく、**内部で生成した予測勾配**によっても流動する。

```
F_total = F_world_perceived + F_pred
```

- `F_pred` は過去の整合パターンから生成される
- **見えていない獲物の匂いをたどるキツネ**は、`F_pred` で仮想的な獲物位置を生成し
  その勾配に向けて流れている

```
F_imagination が強くなりすぎると：
F_imagination >> F_world → 現実の勾配が見えなくなる
→ 誤った追跡・誤った逃走が起きる
```

これは長時間の隔離や過剰ストレスによる異常行動と同じ構造で読める。

## 6. 個体設計（勾配プロファイル）

### 6.1 ウサギ（Rabbit）― 高感度・高反発構造

**勾配プロファイル**

```
w[捕食者]  ： 非常に大きな反発
w[音パルス]： 急激な反発増幅
w[未知]    ： 反発寄り（負の符号）
w[群れ]    ： 強い引力（群れ同期が整合慣性を安定させる）
```

**基層構造（神経物質近似）**

```
ノルアドレナリン α1 ： 極大（危険兆候への感度最大）
セロトニン 5-HT1    ： 極小（誤差許容度が低い）
```

**勾配動態**

| 状況 | 勾配の変化 |
|---|---|
| 音パルス検知 | 反発勾配が瞬間最大化 → 即座の流動 |
| 視認不能の匂い | 仮想反発勾配が膨張 → 熱蓄積 |
| 群れ内安全 | 全体の勾配が収束 → 熱低下 |
| 長時間安全継続 | 勾配フラット → 退屈熱発生 → 探索流動 |

**退屈と探索**

```
誤差がない → 勾配がほぼフラット → 流れる方向が決まらない
→ 基層構造が仮想熱を発生 → 探索勾配が内部生成される
→ ウサギは「なんとなく」移動を始める
```

これは勾配フラットを基層が危険信号として読む機構である。

**状態遷移（勾配閾値ベース）**

```
Rest      → Graze   ： 退屈熱 > 閾値 → 内部食料引力が顕在化
Graze     → Alert   ： 反発勾配 > 整合慣性
Alert     → Escape  ： 反発勾配 ≫ 整合慣性（勾配が整合慣性を突き破る）
Escape    → Rest    ： 反発勾配がゼロ近傍まで低下 + 熱散逸
```

### 6.2 キツネ（Fox）― 誤差追跡・勾配逆用構造

**勾配プロファイル**

```
w[獲物匂い]  ： 正の引力（履歴追跡）
w[未知]      ： 正（ドーパミン効果で反発を引力に変換）
w[コスト]    ： 誤差解決期待値が上回ると符号が実質反転
w[露出リスク]： 一定以下に抑える（隠密制約）
```

**基層構造（神経物質近似）**

```
ドーパミン D1  ： 高（誤差探索・引力変換）
セロトニン     ： 中〜高（誤差許容度が適切）
オキシトシン   ： 対象ロック時に上昇（関係重み増加）
```

**ドーパミンによる勾配反転**

```
通常 ：   F_unknown → 反発圧（避ける）
介入後：  F_unknown → 引力圧（近づく）

w_unknown の符号が正に反転している
コスト圧はそのまま存在している
誤差解決への期待値がコスト圧を上回るため、勾配に逆らう流れが可能になる
```

**追跡の勾配動態**

```
匂い検知 → 過去勾配の起点を取得
→ F_pred（仮想獲物位置）を内部生成
→ 現実勾配 + 仮想勾配の合成場に沿って流れる
→ 視覚で獲物確定 → F_pred から F_world に切り替え
→ 対象ロック（オキシトシン的重み上昇）
→ w[獲物] が上昇し、勾配への感度がさらに増す
```

**隠密とは「自分の勾配を環境に書かない」操作**

```
キツネが自分を見せないのは、
相手の素流圧に自分の情報を乗せないための操作である
→ ウサギ側で仮想勾配が生成できない
→ ウサギの反発勾配がゼロに留まる
→ キツネは引力勾配の方向だけを使える
```

**状態遷移**

```
Wander → Track  ： 匂い勾配の引力検知
Track  → Stalk  ： 仮想勾配が収束し対象位置が絞れてきた
Stalk  → Chase  ： 視覚確定（F_pred → F_world 切替）
Chase  → Feed   ： 成功捕食（最大の熱解消・勾配消滅）
Chase  → Wander ： 対象ロスト（仮想勾配が崩壊）
```

### 6.3 ハヤブサ（Falcon）― メタ勾配・系全体の整合回復装置

**役割の再定義**

```
ハヤブサは生態系全体の「勾配場の歪み」を検知し
強制的なリセット勾配を注入する機構である
```

**メタ勾配の発生条件**

```
過密        → 特定領域の勾配場が飽和
資源枯渇    → 引力勾配が全域で低下
停滞        → 勾配フラット化が広域に拡大
```

これらは生態系SILNが閾値を超えた状態である。
ハヤブサはその跳躍機構として機能する。

**ハヤブサが作る勾配**

```
回避不能な速度 → エージェントの反発勾配計算が成立しない
→ 局所線形近似が崩壊
→ 群れの勾配場が一斉リセット
→ 新たな勾配分布が成立する
```

これは生態系スケールの跳躍である。
個体スケールでは悲劇だが、系スケールでは構造再編である。

## 7. 相互作用の力学（勾配場の干渉）

### 7.1 視認可能性と勾配の安定性

```
露出した捕食者
→ ウサギ側の反発勾配が「確定」する
→ 局所線形近似が成立
→ 熱は上がるが方向は定まる → 整合的逃走

隠れた捕食者
→ ウサギ側の反発勾配の「源泉が不明」
→ 仮想反発勾配が最悪ケースに向けて膨張
→ 熱が持続的に蓄積 → 勾配歪みが増大
```

知覚可能か否かは、勾配の**確定性の違い**である。
SILNが成立するかどうかの違いと整合する。

### 7.2 警告の伝播（群れの勾配同期）

```
一頭が高熱 + 音パルスを放つ
→ 周囲の w[音] が高い個体（ウサギ）に
  反発勾配が一斉書き込みされる
→ 群れ全体の勾配場が同期更新
→ 一括して同方向の流動（逃走）が起きる
```

これは群れの相転移を勾配場の同期書き換えとして読んだものである。

**群れが分散している場合**：
音の減衰により、距離に応じて勾配書き換えの強度が落ちる
→ 同期が不完全になり、群れが分断されることがある

### 7.3 捕食者の「負の素流圧戦略」

```
キツネの隠密
= 自分の勾配情報を環境に書かない
= ウサギ側の仮想反発勾配の生成を防ぐ

F_pred（ウサギの仮想勾配） ≈ 0  → パニックが起きない
F_world（実際の近接） → 視認時に初めて勾配が確定
                       → 反発勾配が最大値で突然立ち上がる
                       → 対処時間がない
```

## 8. 数理更新（v2.1 統合版）

### 8.1 勾配統合

各時刻の合成勾配ベクトルを構成する：

```
F_total(t) = F_world_perceived(t) + F_pred(t)

F_world_perceived(t) = g(H_i, A_i, M_i) ∘ Sensor(F_world(t))

F_pred(t) : 過去の整合パターンから内部生成された仮想勾配
```

### 8.2 状態更新（コア方程式）

```
V(t+1) = M · V(t) + Σ w_x F_x(t) + ξ(t)
```

### 8.3 誤差と熱

```
E(t)      = || F_total(t) - M · V(t) ||      // 誤差（予測との乖離）
σ(t)      = || E(t) ||²                      // 熱生成率
dH/dt     = σ(t) - α H(t)                    // 熱蓄積
B(t)      = max(0, θ_flat - ||∇F_total(t)||) // 退屈熱（勾配フラット検知）
H_eff(t)  = H(t) + β B(t)                   // 実効熱
```

### 8.4 勾配歪み係数

```
g(H, A, M) = 1 + λ_H · (H - H_base) + λ_A · A
```

- `λ_H` ：熱による勾配増幅率
- `λ_A` ：注意による勾配感度
- `H_base` ：基準熱（平静状態）

熱が高いと勾配が増幅・歪み、低いと素直に環境を受け取る。

### 8.5 跳躍（v2.1 修正）

```
H_eff(t) > θ_i + ξ(t)  のとき：
    V  → V'        （新状態への遷移）
    w  → w'        （重み更新＝学習）← 主要な変化
    θ  → θ'        （閾値更新）      ← 主要な変化
    F_pred → F_pred'（仮想勾配更新） ← 主要な変化
    M  → M のまま  （土台基準は不変）← v2.0からの修正
    H  → 0         （熱解放）
```

M自体が変わるように見える場合は、
十分に長い時間スケール（道具・習慣・文化）での変化であり、
単一の跳躍イベントでは起きない。

### 8.6 ドーパミン的重み反転

```
通常状態 ：  w[F_x] < 0  （反発）
ドーパミン介入時：
    誤差解決期待値 E_expected が反発コストを上回る場合
    w[F_x] の実効値が正転（引力化）

実装：
    w_eff[x] = w[x] + d_i · E_expected[x]
    d_i ： 個体のドーパミンパラメータ
```

これによりキツネは危険・コスト・未知を引力として扱える。
ウサギのドーパミン値が低い場合、この反転は起きにくい。

---

## 9. 実装上の重要ポイント（v2.1 更新）

### 9.1 「知能らしさ」は勾配場の設計から生まれる

> NPCを賢くしようとするな
> 勾配場を豊かにせよ
> NPCは勾配に従って流れるだけでよい

個体の行動多様性は `w[x]` の符号と強度の違いから自然に生まれる。

### 9.2 F_pred（仮想勾配）の実装優先度

内部予測勾配を実装することで、「空間に対象がいない状況での行動」が生まれる。

```
キツネが匂いを追う   → F_pred（仮想獲物位置）に向けて流れる
ウサギが隠れて待つ   → F_pred（仮想捕食者）の反発が持続
群れが散り散りになる → 個体ごとの F_pred がバラバラになる
```

`F_pred` なしでは、エージェントは「見えているもの」にしか反応できない。

### 9.3 退屈熱の必要性

勾配フラット検知（退屈熱）を入れることで、
「何もないときに動く」という生物らしさが生まれる。

```
退屈熱なし → 安全地帯にいるエージェントは永遠にそこに留まる
退屈熱あり → 一定時間後に探索流動が発生する
```

### 9.4 環境の勾配場を動的に設計する

```
静的な障害物マップ      → コスト勾配
資源の湧き・枯渇       → 引力勾配の時間変化
風の変化              → 匂い・音勾配の方向バイアス
個体の痕跡            → 過去勾配の残響（匂い場）
群れの密集            → 排他的なコスト勾配増加
```

世界を豊かにすれば、NPCは自動的に豊かに振る舞う。

### 9.5 状態名より遷移条件が本体

RDF的には状態名よりも、
**どの勾配がどの強度で整合慣性を突き破るか**の方が重要である。

## 10. 応用可能性（v2.1）

本モデルは生態系NPCの設計書であるが、
勾配場フレームワークはそのままスケール変換可能である。

| スケール | 勾配の意味 |
|---|---|
| 細胞 | 化学濃度勾配 |
| 動物 | 感覚的素流圧 |
| 人間 | 感覚＋概念空間の勾配 |
| 社会 | 規範・経済・言論の勾配 |
| 市場 | 価格・需給の勾配 |

ウサギ型・キツネ型・ハヤブサ型のプロファイルは
社会構造の分析にも転用できる。

## 11. 総括

本設計の本質：

> NPCは行動表で動く駒ではなく
> 勾配場の中を流れる構造である

生態系とは：

> 複数の構造が相互に勾配を書き換え合い
> 引力・反発・コストの合成場の中で
> 整合と跳躍を繰り返す動的場である

そしてその全体は一つの方程式で読める：

```
V(t+1) = M · V(t) + Σ w_x F_x + ξ(t)
```

## 12. v1.0 / v2.0 との対応表

| v1.0 概念 | v2.0 対応 | v2.1 修正 |
|---|---|---|
| 熱が行動を引き起こす | 熱が勾配を歪め、歪んだ勾配が流動を変える | 変更なし |
| 熱管理が基本目的 | 勾配場の中での整合維持が基本動態 | 変更なし |
| 跳躍時 M → M' | 跳躍時 M → M' | 跳躍で変わるのは w, θ, F_pred。Mは土台基準で不変 |
| 資源は熱の解消源 | 資源は引力勾配の源泉 | 変更なし |
| 警告は熱の同期 | 警告は勾配場の同期書き換え | 変更なし |
| 隠密は情報遮断 | 隠密は相手の勾配場への情報書き込みを防ぐ操作 | 変更なし |
