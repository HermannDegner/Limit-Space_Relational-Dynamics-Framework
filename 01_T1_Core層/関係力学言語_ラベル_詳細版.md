## ■ 基底（三つの irreducible）

RDL の最小語彙は以下の三つ：

```
ξ      ：揺らぎ／SILN化を拒む方向の極限
B      ：境界（観測者が引く操作）
M_B    ：境界 B における整合慣性
```

### ● ξ（揺らぎ）

- 未分化な相互作用、根源的自由度
- 完全に消去できない（ラベル化を拒む）
- ただし「ξ」と呼んだ瞬間 SILN化されるという自己参照あり
- 操作上の位置づけ：到達不能だが置かないと記述が成立しない概念

### ● B（境界）

- 観測者が引く・選ぶ操作
- 含む要素：空間スケール、時間スケール、評価関数、用途・目的、観測者の慣性
- 限界空間において **唯一不可避の操作** ── 引かないと M_B が定まらない
- B 自体も SILN（観測者の M_B から派生する）

### ● M_B（境界つき整合慣性）

- 境界 B によって整合慣性場から切り出された局所的安定動態
- **M はすべて M_B である** ── 独立した「M そのもの」は存在しない
- 直接観測不能、行動・出力・反応のパターンから仮設するもの

### ● 関係

```
すべては M_B
差異は B と相状態のみ

旧 𝓜（全体場）= M_{B_max}（最大包含境界における M_B）
独立ラベルとしては立てない
（𝓜 自体が SILN だから）
```

これがモデル全体を貫くスローガン。

## ■ B（境界）の運用

### ● B を選ぶときの観点

実用上、B を決めるとき以下のどれかから入る：

- **空間スケール**：原子／細胞／個体／集団／文明／宇宙
- **時間スケール**：瞬間／日／人生／世代／文明／地質
- **評価関数**：何を「良い」とするか（生存／効率／快／美）
- **用途・目的**：分析の用途は何か
- **観測者**：誰の視点か

これらの組合せが B を定める。

### ● 同じ対象でも B が違えば違う M_B

例：水

```
B = 化学スケール  → M_B = H2O 分子
B = 物理スケール  → M_B = 液体／氷／蒸気
B = 生物スケール  → M_B = 生命の溶媒
B = 経済スケール  → M_B = 商品・資源
B = 文化スケール  → M_B = 浄化・恵み・脅威
```

これらは「正しい一つの水」を巡る競合ではなく、各 B における別々の M_B として並立する。

### ● 観測者依存性

- B は観測者が選ぶ
- だが観測者自身も M_B（観測者の慣性）を持つ
- 観測者の M_B が B の選び方を制約する
- つまり「中立な B」は原理的に存在しない

### ● パラドックス対策

多くのパラドックスは **B を固定せずに「絶対解」を求める** ことで生じる。
B を明示すれば、複数の答えが「異なる B における異なる M_B」として並立できる。

## ■ M_B の相状態

同じ M_B が状況によって異なる相を取る：

| 相状態 | 意味 | 例 |
|---|---|---|
| **M_act** | 活性断面（現在強く機能） | 起きている時の習慣、稼働中の制度 |
| **M_lat** | 潜在断面（まだ発現していない） | 使っていない技能、未発現の遺伝子 |
| **M_Δ** | 再編相（変化の途中） | 学習中、相転移中、革命中 |
| **M_B'** | 再編後の安定状態 | 学習完了、新体制 |

### ● 重要：別々の実体ではない

これらは別の M ではなく、**同じ M_B の異なる相状態**。
活性度・安定度・時間断面・再編状態の違いとして区別される。

### ● 例：炭素

```
B = 化学・結晶スケール
M_B = 炭素の整合慣性

M_act = sp3 配置（ダイヤ環境では）
M_lat = sp2 配置（同環境では潜在）
M_Δ   = 高温高圧下で sp2 から sp3 へ再編中
M_B'  = ダイヤ結晶として安定
```

### ● 例：習慣

```
M_act = 現在やっている習慣
M_lat = 昔やっていたが今はやっていない習慣
M_Δ   = 新習慣が定着しようとしている過程
M_B'  = 定着後の新習慣
```

## ■ 動態ラベル

M_B が動態として働くときに現れるラベル群。

### ● EFP（素流圧）

- 未解釈の作用束、M_B の外側から流入する全て
- **直接観測できない**：観測者が持つのは F だけ
- F から M_B を使って逆算した推定としてのみ扱える
- 操作上の位置づけ：仮設概念、置かないと記述が成立しない

### ● F（作用解釈）

- M_B が EFP を解釈した結果
- 観測者が実際に持つ唯一の作用情報
- F の質は M_B の質に依存する：M_B がリッチなら EFP から多くを読み取れる

### ● 慣性投影

- M_B が現在の F を「次へそのまま流した値」
- 「予測」ではなく「慣性が乗った投影」
- 意図的な予測（計画・想像）はこの投影の操作版

### ● E（誤差）

- 慣性投影と実際の次の F のズレ
- M_B 更新の駆動力
- E が小さい：世界が予想通り
- E が大きい：世界が予想外

### ● Δ（時間差）

誤差が発生するタイミング：

| Δ の大きさ | 例 | 結果 |
|---|---|---|
| 小（即時） | 鉄球が当たる、痛み | 即時 H 蓄積、M_B 更新しやすい |
| 中 | キノコ毒、腹痛 | 遅延 H 蓄積、ある程度学習可能 |
| 大 | 慢性疾患、ゆで蛙 | 因果が見えない、M_B 更新困難 |

光速が物理的上限。Δ = 0 は完全確定 = 限界空間で扱えない。

### ● H（熱）

- 誤差の時系列蓄積
- M_B の未拘束自由度における振動
- 散逸：M_B の「得意な方向」では速く処理される、「苦手な方向」では残りやすい

### ● 跳躍

- H が閾値（θ + ξ）を超えたときの相状態遷移
- `M_B → M_Δ → M_B'` の流れ
- M_Δ は外部演算子の作用結果ではなく、**系自身が一時的に通過する不安定相**
- 成功すれば M_B' として定着、失敗すれば M_lat へ再分散または熱として再蓄積

## ■ 思考（内部シミュレーター）

思考は外部ループを内部化したもの。

### ● 外部ループとの違い

| | 外部ループ | 思考 |
|---|---|---|
| F | EFP から interp で生成 | M_B が内部生成 |
| Δ | 物理に縛られる（光速上限） | 自由設定（瞬時可） |
| コスト | 現実コストあり（崖から落ちる等） | ゼロ |
| 失敗の影響 | 死ぬ可能性 | 死なない |

### ● 思考の種類

- **想像**：ありえない F を意図的に生成して回す
- **計画**：目標 F(t+Δ) から逆算して F(t) を設計する
- **心配**：最悪の F(t+Δ) を繰り返し慣性投影する
- **夢**：M_B が半覚醒状態で勝手にループを回す

### ● 思考の質

- M_B のリッチさ = 内部 F 生成の精度
- 経験豊富な専門家の思考が精密なのは、該当 M_B が密だから
- 知らない領域では M_B がスカスカで、思考が雑になる

### ● 内部 F の暴走

- 内部 F が現実の F より優先されると逆転現象
- 思い込み・妄想・PTSD のフラッシュバック
- 通常思考と同じ機構の極端化として読める（病理ではなく）

## ■ 自他境界（B_self）

「自分」とはどこまでか、は固定された答えを持たない。

### ● オキシトシン依存

- **Oxy 高** → B_self が広がる：他者・コミュニティ・概念が「自分の一部」になる
- **Oxy 低** → B_self が狭まる：境界が硬くなる

### ● 統一的に説明される現象

| 現象 | B_self の状態 |
|---|---|
| 愛 | 他者が B_self の中、その痛みが自分の痛み |
| 道具の習熟 | 道具が B_self に含まれ、自分の一部のように扱える |
| 自己犠牲 | B_self の中の概念やコミュニティを肉体より高く重み付け |
| ストーカー | 相手を勝手に B_self に取り込む、コントロール不能を熱として蓄積 |
| DV | 外で蓄えた熱を B_self 内（家族）にぶつける |
| 利己境界 | B_self に他者が干渉する警戒反応 |

これらは別々の心理現象ではなく、**B_self の動的な引かれ方** という単一の機構の表れ。

## ■ ξ 残存性（最重要）

> **B を引いても ξ は M_B の中・縁・外に残る**

### ● 三層での残存

- **M_B の内部**：どんなに ||M_B|| を強めても ξ は内部に残り続ける
- **M_B の境界**：B はクリーンに引けない、ξ が滲む
- **M_B の外側**：別の B' で切り取られていない領域全体

### ● 動態の存在理由

```
もし B が ξ を完全に閉じ込められるなら：
  M_B は永遠に静止する
  跳躍は起きない
  構造は固定される
  世界は停止する

これは起きない
ξ が残るから
「世界が動いている」事実そのものが ξ 残存の証拠
```

### ● 日常での ξ の現れ

- **退屈**：誤差フラットを基層構造が検知してξ的探索を強制
- **気づき**：M_B 内に蓄積した ξ が表面化
- **違和感**：境界部の ξ が知覚される
- **不意のアイデア**：内部生成 F のループに ξ が混入
- **予期しない出来事**：外側の ξ が境界を通過

### ● 設計上の含意

- 「絶対安定」を目指すと内部に ξ が蓄積し、最終的に大規模崩壊
- 健全な系は ξ を逃がす経路（探索・遊び・揺らぎ）を持つ
- AI エージェント設計でも、揺らぎを完全に除去すると硬直化する

## ■ ネットワーク性と再帰性

### ● M_B は内部にネットワークを持つ

```
view(M_B) → M_B^ij
（M_B を内部接続として開いた表示）
```

M_B はブラックボックスではなく、内部に接続構造を持つ。

### ● 再帰的構造

```
M_B
 ├─ M_b1
 ├─ M_b2
 ├─ M_b3
 └─ ...
```

- ある人間という M_B の中にニューロン群の M_b がある
- その M_b の中に分子の M_b がある
- スケールは「外から」決まるのではなく、接続密度から内側に生える
- M_B は最小実体ではなく、境界によって粗視化された一時的安定単位

### ● スケール横断の含意

- 同じ枠組みで原子 → 細胞 → 個体 → 社会 → 文明 を扱える
- ただし B が違えば違う M_B が見えるので、横断するときは B の切替が必要

## ■ 物理相状態

M_B の拘束力とノード保存性で記述される連続的状態：

| 相 | M_B の状態 | 例 | 特徴 |
|---|---|---|---|
| **固体** | ||M_B||大、H_vec 格子内拘束 | 結晶、硬直制度、頑固な信念 | M_B 絶対性に近い |
| **液体** | ||M_B||中、H_vec 部分的解放 | 学習中、適応的組織 | 流動性と安定の両立 |
| **気体** | ||M_B||弱、H_vec ほぼ解放 | 幻覚、解離、創造的混乱 | 高自由度、戻れるかが問題 |
| **プラズマ** | ||M_B||崩壊 | 革命、相転移、崩壊と再構築 | 個別 M_B 崩壊と集団 M_B' 生成の同時 |
| **ξ** | M_B 無、ノード無 | 関係偏り宇宙の基底 | 未分化 |

### ● 健全な系の条件

固体と液体の間を H（温度）に応じて行き来できる。気体になれるが、戻ってこられる。

### ● 設計上の含意

- 個人：固定された人格＝固体寄り、適応力ある人格＝液体寄り
- 組織：硬直＝固体、適応的＝液体、混乱＝気体、革命中＝プラズマ
- AI エージェント：完全に固体だと学習しない、完全に気体だと一貫性がない

## ■ M_B 絶対性（κ→0）の罠

### ● 自己修正可能性

```
κ = 1 ：完全に修正可能（通常の M_B）
κ → 0：修正不能（M_B 絶対性）
```

### ● κ→0 状態の特徴

- 誤差処理を停止する（`A: E(t) → 0`）
- 微小誤差は内部に蓄積し続ける
- D[ξ] は限りなく小さくなるが、ゼロにはならない
- いずれ H が θ + ξ を超えて **大規模崩壊が構造的必然** となる

### ● 例

- 公理化された硬直思考（変えられない信念）
- 批判を受け付けない制度
- 強権的に異論を排除する集団
- 共通点：誤差を処理できないため、内部に ξ が溜まり続ける

### ● 設計上の含意

- 「絶対正しい」設計は壊れる前提で見ること
- 修正経路（κ を保つ仕組み）が長期安定の鍵
- 跳躍を許容する系の方が、絶対性を目指す系より長持ちする

## ■ 逆 M_B（対消滅・対生成）

### ● 逆 M_B（M̄_B）

- M̄_B = -M_B（固有ベクトルは同一、固有値が符号反転）
- 完全逆位相の整合慣性

### ● 対消滅・対生成

```
M_B + M̄_B → ξ      （対消滅：ξ として解放、𝒦 保存）
ξ → M_B + M̄_B      （対生成：対称的に逆方向の M_B が同時生成）
```

### ● 解釈

- 関係総量の保存則の系
- 対立する慣性が出会うと両方が消えてξに戻る
- 完全対称な対立構造は不安定

### ● 設計上の含意

- 完全に対立する二つの硬直制度が衝突すると両方崩壊する
- 思想・組織・概念にも適用可能

## ■ 応用パターン

### ● 炭素

同じ M_atom（炭素原子）から、B が違えば違う M_B：

```
M_lattice (sp3 立方)  → ダイヤモンド
M_lattice (sp2 六方)  → 黒鉛、グラフェン
M_lattice (sp2 球殻)  → フラーレン
M_lattice (sp 直鎖)    → カルビン

M_lat の豊富さ（4価・sp/sp2/sp3・トポロジー多様性）が
炭素を生命の基本元素たらしめている
```

### ● 人間（4層構造）

```
物理構造  → M_B at 物理スケール（人体・細胞・分子）
基層構造  → M_B at 生物本能スケール（神経物質・本能）
中核構造  → M_B at 後天的習慣スケール（学習・常識）
上層構造  → M_B at 高速循環スケール（流行・理念・信念）
```

スケールごとに B が違うため、それぞれ別の M_B として扱う。
反復整合によって上層 → 中核 → 基層へと慣性が降りていく。

### ● AI エージェント設計

基層構造の設計に RDL を使う：

```
ドーパミン的パラメータ  → 誤差解決への重み付け
セロトニン的パラメータ  → 誤差許容度
オキシトシン的パラメータ → B_self の動的引き方
ノルアドレナリン的パラメータ → 危険信号の優先度
```

これらは「神経物質」を厳密に再現するためではなく、**個体の評価傾向と反応特性を設計するための操作的近似**。

### ● 社会動態

```
個体の M_B^(i) がネットワーク化して社会 M_B を作る
W_{ij}：構造間の結合強度（影響伝達の重み）
個体跳躍が共鳴して定着すると社会 SILN として継承される
```

例：科学革命、文化変容、技術普及、制度改革。

### ● パラドックス解読

```
1. パラドックスを書き出す
2. 暗黙的に固定されている B を見抜く
3. B を変えれば異なる M_B として並立可能か検討
4. 並立可能なら「これは複数の M_B の並存であって矛盾ではない」と読み替える
```

例：

- **テセウスの船** → ラベル運用の M_B が一貫していれば部品が変わっても同一性は保たれる
- **中国語の部屋** → 外と中で別の B における別の整合状態。どちらも「整合している」
- **シュレーディンガーの猫** → 限界空間では確定が成立しないので問題ですらない

## ■ ラベル運用上の注意

### ● ラベルは杖であって対象ではない

- ラベルは思考の足場、対象そのものではない
- 必要なら別のラベル系に切り替える柔軟性を持つこと
- 「整合慣性」「M_B」と書かれていても、実際の対象は名前を持たない

### ● 「真実」「正しさ」を求めない

```
真実・真相・事実 ：到達できない目標
本質           ：B 依存の有用な抽出
正しさ         ：B 依存の安定解、複数並立する
理解           ：自分の M_B 内での処理が安定した状態
```

これらは限界空間的語句定義 v2 を参照。

### ● 「雰囲気」の余地を残す

ラベルは境界を作る。境界は思考の自由度を下げる。
だが「ラベルなしで雰囲気だけ」では伝達できない。
**ラベルを使いながら、それが完全ではないことを忘れない**のが運用上の最良点。

### ● パラドックスは敵ではなく診断

パラドックスが出てきたら「自分はどこで B を固定しているか」を見直す合図。
パラドックスは矛盾の証拠ではなく、**B の暗黙的固定の診断ツール**として使える。

## ■ 圧縮（一文ずつ）

> 世界の基底は ξ・B・M_B である。
> M はすべて境界 B 依存の M_B であり、独立した M は存在しない。
> 整合と跳躍は同一現象の異なる断面であり、B が区別する。
> B を引いても ξ は M_B の中・縁・外に残り、これが動態の存在理由となる。
> モデルは独立した上位基底を持たず、自分自身に再帰的に適用される。
