# SILN分解の数理フレームワーク v2.0
*（RDF概念的数理モデルv2.2対応／S除去・EFP-F分離・時間差誤差・慣性投影パッチ適用済み）*

---

## ■ 概要

SILN（Survival-biased Integration of Local Linear approximations in a Nonlinear world）は
そのままでは抽象度が高すぎるため、分解なしでは意味を持たない。

したがって、実用上は

**「境界を仮定し、整合慣性 M を仮設する操作」**

として扱う。

※ Mは「抽出」ではなく「仮設」である。直接観測できないため、
　行動・出力・反応のパターンから推定した仮説に過ぎない。

---

## ■ 全体プロセス

```
① ラベル設定（対象定義）
② 境界仮定（仮説）
③ 状態展開（M, EFP, F, E, H, ξ）
④ 構造仮設（Mの推定）
```

※ v1.1までのS(t)はMの読み出し断面に過ぎないため除去。
　必要な場合はMの断面として取り出す。

SILN分解における最重要ステップ。

以下を仮定する：

- 空間スケール（個体／社会／物理）
- 時間スケール（瞬間／人生／文明）
- 評価関数（生存／効率／快）

```
仮説 = 観測者が引いた境界
```

※ 限界空間では絶対前提は存在しないため必須

## ■ ③ 状態展開

### ● 整合慣性（SILN本体）
```
M
```

- 習慣
- 制度
- 構造パターン
- 認識のクセ

Mは独立して観測できない。
行動・出力・反応のパターンからの仮設に過ぎない。

---

### ● 素流圧（EFP：Elementary Flux Pressure）
```
EFP
```

未解釈の作用束。Mの外側から流入する。

**重要：EFPは直接観測できない**
```
観測者が実際に持つのはF（作用解釈F）だけである
EFPはFからMを使って逆算した推定概念に過ぎない

EFP_estimated = interp⁻¹(M, F)

𝒦と同じ位置づけ：到達不能だが操作上有用なため仮設置する
```

---

### ● 作用解釈F
```
F = interp(M, EFP)
```

MがEFPを解釈した結果。
観測者が実際に持つ唯一の作用情報。

```
知覚の質はMに依存する
MがリッチなほどEFPから多くを読み取れる
MがフラットならEFPはほぼ素通り
```

---

### ● 誤差（慣性誤差）
```
E(t) = F(t+Δ) - M·F(t)
```

- `M·F(t)`：Mの慣性がF(t)に乗ったまま次へ流れた値（**慣性投影**）
- `F(t+Δ)`：実際に来た次のF
- `E(t)`　 ：慣性投影と実際のFの差 = 慣性誤差

**Δ（時間差）の意味：**
```
Δ = 0 は完全確定 = 限界空間で扱えない
Δ > 0 が必然

Δ小　：因果が明確・M更新しやすい（鉄球衝突）
Δ大　：因果が曖昧・M更新しにくい（キノコ毒・エキノコックス）
Δ∞　：因果不明・Mは更新されない（温暖化・ゆでガエル）
```

---

### ● 熱
```
H = ||H_vec||
H_vec ≡ δM(t)　// Mの未拘束自由度における振動
```

慣性誤差の蓄積。
HはEの時系列積分として蓄積する。

---

### ● 揺らぎ
```
ξ(t)
```

確率的変動・根源的自由度。
D[ξ] = 𝒦 / ||M|| により慣性と交換可能。

---

## ■ 基本ダイナミクス

```
F(t)  = interp(M, EFP)          // EFPをMが解釈
E(t)  = F(t+Δ) - M·F(t)        // 慣性誤差
dM/dt = f(M, E, ξ)              // Mは慣性誤差と揺らぎで更新される
```

**流れ：**
```
EFP（仮設・直接観測不能）
    ↓ interp(M, EFP)
F(t)（作用解釈F・観測者が持つ唯一の情報）
    ↓ M·F(t) 慣性投影
    ↓ F(t+Δ) が実際に来る
E(t) = F(t+Δ) - M·F(t)（慣性誤差）
    ↓
H蓄積 → 跳躍 → M更新
```

## ■ テセウスの船への適用

```
SILN = (M, L, W)
```

- M　：整合慣性（SILN本体）
- L　：ラベル
- W　：社会的関係

※ v1.1のS_physはMの物理断面として読み直す

本質：

```
船 = M(t) の軌跡
```

観測者ごとにMは微妙に異なる。
造船士のM、乗組員のM、港の人のM、それぞれ別物である。
それでも「テセウスの船」というラベルが共有できるのは、
Mが完全一致しなくても機能する共有装置としてラベルが働くからである。

## ■ ラベルの二面性

```
① 境界を作り、思考の自由度を下げる
② 観測者間のMのズレを束ね、共有を可能にする
```

Mが完全に一致していればラベルは不要。
完全にバラバラならば共有できない。
**微妙にズレているちょうどその状態**でラベルが意味を持つ。

ラベルとは観測者間の慣性誤差を吸収するバッファである。

## ■ パラドックスの解体

```
argmax Identity が複数存在
```

にも関わらず単一解を要求するため破綻する。

本来、事象は複数のM（SILN）として並立可能である。
単一の絶対解へ収束させようとすることで整合が破綻する。

## ■ SILN分解の限界性

**Mの検証問題**

ラベルのMを強固にするには、複数の観測者から収集する操作が有効である。
しかしその操作自体がEFPとしてMに作用する。
したがってラベルのMは観測プロセスと不可分であり、
観測前の純粋なMには到達できない。

**絶対解は存在しない**

```
M_観測者A ≠ M_観測者B ≠ M_観測者C ...
```

複数観測によってラベルの輪郭は統計的に近似できるが、
真のMへの収束は保証されない。

SILN分解は絶対解を出すツールではなく、
**目的に応じた有効な近似を仮設するツール**である。

限界空間において確定は局所的確定に過ぎない。
SILN分解もその例外ではない。
