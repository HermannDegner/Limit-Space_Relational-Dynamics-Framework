# 既存倫理体系のSILN優先構造 v0.1

*RDL的 倫理解釈 別冊：既存倫理体系はどのSILNを優先していたか*

---

## ■ 0. 位置づけ

本文書は「RDL的 倫理解釈 v0.1」の別冊である。

本書では、既存の倫理体系を「正しい／間違っている」で比較しない。
それぞれの倫理体系を、

```text
どの境界 B を採用しているか
どの階層の M_B を優先しているか
どのHを下げようとしているか
どのHを誰に負担させているか
どの破断を最も危険視しているか
```

という観点から読む。

つまり、既存倫理体系とは、絶対善悪の一覧ではなく、
特定のSILNを優先する境界維持反応の体系である。

---

## ■ 1. 基本命題

```text
倫理体系とは、
複数存在する正しさ感のうち、
どのSILNを優先し、どのSILNを抑制するかを定める
境界選択の様式である。
```

より圧縮すると：

```text
既存倫理体系とは、優先Bの歴史的パターンである。
```

倫理体系の違いは、しばしば次の違いとして現れる。

```text
個体を優先するか
関係を優先するか
集団を優先するか
制度を優先するか
結果を優先するか
規則を優先するか
徳性を優先するか
苦痛低減を優先するか
自由を優先するか
平等を優先するか
秩序を優先するか
生命圏を優先するか
```

これらはいずれも、異なるB・異なるM_B・異なるH処理を選んでいる。

---

## ■ 2. 分析フォーマット

各倫理体系は、以下の形式で分解できる。

```text
ラベル：
    倫理体系名

優先B：
    どの境界を主に守るか

優先SILN：
    どのM_Bの安定を優先するか

主なH処理：
    どの誤差熱を下げようとするか

抑制されるSILN：
    その体系で後回しにされやすいもの

有効範囲：
    その体系が強く機能する状況

破断条件：
    その体系が壊れやすい状況
```

## ■ 3. 功利主義

```text
ラベル：
    功利主義

優先B：
    個体を超えた集計境界
    集団全体・社会全体・人類全体

優先SILN：
    苦痛低減SILN
    快・効用最大化SILN
    結果評価SILN

主なH処理：
    全体Hを最小化しようとする
    個別のHより、総量としてのHを重視する

抑制されるSILN：
    個体境界
    権利境界
    約束・義務・尊厳などの非集計的SILN

有効範囲：
    政策判断
    資源配分
    公衆衛生
    災害対応
    大規模意思決定

破断条件：
    少数者のHを多数者のH低減のために過剰負担させる場合
    個体を集計単位としてのみ扱う場合
    測定不能な価値を無理に数値化する場合
```

功利主義は、個体ごとの境界維持反応を一段抽象化し、全体Hの最小化を優先する倫理体系である。

強いが、個体Bを踏み潰しやすい。

## ■ 4. 義務論

```text
ラベル：
    義務論

優先B：
    行為規則の境界
    普遍化可能なルールの境界

優先SILN：
    規則整合SILN
    義務SILN
    人格尊重SILN

主なH処理：
    ルール破壊による社会的・概念的Hを抑える
    結果に流されて境界が崩れることを防ぐ

抑制されるSILN：
    結果最大化
    状況依存的な柔軟対応
    関係固有の事情

有効範囲：
    権利保護
    約束
    法制度
    手続き的正義
    人格を手段化しない判断

破断条件：
    規則を守ることで大きな下層破断が起きる場合
    例外状況への対応が硬直する場合
    ルールそのものが古いM_Bとして固定化している場合
```

義務論は、結果ではなく、規則境界の維持を優先する。

これは社会M_Bを安定させるが、過剰化するとM_B絶対性に近づく。

## ■ 5. 徳倫理

```text
ラベル：
    徳倫理

優先B：
    個体の長期的形成境界
    人格・習慣・生き方の境界

優先SILN：
    人格形成SILN
    習慣SILN
    中庸・実践知SILN

主なH処理：
    個別行為よりも、長期的に安定したM_B形成を重視する
    その人がよく生きられる構造を整える

抑制されるSILN：
    短期結果
    単発行為の形式的評価
    外部ルールへの単純従属

有効範囲：
    教育
    人格形成
    熟練
    リーダーシップ
    生活倫理

破断条件：
    徳とされるものが特定文化の中核構造に閉じる場合
    身分秩序や既存規範を自然化する場合
    社会的弱者に「よい人格」を要求しすぎる場合
```

徳倫理は、行為単体よりも、行為を生むM_Bの長期形成を優先する。

RDL的には、かなり中核構造寄りの倫理である。

## ■ 7. 権利論・自由主義

```text

優先SILN：
    個体自律SILN
    不可侵境界SILN
    選択自由SILN

主なH処理：
    他者・国家・集団による個体B侵害を抑える
    個人が自分のM_Bを再記述する余地を守る

抑制されるSILN：
    集団秩序
    伝統
    共同体義務
    結果最大化のための個人犠牲

有効範囲：
    国家権力の制限
    表現・信仰・身体の自由
    少数者保護
    強制への抵抗

破断条件：

    共同体維持に必要な中核SILNを破壊しすぎる場合
```

自由主義は、個体Bの不可侵性を強く守る倫理体系である。

悪用判定においては非常に重要だが、関係性や構造的制約を見落とすと破断する。

社会安定には強いが、個体の再記述可能性を奪う危険がある。

契約論は、社会M_Bを恣意ではなく合意により構成しようとする倫理体系である。

自然法論は、制度より深い階層にある秩序を優先する。

ただし「自然」と呼ばれるもの自体がSILNであるため、破壊検査が必要である。

## ■ 23. 総合命題

既存倫理体系の違いは、どの正しさ感が本物かの争いではない。

それぞれが、異なるB・異なるM_B・異なるH処理を優先している。

```text
功利主義は全体Hを優先する
義務論は規則Bを優先する
徳倫理は長期人格M_Bを優先する
ケア倫理は関係Bを優先する
自由主義は個体Bを優先する
共同体主義は集団Bを優先する
環境倫理は生命圏Bを優先する
解放倫理は再記述可能性を優先する
応報主義は責任H再配置を優先する
修復倫理は関係再編を優先する
```

したがって、倫理論争とは多くの場合、

```text

```

の争いである。

## ■ 24. RDL的利用法

この別冊は、既存倫理体系を採用・否定するためのものではない。

目的は以下である。

```text
1. 倫理体系ごとの優先Bを可視化する
2. 倫理衝突をB衝突として読む
3. 各体系の有効範囲と破断条件を明示する
4. RDL道具特性における悪用判定へ接続する
5. 法・罰・責任・同意・権利の再分解に使う
```

RDL的には、倫理体系は「正解候補」ではなく、
特定条件下で機能するM_Bである。

したがって、運用時には必ず構造破壊学的に扱う。

```text
どこまで使えるか
どこで壊れるか
何が残るか
他のBへ移植できるか
```

を確認する。
