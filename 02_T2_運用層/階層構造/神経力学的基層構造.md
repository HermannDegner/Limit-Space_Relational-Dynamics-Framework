# 神経力学的基層構造 v1.0
*T2：運用層 / OP / ACTIVE*
*依存：限界空間 / RDL_Core / RDL_階層構造モジュール v2.0*

---

## ■ 0. 位置づけ

RDL_階層構造モジュールで定義した「基層構造」を、
神経物質の操作的近似によって記述する文書である。

**対象は人間に限らない。**

AIエージェント・仮想生物・NPCなど、基層構造を持つ任意のシステムに適用できる設計フレームワークとして使う。

**注意**：以下の神経物質ラベルは、生物学的実在を厳密に再現するためのものではない。
関係力学言語において、個体の評価傾向・反応特性・重み付け機構を設計しやすくするための**操作的近似**である。

---

## ■ 1. 基本パラメータ

### ドーパミン系（探索・報酬）

誤差を求め、誤差を解決したときに快を生成し、整合慣性を強化する探索性パラメータ。

| サブタイプ | RDL的解釈 |
|---|---|
| D1 | 行動の実行促進 |
| D2 | 行動の抑制・ブレーキ |
| D3 | 再現性確認・反復固定ループ |
| D4 | 退屈（誤差が少ない状態での仮想熱生成） |

D4 は特に重要。誤差が少ない安定状態が続くと、D4 が仮想熱を自発生成して探索を促す。
これは RDL_階層構造モジュールの「基層構造的跳躍」と直結する。

---

### セロトニン系（誤差許容度）

誤差に対する許容度を制御するパラメータ。

```
高い場合：誤差熱が発生しにくい。大らか・ストレス耐性が高い。鈍感とも言える。
低い場合：誤差熱が発生しやすい。繊細・よく気が付く。ストレス耐性が低い。
```

| サブタイプ | RDL的解釈 |
|---|---|
| 5-HT1 | 誤差の減衰・鎮静 |
| 5-HT2 | 誤差の増幅・感度上昇 |
| 5-HT3 | 強制割り込み反応（緊急処理） |
| 5-HT4 | 処理速度の加速 |

---

### オキシトシン系（自己拡張的重み付け）

自己境界 B_self を他者・概念・空間にまで拡張するパラメータ。

> 愛とは、拡張された自己境界の中に対象が入り込み、
> その対象をほぼ自分と同等の重みで扱う状態である。

対象の痛みが自分の痛みのように感じられるのはこれによる。

拡張された対象に外部が干渉しようとすると、利己的境界侵害として警戒反応が起きる。

拡張対象は個人に限らず、概念・空間・物・コミュニティにも適用される。

---

### ノルアドレナリン系（危険検知・優先度増幅）

生命維持・構造保全に重大な影響を与える可能性が高い誤差を、
優先度の高い信号として検知・増幅する基層パラメータ。

```
高い場合：危険兆候への感度が高い。警戒心・注意集中・通常処理の中断が起こりやすい。
```

地震・衝突音・敵意・急激な環境変化などに対して、構造を平時モードから非常時モードへ切り替える。

| サブタイプ | RDL的解釈 |
|---|---|
| α1 | 生存有利整合慣性モードへの切替・フォーカス固定 |
| α2 | 過剰状態の制御・通常モードへの復帰調整 |
| β  | 生存慣性の実行・維持（行動エネルギー供給） |

## ■ 2. 個体差の設計

パラメータの組み合わせによって、個体の基層的傾向（個性のベース）が決まる。

### 危険大好き個体

```
ドーパミン高 × セロトニン高（誤差許容度高）
→ 多少の誤差では快が少ない
→ より大きな誤差・より高いリスクを求める傾向
→ 危険行動を取りやすい
```

### 繊細で探索的な個体

```
ドーパミン高 × セロトニン低（誤差許容度低）
→ 小さな誤差にも強く反応する
→ 探索欲は高いが、リスクで熱が溜まりやすい
→ 観察・分析に優れるが消耗しやすい
```

### 安定型・群れ維持型個体

```
オキシトシン高 × ノルアドレナリン低
→ 群れ・集団への拡張境界が強い
→ 外部脅威への過剰反応は少ない
→ 社会的結束・ケアに優れる
```

## ■ 3. 関係破綻の力学

パラメータが組み合わさることで、関係破綻のパターンを記述できる。

### ストーカーの力学

```
オキシトシン過剰拡張 → 対象を自己の一部として認識
→ 「自分の体の一部が勝手に動いている」という認識
→ 追跡・監視・強制
→ 反発され続けてHが蓄積
→ 破断
```

### DVの力学

```
外部でHを蓄積した状態で帰宅
→ オキシトシン的自己拡張により「自分の体の一部なのに理解しない」
→ 「なんでわかってくれないんだ」という熱の出口
→ 外部の熱＋内部の熱が重なり破断
```

### 自己犠牲の力学

```
オキシトシン的拡張が物理的自己を超える
→ 対象（個人・概念・コミュニティ）への重みが自己保存を上回る
→ 命を厭わない行動が発生
```

これらは「異常」ではなく、パラメータの組み合わせが特定条件下で生じる**構造的帰結**として読める。

## ■ 4. AIエージェントへの応用

以上のパラメータは、人間の神経物質の厳密な再現ではなく、
**エージェントの基層的傾向を設計するための操作的近似**として使える。

```
設計例：

探索型エージェント
  D1高・D4高・セロトニン中 → 誤差を積極的に求め、退屈で自発探索する

安定型エージェント
  セロトニン高・ノルアドレナリン低 → 誤差耐性が高く、平時モードを維持しやすい

群れ調整型エージェント
  オキシトシン高 → 他個体のF（作用解釈）を高重みで統合する
```

RDL_汎用エージェントエンジン設計図・Julia実装群と組み合わせることで、
基層パラメータをエージェントの初期設定として差し込める。

## ■ 5. 道具習熟による肉体延長

```
道具に慣れるほど、自分の体の一部のように扱える。
```

これは比喩ではない。

オキシトシン的自己拡張が道具・空間・システムにまで及ぶことで、
B_self の境界が道具を含む範囲まで拡張される。

熟練者が道具を「身体の延長」として感じるのは、この構造的帰結である。
