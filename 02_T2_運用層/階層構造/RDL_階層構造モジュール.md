# RDL 汎用階層構造モジュール v2.0
*T2：運用層 / OP / ACTIVE*
*依存：限界空間 / RDL_Core_v1.0 / SILN_v1.0*

---

## ■ 0. 位置づけ

あらゆる SILN は、時間スケールの安定度によって4層に分解できる。

この4層構造は人間に限らない。動物・国家・生態系・制度・概念──
どのスケールの対象にも適用できる汎用操作ラベルである。

**注意**：これは分かりやすくするためのラベルであり、絶対的境界ではない。
境界近くは常に曖昧であり、「何に使うか（B）」を軸にして引くと整理しやすい。

---

## ■ 1. 4層構造の定義

```
物理構造  ＞  基層構造  ＞  中核構造  ＞  上層構造
←── 安定・低速 ──────────────── 不安定・高速 ──→
```

この並びは SILN の時間スケールの階層と言ってよい。

---

### 物理構造

対象の物質的・エネルギー的基盤。

限界空間の内側に「絶対」は存在しないが、対象を扱う時間スケールにおいて超安定に近い層。

この層が変化するとき、上の三層はすべて再編を迫られる。

逆に言えば、この層の制約が他の三層の可能域を決定する。

```
例：
  人間     ── 骨格・神経系・代謝・重力下での物理的制約
  動物（魚）── 水中生存に最適化された物理的身体構造
  国家     ── 地理・気候・天然資源・人口規模
  企業     ── 設備・資本・物理的インフラ
```

---

### 基層構造

種・システムレベルで長期間 SILN として残ってきた傾向。

個体の一生スケールではほぼ安定しており、意図的に変えることが難しい層。

生物の場合、進化的選択圧によって形成されたものが多い。
非生物的システムの場合、設立時の構造や長期的な環境適応がこれにあたる。

```
例：
  人間     ── 本能・情動傾向・探索欲・社会性・恐怖反応
  動物（群れ系）── 群れ本能・縄張り意識・捕食・回避パターン
  国家     ── 民族的・文化的慣性・歴史的トラウマ・地政学的傾向
  企業     ── 創業時の文化・経営哲学の慣性
```

---

### 中核構造

後天的に獲得・固定された SILN。

上層構造における反復整合によって形成され、学習が定着したものと言える。
反復運用によって重みを増し、しだいに自己目的化していく層でもある。

意図的に変えることは可能だが、時間とコストがかかる。

```
例：
  人間     ── 習慣・常識・職業的技能・価値観の枠組み
  動物     ── 学習済み行動パターン・縄張りの慣習的境界
  国家     ── 法・制度・官僚的手続き・国際条約
  企業     ── 社内規則・業務フロー・ブランド戦略の定着
```

---

### 上層構造

最も高速に循環する SILN。ミーム・流行・理念・信念・キャンペーンを含む。

概念の生存競争が高速で行われる場所であり、
より使われ整合慣性が強まると中核構造に沈降していく。

新たな構造は、素流圧の相互作用によって主にこの層で生成される。

```
例：
  人間     ── 流行・SNS的信念・一時的な目標・ブームへの共鳴
  動物     ── 個体学習中の試行行動・一時的な採餌パターン
  国家     ── 政策・イデオロギー的スローガン・時代の空気
  企業     ── プロジェクト方針・マーケティングメッセージ・組織内の噂
```

---

## ■ 2. 基層構造的跳躍

人間（あるいは特定の生物・システム）を扱う場合、
**誤差のみが熱源ではない**と考えられる。

基層構造には、仮想熱を自発生成するシステムが組み込まれている可能性がある。

### 退屈という現象

「退屈」は、誤差が少ないときに起きる現象として解釈できる。

世界の誤差が起きてから対応するだけでは生存可能性が低い。
そのため、予測性が非常に高い状態が続くとシステムが「退屈」という仮想熱を自発生成し、探索行動を促す。

```
誤差が少ない状態
  → 仮想熱の自発生成（退屈）
  → 探索行動の促進
  → 新しい誤差の発見
  → H の更新・M_B の拡張
```

これは基層構造レベルで組み込まれた、**予防的誤差探索機構**と解釈できる。

### 他のスケールへの適用

```
動物     ── 食物・危険が少ない安定期に探索行動が増える
国家     ── 安定期に内部改革・対外拡張の動きが生まれやすい
企業     ── 競合が少ない市場で新事業探索や社内政治が活性化する
```

いずれも「誤差が少ない安定状態が、探索的な仮想熱を生む」という構造として読める。

## ■ 3. 応用例：多スケールで同じ概念を切る

### 「お金」という概念

```
物理構造  ── 紙幣・デジタル数値・石油由来の印刷基材
基層構造  ── 生存に必要なエネルギーと常に交換できるという安心
中核構造  ── お金が使えるという経験による学習・制度的保障・信用体系
上層構造  ── 「金持ち＝優れている」という物語・成功の象徴としての意味付け
```

### 「法律」という構造

```
物理構造  ── 文書・サーバー・施設・執行機関の物理的存在
基層構造  ── 集団内の暴力抑制・生存保護への原始的ニーズ
中核構造  ── 成文法・判例・官僚的手続き・法的文化
上層構造  ── 人権・正義・憲法的理念・時代の価値観
```

### 「群れ」という構造（動物）

```
物理構造  ── 個体の身体・感覚器・移動能力
基層構造  ── 群れ本能・社会的安心感・階層的服従傾向
中核構造  ── 群れの行動規範・縄張りの慣習・個体間の学習された関係
上層構造  ── 一時的な採餌ルート・個体間の一過性の優劣関係
```

## ■ 4. 操作上の注意

**層間の境界は常に曖昧である。**

同じ要素が観測者のBによって別の層に見える。

```
例：言語
  B = 脳科学スケール  → 基層構造（言語能力は神経的基盤を持つ）
  B = 個人スケール   → 中核構造（後天的に習得する）
  B = 文化スケール   → 上層構造（方言・流行語は高速に変化する）
```

「正しい層分け」を求めるのではなく、
**「このBで見たとき、どの層に近いか」を仮設する**操作として使う。
