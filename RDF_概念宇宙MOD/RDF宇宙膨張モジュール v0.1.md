# RDF宇宙膨張モジュール v0.1（RDF-CE）

---

## ■ 位置づけ

```text
RDF宇宙膨張モジュール（RDF-CE）は、
宇宙スケールにおける距離増大（ハッブル流）を
関係構造の時間変化として記述するモジュールである

本モジュールでは
宇宙膨張を空間の実体的拡張とはみなさず

関係更新遅延構造の再配置として扱う
```

---

## ■ 基本命題

```text
宇宙膨張とは
関係距離の時間的再スケーリングである

距離とは
関係更新の伝播遅延として定義される
```

---

## ■ 関係距離と遅延

[
d_{ij}(t)
]

[
\tau_{ij}(t)=\frac{d_{ij}(t)}{c}
]

```text
d：
    関係距離

τ：
    関係更新の遅延

c：
    関係更新の最大伝播速度
```

---

## ■ 関係尺度因子

```text
関係距離は以下の形で分解される
```

[
d_{ij}(t)=a_R(t),\chi_{ij}
]

```text
a_R(t)：
    関係尺度因子（時間依存）

χ：
    固定関係座標（初期構造）
```

---

## ■ RDFハッブル則

距離の時間変化より

[
v_{ij}(t)=\frac{d}{dt}d_{ij}(t)
]

[
v_{ij}(t)=H_R(t),d_{ij}(t)
]

[
H_R(t)=\frac{\dot a_R(t)}{a_R(t)}
]

---

## ■ 解釈

```text
H_R：
    関係構造の時間変化率

ハッブル則とは
関係距離が一様倍率で更新される場合に生じる比例関係である
```

---

## ■ 関係保存則との接続

[
||M||\cdot D[\xi]=C
]

```text
整合慣性が減少すると
揺らぎ（自由度）が増加する

この自由度の増加は
構造拘束の緩和として現れ

結果として関係距離が拡大する
```

---

## ■ ハッブル率の導出

宇宙スケール平均をとると

[
D[\xi]*{\text{cosmic}}=\frac{C}{||M||*{\text{cosmic}}}
]

したがって

[
H_R(t)
======

\beta,\frac{d}{dt}\ln D[\xi]_{\text{cosmic}}
]

または

[
H_R(t)
======

-\beta,\frac{d}{dt}\ln ||M||_{\text{cosmic}}
]

```text
β：
    スケール依存係数
```

---

## ■ 解釈

```text
宇宙膨張とは

整合慣性の緩和
↓
自由度（揺らぎ）の増加
↓
関係距離の拡張

という過程として理解される
```

---

## ■ 局所性の可能性

```text
ハッブル率は必ずしも一様とは限らない
```

[
H_R = H_R(x,t)
]

```text
したがって

宇宙膨張は
大域一様現象ではなく

スケール依存または局所現象として
観測される可能性がある
```

---

## ■ ダークエネルギーとの関係

```text
Λ_Rは宇宙定数ではなく
関係構造のスケール依存項として解釈できる
```

[
H_R^2 \sim f(C, ||M||, D[\xi])
]

```text
加速膨張は

自由度増加率の非線形性
または

Cのスケール効果として現れる
```

---

## ■ RDF-GRとの接続

```text
関係計量R_{μν}は
整合慣性・揺らぎ・素流圧によって決定される

その時間発展が
関係尺度因子a_R(t)として観測される
```

---

## ■ NAMとの接続

```text
小スケール（銀河・局所構造）では

整合慣性が強く
揺らぎが小さいため

a_R(t) ≈ const

となり膨張は観測されない
```

---

## ■ スケール分離

```text
小スケール：
    NAM支配（膨張なし）

中スケール：
    RDF-GRC（補正あり）

大スケール：
    RDF-CE（膨張顕在化）
```

---

## ■ 観測的帰結

本モジュールが成立する場合、以下が期待される：

```text
1. ハッブル率の微小非一様性
2. 環境依存的膨張率
3. スケール依存的加速
4. バリオン分布との弱い相関
```

---

## ■ 総括

[
d_{ij}(t)=a_R(t)\chi_{ij}
]

[
v_{ij}(t)=H_R(t),d_{ij}(t)
]

[
H_R(t)=\frac{\dot a_R(t)}{a_R(t)}
]

```text
宇宙膨張とは

関係距離の一様倍率更新であり

その起源は
整合慣性と揺らぎの宇宙スケール再配分である
```

---

## ■ 最終定義

```text
宇宙膨張とは

関係更新遅延構造の再配置が
距離として観測される現象であり

必ずしも
空間そのものの実体的拡張を意味しない
```
