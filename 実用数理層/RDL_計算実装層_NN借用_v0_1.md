**借用元との対応表（概観）**

```
NN 概念                 RDL 概念
────────────────────────────────────────
重み行列 W_ij      ↔   M_B^ij（整合慣性ネットワーク内部接続）
損失関数 L         ↔   誤差 E(t)
誤差逆伝播         ↔   dM_B/dt = f(M_B, E, ξ)
順伝播 σ(Wx)       ↔   F = interp(M_B, EFP)
活性化関数 σ       ↔   interp() 内の非線形変換
Hebb 則            ↔   W_ij 学習則（共鳴による結合強化）
ドロップアウト     ↔   ξ 注入（根源的揺らぎ・消去不可）
L2 正則化          ↔   保存則 D[ξ] = 𝒦/||M_B||
層の積み重ね       ↔   M_B の再帰性（M_b1, M_b2 ...）
固有値スペクトル   ↔   散逸行列 A と M_B 固有構造の対応
適応的学習率       ↔   κ(M_B)（自己修正可能性指標）
────────────────────────────────────────
```

**流用できない概念（RDL 固有・NNに存在しない）**

```
跳躍 M_B → M_Δ → M_B'  ：不連続相転移（NNは連続更新のみ）
ξ の残存性              ：消去不可能な根源的揺らぎ（NNのノイズは除去可能）
EFP/F 分離              ：EFP は直接観測不能（NNは input を直接扱う）
```

## ■ 1. dM_B/dt の具体形

### ● 導出の考え方

```
v2.3 の基本式：
    dM_B/dt = f(M_B, E, ξ)     // 関数形が残課題

NN の誤差逆伝播（勾配降下法）：
    dW/dt = η · E · x^T         // 誤差 × 入力の外積で重みを更新

RDL への読み替え：
    E(t) = F(t+Δ) - M_B·F(t)   // 時間差誤差（v2.3）
    F(t) が NN でいう x（入力）に相当
    → E(t) · F(t)^T が更新の駆動項
```

### ● 借用した具体形

```
dM_B/dt = η · E(t) · F(t)^T  -  λ · M_B  +  ξ(t)

各項の意味：
    η · E(t) · F(t)^T  ：誤差×入力の外積（Hebb的更新・整合方向への慣性強化）
    -λ · M_B           ：L2正則化相当（慣性の過強化を防ぐ・保存則との整合）
    ξ(t)               ：根源的揺らぎ注入（v2.3 の ξ 残存性を保証する項）

パラメータ：
    η  ：更新率（= NN の学習率）
    λ  ：正則化強度（D[ξ]=𝒦/||M_B|| の保存則が破断しない範囲で設定）
```

**正則化項と保存則の接続**

```
-λ · M_B は L2正則化として借用したが
RDL では D[ξ] = 𝒦/||M_B|| の保存則と対応する

||M_B|| が大きくなるほど D[ξ] が小さくなる（系の硬直化）
-λ · M_B はその硬直化に対する自動的な抑制として機能する
= 保存則を維持するための構造的正則化として解釈できる
```

**κ(M_B) との接続**

```
κ = 1：通常の更新（上式がそのまま動く）
κ → 0：η → 0 に近づく（自己修正不能・M_B 絶対性）

実装上は η を κ でスケールする：
    dM_B/dt = κ · η · E(t) · F(t)^T  -  λ · M_B  +  ξ(t)
```

### ● 有効範囲と破断条件

```
有効：
    M_B が安定した液体相にある通常時
    E(t) が小〜中程度の連続更新時

破断の兆候：
    H(t) > θ + ξ(t) → 跳躍条件に達した時
    → 本式は使えない・v2.3 の M_B → M_Δ → M_B' に切り替わる
    跳躍後は M_B' を初期値として再起動する

κ → 0 時：
    η が実質ゼロになり更新が止まる
    ξ(t) 項のみが残る
    → 誤差が内部蓄積して大規模崩壊が構造的必然になる（v2.3 記述済み）
```

## ■ 2. interp(M_B, EFP) の具体形

### ● 導出の考え方

```
v2.3 の記述：
    F = interp(M_B, EFP)    // 関数形が残課題

NN の順伝播：
    y = σ(W · x)            // 重みで入力を変換→活性化

RDL への読み替え：
    M_B ↔ W（整合慣性が重みとして機能）
    EFP ↔ x（未解釈入力）
    F   ↔ y（解釈済み出力）
    σ   ↔ 非線形整合変換
```

### ● 借用した具体形（単層版）

```
F(t) = σ( M_B · EFP(t) )

σ の候補：
    tanh(x)    ：[-1, 1] の双方向整合（整合の方向性を保持）
    sigmoid(x) ：[0, 1] の一方向整合（活性化・非活性化の二値的整合）
    ReLU(x)    ：max(0,x)（非整合を切り捨てる・整合方向のみ通す）

RDL 的には tanh が最も概念に近い
    整合（正）と逆整合（負）を連続的に扱える
    M_B がリッチなほど σ の前段が精密になる
```

### ● 多層版（M_B のリッチさへの対応）

```
v2.3 の記述：
    「M_B がリッチなほど EFP から多くを読み取れる」
    「M_B の再帰性：内部に下位 M_bi を含む」

多層版：
    h_1 = σ( M_b1 · EFP )      // 下位境界 b1 による一次解釈
    h_2 = σ( M_b2 · h_1  )     // 下位境界 b2 による二次解釈
    F   = σ( M_B  · h_2  )     // 境界 B による最終解釈

層数 = M_B の再帰深度（境界の入れ子の深さ）
M_B がリッチ = 層が多い・各層の重みが精密
M_B がスカスカ = 単層・EFP がほぼ素通り
```

### ● 有効範囲と破断条件

```
有効：
    EFP の分布が M_B の学習範囲内にある時
    通常の連続更新時

破断の兆候：
    EFP が M_B の解釈可能範囲を大幅に超える入力の時
    → F が異常値を出力し E(t) が急増する
    → H 蓄積が加速 → 跳躍条件に近づく

EFP の認識論的限界（v2.3 より）：
    EFP は直接観測不能・F からの逆算推定に過ぎない
    この実装式も「EFP があると仮定した上での近似」として読む
```

## ■ 3. W_ij 学習則（SILNネットワーク拡張）

### ● 導出の考え方

```
v2.3 の残課題：
    W_{ij} の学習則（共鳴による結合強化）

v2.3 の SILNネットワーク拡張式：
    dM_B^(i)/dt = f(M_B^(i), F^(i) + Σⱼ W_{ij}M_B^(j), ξ^(i))

NN の Hebb 則：
    「共に発火するノードは結合が強まる」
    dW_ij/dt = α · x_i · x_j

RDL の「共鳴による結合強化」への読み替え：
    ノード i と j の M_B が誤差を共有した時に結合が強まる
    = 同じ誤差に同期して応答する構造間の結合が強化される
```

### ● 借用した具体形

```
dW_ij/dt = α · E^(i)(t) · ||M_B^(j)(t)||  -  β · W_ij

各項の意味：
    α · E^(i) · ||M_B^(j)|| ：i の誤差 × j の整合慣性強度（共鳴項）
                               j が強い整合を持つほど i への影響が大きい
    -β · W_ij               ：結合の自然減衰（使われない結合は弱まる）

パラメータ：
    α  ：共鳴感度（どれだけ誤差共有で結合が強まるか）
    β  ：減衰率（使われない結合が消える速度）
```

**跳躍の定着条件との接続（v2.3）**

```
v2.3：
    個体 i の跳躍後 M_B'^(i) が他の構造 j の整合慣性と共鳴
    → より広い B で安定パターンとして定着

W_ij の文脈では：
    跳躍後の M_B'^(i) が M_B^(j) と誤差を共有し続ける
    → dW_ij/dt の共鳴項が持続的に正になる
    → W_ij が強化される
    → 𝒦^(i) が社会 SILN として引き継がれる（v2.3 記述と整合）
```

### ● 有効範囲と破断条件

```
有効：
    ネットワーク内の M_B が比較的均質なスケールにある時
    結合が疎で計算コストが許容範囲内の時

破断の兆候：
    W_ij が全結合に近づく（計算爆発）
    一部の W_ij が極端に大きくなる（ハブ依存・単一障害点）
    → スパース化・閾値カットが必要
    → この判断は構造破壊学が担当
```

## ■ 4. 散逸行列 A の具体形

### ● 導出の考え方

```
v2.3 の記述：
    dH_vec/dt = σ_vec(t) - A · H_vec(t)
    A：散逸行列
    「M_B の得意な方向ほど散逸が速い」

NN のスペクトル分析：
    M_B = V_B Λ_B V_B^{-1}（固有値分解・v2.3）
    各固有方向の散逸速度は固有値の大きさに比例すると仮定
```

### ● 借用した具体形

```
A = V_B · diag(a_1, a_2, ..., a_n) · V_B^{-1}

各対角成分：
    a_k = γ · λ_k     （γ：散逸係数・λ_k：M_B の k 番目の固有値）

解釈：
    固有値 λ_k 大（M_B の強い整合方向）→ a_k 大 → 散逸速度速い
    固有値 λ_k 小（M_B の弱い整合方向）→ a_k 小 → 熱が残りやすい
    = M_B の苦手な方向に熱が蓄積するという v2.3 の記述を形式化
```

## ■ 5. κ(M_B) の計算方法

### ● 導出の考え方

```
v2.3 の記述：
    κ(M_B) ∈ [0, 1]
    κ = 1 : 完全に修正可能
    κ → 0 : 修正不能（M_B 絶対性）

NN の適応的学習率スケジューリング：
    学習が進む（||W|| が大きくなる）ほど学習率を下げる
    = 過学習防止
```

### ● 借用した具体形

```
κ(M_B) = exp( -||M_B|| / M_0 )

または sigmoid 版：
κ(M_B) = 1 / (1 + ||M_B|| / M_0)

パラメータ：
    M_0 ：基準慣性スケール（観測者・スケールに依存する正規化定数）

解釈：
    ||M_B|| が小さい（弱い整合慣性）→ κ ≈ 1 → 更新しやすい
    ||M_B|| が大きい（強い整合慣性）→ κ → 0 → 更新しにくい
    ||M_B|| → ∞ → κ → 0 → M_B 絶対性（v2.3 と整合）

保存則との接続：
    D[ξ] = 𝒦/||M_B|| が小さくなるにつれ κ も小さくなる
    = 揺らぎが抑圧されると同時に自己修正性も低下
    = ξ 内部蓄積と大規模崩壊が構造的に対応
```

## ■ 全体まとめ：実装層の全体式

```
// 通常時（連続更新）
F(t)     = σ( M_B · EFP(t) )                              // interp() 具体形
E(t)     = F(t+Δ) - M_B · F(t)                            // 時間差誤差（v2.3）
σ_vec(t) = E(t)
dH_vec/dt= σ_vec(t) - A · H_vec(t)                        // A = V_B Λ_B^γ V_B^{-1}
H(t)     = ||H_vec(t)||
dM_B/dt  = κ · η · E(t) · F(t)^T  -  λ · M_B  +  ξ(t)   // 更新則 具体形
κ        = exp(-||M_B|| / M_0)                             // 自己修正可能性

// SILNネットワーク
dW_ij/dt = α · E^(i)(t) · ||M_B^(j)(t)||  -  β · W_ij    // W_ij 学習則

// 跳躍条件（v2.3 より・変更なし）
H(t) > θ + ξ(t)  →  M_B → M_Δ → M_B'
```

**v2.3 から変更していないもの**

```
D[ξ] = 𝒦 / ||M_B||     保存則
E(t) = F(t+Δ) - M_B·F(t)  誤差定義
跳躍条件・M_Δ 遷移・熱残留率ρ
RDL-GR 方程式
NAM 方程式
哲学的基底 ξ + B + M_B
```
