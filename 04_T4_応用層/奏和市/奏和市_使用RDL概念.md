# 奏和市：使用RDL概念マップ
*随時更新 ── RDL体系が拡張されるたびに追記する*

---

> 奏和市はRDL応用の最大射程事例である。
> 都市設計・運用・倫理・市民理解のすべてにRDL概念が貫通している。
> 本文書は、どの概念がどこで機能しているかを追跡するための生きた索引である。

---

## T0：哲学的基底層

| 概念 | 奏和市における機能 |
|---|---|
| **限界空間** | プラクシスも全知全能ではない。観測限界の中での運用を前提とする。「完璧な整合」は到達不能として設計に組み込まれている |
| **形而上空間** | 「完璧な幸福」「絶対的正義」は形而上に置き、限界空間内での持続可能な循環を目標とする。プラクシスはこの区別を設計上の原則とする |

---

## T1：Core層

| 概念 | 奏和市における機能 |
|---|---|
| **EFP（素流圧）** | プラクシスが景観・気候・動線・遭遇確率・祭り・仕事負荷などとして都市環境側から配置する。市民は直接観測できない |
| **F（作用解釈）** | 市民がEFPをどのFとして受け取るかは市民のM_Bに依存する。プラクシスはF統計を観測して配置を調整する |
| **E（誤差）** | 適度なEは幸福循環の燃料。過剰なEは熱化・崩壊を招く。プラクシスはE量を都市スケールで管理する |
| **H（熱）** | 市民・地域・文化単位でHを監視し、閾値超過前に分散・緩和する。これが奏和市の中核運用 |
| **M_B（整合慣性）** | 市民個人・集団・文化のM_Bを読んで素流圧配置を最適化する |
| **ξ（揺らぎ）** | 意図的にξを都市に残す。完全整合＝停滞・退屈であるため、安全な揺らぎが幸福循環に必要 |
| **関係保存則** | 都市スケールで||M_B||とD[ξ]のバランスを維持する設計思想 |

---

## T2：運用層

| 概念 | 奏和市における機能 |
|---|---|
| **SILN分解** | 都市・地区・市民・文化・習慣・制度をSILNとして切り出し、B仮設・M_B追従・誤差管理のサイクルで運用する |
| **構造破壊学** | 跳躍の選択的制御。個人成長・芸術的創発は許容し、革命・権力集中・技術的暴発は抑制する。破断条件を事前設計 |
| **汎用階層構造（4層）** | 市民の物理/基層/中核/上層構造を別々に扱う。施策の届く層と届かない層を区別して介入設計する |
| **神経力学的基層構造** | 市民の基層的傾向（探索性・誤差許容度・関係拡張性・危険感度）をパラメータとして読み、素流圧配置を個別最適化する |
| **倫理解釈** | 倫理＝境界維持反応の分布として都市設計に組み込む。強制でなく、境界が自然に維持されやすい環境を作る |
| **創造論** | 適度な誤差・探索勾配を意図的に都市に組み込む。「安全な挑戦」「即時フィードバック」「余白」が設計要件 |
| **幸福モデル** | 都市の究極目標の定義式。幸福＝持続可能な整合循環。超安定社会の罠（勾配消失・退屈・疑似勾配）を知った上で設計 |
| **切替判定** | プラクシスが状況に応じてどの介入モジュールを使うかを判定する手順論 |

## T3：外部接続層

| 概念 | 奏和市における機能 |
|---|---|
| **NAM（ニュートン近似）** | 日常スケールの市民行動・都市動態の低負荷記述 |
| **FluctMOD（揺らぎ）** | 市民の確率的行動・文化の統計的分布・ξの定量的扱い |
| **RDL-GR（一般相対論）** | 都市構造の大スケール・長期的な幾何学的変化の記述 |
| **NN借用** | プラクシスのinterp・M_B更新関数の非線形化。市民行動予測の精緻化 |

---

## T4：応用層

| 概念 | 奏和市における機能 |
|---|---|
| **SFO（空間流向診断）** | 市民の性格断面を三軸で読み、素流圧配置・仕事・環境・遭遇設計に反映する |
| **移動・思考・流れ** | 市民の身体・移動・思考・感情を同一ダイナミクスとして扱い、都市空間の動線設計に接続する |

---

## T5：メタ層

| 概念 | 奏和市における機能 |
|---|---|
| **跳躍考察（網絡化）** | 都市スケールの跳躍（文化変動・制度再編・集団的覚醒）を網絡化として扱い、M_Δ相の管理をプラクシスの高度機能とする |
| **網絡化演算子N** | N(U, F, M, H_dist)→M' として都市レベルの構造遷移を記述・予測する |
