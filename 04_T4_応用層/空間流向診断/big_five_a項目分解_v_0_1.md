# Big Five：A（協調性）項目分解 v0.1

## 位置づけ

本書は Big Five の A（Agreeableness／協調性）を、そのまま「優しさ」「良い人度」「性格の良さ」として採用するための文書ではない。

A は、SFO（空間流向診断）では以下のように再定義する。

```text
A = 他者境界への流入制御
```

または、

```text
A = 他者をどこまで自己側に入れ、どこで相手の境界前に止まれるか
```

さらに基層構造寄りに言えば、

```text
A = 他者空間における OXT結合範囲 と NA警戒閾値 の合成表現
```

である。

つまり、A は単なる「他人に優しいか」ではない。

```text
他者へ流れる力
他者を危険化しない力
他者の痛みを処理対象に入れる力
他者境界を侵害しない力
必要なときに境界を張る力
内側化した対象を守る力
```

これらが混ざった、かなり状況依存性の強い因子である。

---

## A が難しい理由

Big Five の A は、一般には以下のような項目で測られる。

```text
人を信頼する
人に親切にする
人を助ける
他人に思いやりがある
対立を避ける
他人を尊重する
謙虚である
人を利用しない
```

しかし、これらはすべて「相手が誰か」「場が安全か」「外敵がいるか」「その相手が自分の内側か外側か」によって大きく変わる。

標準的な A スコアは、これらの対象差・状況差をかなり丸めてしまう。

したがって SFO では、A を全体スコアではなく、

```text
A_target_map
```

として扱う。

---

## 基本定義

### A の圧縮定義

```text
Aとは、他者空間に対して、
接続・配慮・非侵害・警戒・防衛をどのように配分するかの傾向である。
```

さらに短く、

```text
A = 他者境界への流入制御
```

相談向けには、

```text
A = 人との距離を、どれくらい柔らかく扱うか
```

と翻訳できる。

ただし、「柔らかいほど良い」ではない。

```text
流れない
→ 無関心・冷淡

流れすぎる
→ 侵入・支配・過干渉

流れつつ止まれる
→ 尊重・協調・信頼
```

---

## A の基層構造：OXT + NA

A は OXT 単体ではなく、OXT と NA の合成として見る。

```text
OXT：自他境界を広げる／他者を自己側に取り込む
NA ：危険検知・警戒・防衛モードを立ち上げる
```

したがって、A は以下のように読める。

```text
A = 他者を自己側に入れる力 − 他者を危険化する力
```

より正確には、

```text
A_target = OXT_target - NA_threat_target
```

これは厳密な数式ではなく、読みの形である。

---

## OXT × NA の4象限

### 1. OXT高 × NA低

```text
相手を自己側に入れやすく、危険化しにくい
```

出方：

```text
信頼的
協力的
援助的
受容的
高Aとして出やすい
```

破断：

```text
騙されやすい
境界が弱い
他者のHを引き受けすぎる
断れない
```

---

### 2. OXT低 × NA低

```text
相手にあまり流れないが、敵意も強くない
```

出方：

```text
淡白
距離がある
個人主義的
干渉しない
```

破断：

```text
冷たいと見られる
相手の困りごとに反応しにくい
関係が薄くなりやすい
```

---

### 3. OXT低 × NA高

```text
相手を自己側に入れにくく、警戒も強い
```

出方：

```text
疑い深い
防衛的
対立的
他者を危険源として読みやすい
```

破断：

```text
孤立
攻撃的解釈
関係構築の困難
敵味方分離
```

---

### 4. OXT高 × NA高

```text
内側化した対象には強く流れるが、外側には強く警戒する
```

出方：

```text
仲間には厚い
家族や味方を守る
外敵に強い
危機時には頼もしい
```

破断：

```text
排他的
過干渉
支配的
外部を敵認定しやすい
相手の自律を許しにくい
```

この象限は特に重要である。

OXT が高いことは、常に協調性が高いことを意味しない。
OXT が高いほど、内側化した対象を守る反応が強くなり、外側への NA が高い場合には、排他性・防衛性・攻撃性として出る可能性がある。

# 尊重と支配

A 分解における中心論点は、

```text
相手を内側に入れること
```

と、

```text
相手を尊重すること
```

は同じではない、という点である。

## 尊重とは何か

尊重とは、相手を内側に入れないことではない。

```text
尊重 = 相手との接続を保ちつつ、相手の独立Bを残すこと
```

つまり、

```text
あなたは大事。でもあなたはあなた。
```

という状態である。

### 尊重的協調

```text
OXT結合あり
+
相手Bの独立性を認める
+
NAによる支配・防衛の過剰起動が低い
```

出方：

```text
配慮
信頼
非侵害
相手の選択を待てる
相手の境界前で止まれる
```

---

## 支配的内側化

一方、相手を自己側に入れていても、相手の独立Bを認めない場合がある。

```text
支配的内側化 = 相手を大事にするが、相手の自律境界を認めない
```

これは以下のように出る。

```text
あなたは大事。だから私の内側にいて。
```

流れ：

```text
OXT高
→ 相手を自分側に入れる

ただし相手の独立Bを認めない
→ 相手を「自分の一部」として扱う

外部からその相手が攻撃される
→ 自分が攻撃されたように防衛する

相手自身が自分の意図と違う動きをする
→ 自分の一部が勝手に動いたように感じる
→ 怒り・支配・監視・矯正が出る
```

表面上は、

```text
守っている
心配している
尽くしている
助けている
味方している
```

ように見えることがある。

しかし内部構造は、

```text
相手のため
```

ではなく、

```text
自分の内側の秩序を守るため
```

になっている場合がある。

---

## A の二種類の内側化

```text
A_respectful_inclusion
= 相手を内側に入れつつ、相手の自律境界も認める

A_possessive_inclusion
= 相手を内側に入れるが、相手の自律境界を認めない
```

この違いを見ないと、A高を「良い人」と誤読する。

# 状況依存性

A は、状況・環境依存性が非常に強い。

同じ人でも、以下の条件で A の出方が変わる。

```text
安全な環境
敵がいる環境
内集団が攻撃されている環境
自分が疲れている環境
相手が弱者として見えている環境
相手が侵入者として見えている環境
```

特に、

```text
OXT高 × NA高
```

の人は、外敵がいる状況では非常に頼もしく見えることがある。

```text
守ってくれる
味方には厚い
危険を察知する
戦える
境界を張れる
頼もしい
```

しかし、平時や安全な関係の中では、同じ構造が、

```text
過干渉
支配的
疑い深い
相手の自律を許しにくい
外部を敵認定しやすい
味方／敵の線引きが強い
```

として出る可能性がある。

つまり、A は人格の優しさではなく、

```text
その場で必要な境界の硬さと、その人の他者境界制御が合っているか
```

で評価が変わる。

---

## A の相状態

### 平時A

```text
相手への配慮・尊重・信頼として出る
```

### 危機時A

```text
内側化した対象の防衛・外部への警戒として出る
```

### 過負荷時A

```text
支配・束縛・疑念・敵味方分離として出る
```

# 対象別 A

標準的な Big Five の A は、かなり「他人一般」に対する傾向として測られる。

しかし実際には、A は対象依存である。

```text
A_family      ：家族へのA
A_friend      ：友人へのA
A_partner     ：恋人・伴侶へのA
A_child       ：子ども・保護対象へのA
A_stranger    ：知らない人へのA
A_enemy       ：敵対者へのA
A_weak        ：弱者・困窮者へのA
A_authority   ：権力者・上位者へのA
A_in_group    ：内集団へのA
A_out_group   ：外集団へのA
A_value_near  ：価値観が近い相手へのA
A_value_far   ：価値観が遠い相手へのA
```

例：

```text
家族には高A
知らない人には低A

弱者には高A
権力者には低A

仲間には高A
敵対者には低A

価値観が近い人には高A
価値観が遠い人には低A
```

これらを全体Aスコアに丸めると、重要な構造が見えなくなる。

---

# 価値観の多様性との関係

多様な価値観を持つことは、Aを高める可能性がある。

流れとしては、

```text
多様な価値観を持つ
↓
他者の行動をすぐ悪意として読まない
↓
Trust が上がる

違う考えを即座に攻撃対象にしない
↓
Respectfulness / Cooperation が上がる

相手の背景を想像しやすい
↓
Compassion / Sympathy が上がる
```

ただし、価値観の多様性とAは同一ではない。

```text
価値観の多様性 = O_value
他者への流れやすさ = A
```

多様な価値観を持っていても、

```text
理解はするが同調はしない
相手の背景は分かるが、自分の境界は譲らない
```

という場合、Aスコアは高く出ない可能性がある。

# 公式的なAファセットのSFO翻訳

Big Five系のAは、尺度によって少し異なるが、代表的には以下のようなファセットを含む。

```text
Trust        ：信頼
Compassion   ：思いやり
Respect      ：尊重
Altruism     ：利他性
Cooperation  ：協力・対立回避
Modesty      ：謙虚さ
Morality     ：率直さ・他者利用の少なさ
Sympathy     ：同情・柔らかさ
```

SFO翻訳は以下。

---

## Trust：信頼

```text
他者意図空間への警戒感度
```

高い場合：

```text
他者を善意寄りに読む
他者を危険源として見にくい
接続開始が容易
```

低い場合：

```text
他者の隠れた動機を疑う
危険化が速い
接続前に防衛が立つ
```

SFO成分：

```text
NA_threat_target
OXT_opening
```

---

## Compassion / Sympathy：思いやり・同情

```text
他者苦痛空間への感度
```

高い場合：

```text
他者の痛みが自分の処理対象に入る
困っている人に反応しやすい
助けたいという流れが立つ
```

低い場合：

```text
他者の痛みを外部情報として処理しやすい
巻き込まれにくい
距離を保ちやすい
```

SFO成分：

```text
OXT_pain_coupling
A_help_response
```

---

## Respectfulness：尊重

```text
相手の境界を侵害しない力
```

高い場合：

```text
相手の独立Bを認める
強制しにくい
相手の選択を待てる
```

低い場合：

```text
相手Bを越えやすい
押しつけやすい
支配的に見えやすい
```

SFO成分：

```text
B_other_recognition
non_intrusion_control
```

---

## Altruism：利他性

```text
他者支援空間への流れやすさ
```

高い場合：

```text
他者の困りごとに行動が流れる
助けることが自然に起きる
```

低い場合：

```text
他者の問題を自分の処理対象に入れにくい
自分の範囲を明確に保つ
```

SFO成分：

```text
A_help_flow
B_self_expansion
```

---

## Cooperation / Compliance：協力・対立回避

```text
対立空間での衝突抑制
```

高い場合：

```text
衝突より調整へ流れやすい
対立のHを避けやすい
妥協や譲歩が起きやすい
```

低い場合：

```text
必要なら衝突できる
譲歩しにくい
境界侵害に対して押し返せる
```

SFO成分：

```text
conflict_H_avoidance
boundary_defense
```

---

## Modesty：謙虚さ

```text
自己優位空間の抑制
```

高い場合：

```text
自分を大きく見せにくい
場の中で自己主張を抑えやすい
```

低い場合：

```text
自己優位を出しやすい
主張・競争・地位表示が強く出る
```

SFO成分：

```text
status_expression_control
self_priority_display
```

---

## Morality / Straightforwardness：率直さ・他者利用の少なさ

```text
他者操作・搾取空間での抑制
```

高い場合：

```text
他者を道具化しにくい
嘘や操作への抵抗がある
関係をフェアに保とうとする
```

低い場合：

```text
状況によって他者を操作対象として扱いやすい
目的達成のために関係を使う
```

SFO成分：

```text
manipulation_inhibition
fairness_alignment
```

# A項目分解

以下の項目は、Big Five 系質問紙に頻出する A 項目を、SFO用に一般化して整理したものである。

実際の質問紙によって文言は異なるため、ここでは特定尺度の逐語項目ではなく、代表的な意味単位として扱う。

---

## 1. 人を信頼する

```text
項目：人を信頼する
表層意味：他者を基本的に善意寄りに読む
空間タグ：他者意図空間、関係空間、安全空間
階層タグ：基層構造 + 中核構造
入力タイプ：他者の言動、距離、未知の相手
反応タイプ：低NA、接続開始、OXT開放
測っているSFO成分：A_trust、NA_threat_target、OXT_opening
```

### 高スコア時の読み

他者を危険源として読む初期設定が弱い。
関係開始が比較的容易。

### 低スコア時の読み

他者の隠れた意図や危険性を検出しやすい。
防衛的だが、搾取や危険への感度は高い。

### 破断条件

```text
信頼する = 良い人、と読む。
疑う = 悪い人、と読む。
実際に危険な環境を、本人の疑い深さに還元する。
```

### 相談翻訳

> 人を信じやすいかどうかは、優しさというより、他者を最初に危険として読むか、接続可能な相手として読むかの違いかもしれません。

---

## 2. 人に思いやりがある

```text
項目：人に思いやりがある
表層意味：他者の状態を気にかける
空間タグ：他者苦痛空間、関係空間、ケア空間
階層タグ：基層構造 + 中核構造
入力タイプ：相手の困りごと、苦痛、弱さ、表情
反応タイプ：OXT結合、支援流向、H共有
測っているSFO成分：A_compassion、OXT_pain_coupling、A_help_flow
```

### 高スコア時の読み

他者の痛みや困りごとが、自分の中の処理対象に入りやすい。

### 低スコア時の読み

他者の苦痛と自己の処理範囲が分離しやすい。
冷たいとは限らず、巻き込まれにくいとも言える。

### 破断条件

```text
思いやり高 = 健全、と読む。
思いやり低 = 冷酷、と読む。
他者Hを引き受けすぎるリスクを見落とす。
```

### 相談翻訳

> 相手のしんどさが自分の中に入りやすいタイプかもしれません。優しさにもなりますが、引き受けすぎると自分側の余白が減りやすいです。

---

## 3. 人を助けるのが好きである

```text
項目：人を助けるのが好きである
表層意味：他者支援に行動が流れやすい
空間タグ：支援空間、関係空間、役割空間
階層タグ：中核構造 + 基層構造
入力タイプ：困っている相手、依頼、弱者信号
反応タイプ：援助行動、役割取得、OXT拡張
測っているSFO成分：A_altruism、A_help_response、B_self_expansion
```

### 高スコア時の読み

他者支援が自然な行動流路になりやすい。

### 低スコア時の読み

他者の問題と自分の問題を分離しやすい。
自分の範囲を保ちやすい。

### 破断条件

```text
助ける = 善、と読む。
助けない = 悪、と読む。
支援が支配や自己価値維持になっている可能性を見落とす。
```

### 相談翻訳

> 助けたい気持ちが強い場合、それが相手のためだけでなく、自分の役割や安心にもつながっていることがあります。

---

## 4. 人と対立するのを避ける

```text
項目：人と対立するのを避ける
表層意味：衝突より調整・譲歩に流れやすい
空間タグ：対立空間、境界空間、関係維持空間
階層タグ：基層構造 + 中核構造
入力タイプ：意見差、拒絶、衝突、相手の怒り
反応タイプ：H回避、譲歩、沈黙、調整
測っているSFO成分：A_cooperation、conflict_H_avoidance、boundary_softening
```

### 高スコア時の読み

対立によるHを避け、関係維持へ流れやすい。

### 低スコア時の読み

必要なら対立できる。
境界侵害に対して押し返す力がある。

### 破断条件

```text
対立回避 = 優しい、と読む。
対立可能 = 攻撃的、と読む。
対立回避による自己境界の消耗を見落とす。
```

### 相談翻訳

> 対立を避ける力は関係を守る一方で、自分の線引きを後回しにしやすい面もあります。

---

## 5. 他人を尊重する

```text
項目：他人を尊重する
表層意味：相手の独立性・選択・境界を認める
空間タグ：他者境界空間、関係空間、価値観空間
階層タグ：中核構造 + 上層構造
入力タイプ：相手の意見、拒否、選択、距離
反応タイプ：非侵害、待機、境界認識
測っているSFO成分：A_respectfulness、B_other_recognition、non_intrusion_control
```

### 高スコア時の読み

相手を大事にしつつ、相手の独立Bを残しやすい。

### 低スコア時の読み

相手の境界を越えやすい。
自分の正しさ・心配・保護欲が、相手の選択を上書きする場合がある。

### 破断条件

```text
尊重 = 無関心、と誤読する。
心配・保護・支援を尊重と混同する。
```

### 相談翻訳

> 尊重とは、相手を大事にしながらも「相手は相手」として残せることです。

---

## 6. 謙虚である

```text
項目：謙虚である
表層意味：自己優位や地位主張を前面に出しにくい
空間タグ：地位空間、自己評価空間、集団空間
階層タグ：中核構造 + 上層構造
入力タイプ：評価、称賛、競争、比較
反応タイプ：自己抑制、地位表示抑制、場維持
測っているSFO成分：A_modesty、status_expression_control
```

### 高スコア時の読み

自己優位を抑え、場の摩擦を減らしやすい。

### 低スコア時の読み
