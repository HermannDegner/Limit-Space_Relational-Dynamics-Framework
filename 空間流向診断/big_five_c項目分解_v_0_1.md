# Big Five：C（誠実性）項目分解 v0.1

## 位置づけ

本書は Big Five の C（Conscientiousness／誠実性）を、そのまま「真面目さ」「有能さ」「人としての信頼度」として採用するための文書ではない。

C は、SFO（空間流向診断）では以下のように再定義する。

```text
C = 空間別慣性維持性
```

または、

```text
C = ある境界Bで成立した整合慣性M_Bを、時間をまたいで維持・反復・修復・完了させる傾向
```

さらに短く言えば、

```text
C = 流れを保つ力
```

である。

従来語でいう、

```text
誠実性
勤勉性
責任感
計画性
自己統制
秩序性
慎重さ
```

は、SFO/RDF的にはおおむね、

```text
一度成立した流れを維持できるか
約束・手順・役割を反復できるか
途中で崩さず完了まで運べるか
既存のM_Bを安定運用できるか
```

に集約される。

したがって、C は「高いほど良い人格」ではなく、

```text
どの空間で、どの慣性を、どの程度維持できるか
```

を見るための因子である。

---

## C が難しい理由

C は一見すると、社会的に望ましい指標に見えやすい。

```text
片づける
計画する
怠けない
始める
終わらせる
約束を守る
責任を果たす
慎重に判断する
```

これらは学校・職場・制度空間では高く評価されやすい。

しかし、SFO的には C は単なる美徳ではない。

C高は、

```text
信頼される
続けられる
約束を守れる
秩序を保てる
実行できる
ミスを減らせる
```

という強みになる一方で、

```text
硬直する
予定外に弱い
遊びが消える
自己罰化する
他人にも高基準を要求する
中途半端や未確定を許せない
必要精度を超えて検査し続け、出力できない
```

という破断も持つ。

C低も単なる欠陥ではない。

C低には、

```text
柔軟
即興に強い
変化に乗れる
未確定を開いたまま扱える
既存ルールに縛られにくい
```

という強みがある。

ただし破断すると、

```text
始められない
続かない
締切に弱い
散らかる
責任が不安定
実行流路が閉じない
外部足場がないと崩れる
```

となる。

## 基本定義

### C の圧縮定義

```text
Cとは、特定空間で成立したM_Bを、
反復・保存・実行・責任化によって維持する傾向である。
```

さらに短く、

```text
C = 慣性維持性
```

SFO用には、

```text
C = 空間別慣性維持性
```

相談向けには、

```text
C = 一度決めた流れを保つ力
```

と翻訳できる。

---

## C の基本分解

C は最低でも以下の成分に分かれる。

```text
C1：開始力
C2：継続力
C3：完了力
C4：秩序維持力
C5：責任保持力
C6：慎重判断力
C7：自己統制力
C8：達成志向
C9：耐久検査力
C10：修復力
C11：過剰検査による出力停止
```

### C1：開始力

流れを始める力。

タスク・習慣・約束・制作・学習などに対して、最初の実行流路を開けるか。

高い場合、やるべきことに入りやすい。
低い場合、始める前の抵抗が大きい。

---

### C2：継続力

始めた流れを維持する力。

一度立ち上がったM_Bを、日・週・月・年の時間スケールで保てるか。

高い場合、反復・習慣化・長期努力に強い。
低い場合、初動はあっても途中で流れが切れやすい。

---

### C3：完了力

流れを閉じる力。

途中までではなく、最後まで運び、出力として確定させられるか。

高い場合、成果物・提出物・完了状態を作りやすい。
低い場合、着手や探索はできても、完成・提出・固定が難しい。

---

### C4：秩序維持力

物理・情報・予定・手順などを整った状態に保つ力。

高い場合、整理整頓・手順化・予定管理に強い。
低い場合、散らかり・混沌・未分類状態が残りやすい。

ただし、低いから処理能力が低いとは限らない。
一時的な混沌を許容することで、探索や再編が進む場合もある。

---

### C5：責任保持力

約束・義務・役割・期待を時間をまたいで保持する力。

高い場合、信頼されやすい。
低い場合、役割や約束が流れやすい。

ただし、責任保持力が高すぎると、自分が背負うべきでないものまで保持してしまう。

---

### C6：慎重判断力

衝動的に閉じず、実行前にリスクや破断条件を確認する力。

高い場合、ミス・事故・損失を減らしやすい。
低い場合、即断即決・試行錯誤に強いが、失敗や見落としが増える。

---

### C7：自己統制力

短期的な誘惑・衝動・気分変動に流されず、維持したいM_Bに戻る力。

高い場合、習慣維持・遅延報酬・規律に強い。
低い場合、その場の勾配に流れやすいが、柔軟さや即興性にもつながる。

---

### C8：達成志向

高い基準・目標・評価へ向かって努力する力。

高い場合、成果追求・技能向上・競争・自己改善に流れやすい。
低い場合、達成競争への引力が弱く、生活や関係の別の価値を優先しやすい。

---

### C9：耐久検査力

構造破壊学的に、出力前にその構造がどこまで耐えるか検査する力。

```text
この構造は壊れないか？
この出力は耐えるか？
この手順は破綻しないか？
この精度で十分か？
```

を確認する傾向である。

高い場合、品質・安全性・信頼性が上がる。
低い場合、出力は速いが、破断条件の確認が甘くなりやすい。

---

### C10：修復力

崩れた流れを戻す力。

予定が崩れた、習慣が途切れた、失敗した、散らかったときに、再び流路を修復できるか。

高い場合、崩れても戻れる。
低い場合、一度崩れるとそのまま流れが失われやすい。

---

### C11：過剰検査による出力停止

Cの破断形。

必要精度を超えて破壊検査し続けることで、出力が固定されない状態。

```text
作る
↓
検査する
↓
破れを見つける
↓
直す
↓
また破れを見つける
↓
出さない
```

これは「高品質」ではなく、目的境界Bを超えた精度要求によって、M_Bが安定化せず、出力化されない状態である。

# 空間別 C

C は全体値ではなく、空間ごとの分布として扱う。

```text
C_work      ：仕事・職務空間の慣性維持性
C_life      ：生活・家事空間の慣性維持性
C_body      ：睡眠・運動・健康空間の慣性維持性
C_relation  ：人間関係・約束空間の慣性維持性
C_concept   ：概念・理論空間の慣性維持性
C_creation  ：創作・制作空間の慣性維持性
C_money     ：経済・支出・管理空間の慣性維持性
C_institution：制度・手続き空間の慣性維持性
C_environment：物理環境・整理空間の慣性維持性
```

例：

```text
仕事空間ではC高
生活空間ではC低

創作の慣性は維持できる
事務処理の慣性は維持できない

人間関係の約束は守る
物理空間の整理は保てない

概念体系は反復更新できる
身体リズムは崩れやすい
```

これらを C_total に丸めると、重要な構造が見えなくなる。

---

# Oとの関係

C は、O と対になるような関係保存系として扱える。

```text
O = 関係を開く力
C = 関係を保存する力
```

または、

```text
O = M_Bを揺らす／再編する
C = M_Bを維持する／反復固定する
```

ただし、O と C は単純な対立ではない。

創造過程では、両方が必要である。

```text
O：未線形を拾う
O：仮線形化する
C：その線を保持する
C：反復可能にする
C：作品・理論・制度・習慣として固定する
```

Oだけ高い場合：

```text
可能性は開くが、形として残りにくい
```

Cだけ高い場合：

```text
形は残るが、新しい流れが入りにくい
```

O + C が噛み合う場合：

```text
新しい線を引き、それを保存・反復・共有可能にする
```

循環としては、

```text
Oで新しい線を引く
↓
Cでその線を保つ
↓
Cが強まりすぎると硬直する
↓
Oで再び開く
```

となる。

# 構造破壊学的アプローチ差

Cには、構造破壊学的な検査傾向も含まれる。

Cを「関係保存力」と見るなら、保存する前に、

```text
この構造は壊れないか？
この出力は耐えるか？
この手順は破綻しないか？
この精度で十分か？
```

を検査する流れが入る。

これは品質や信頼性を高める。

しかし、必要精度を超えて破壊検査し続けると、出力が固定されない。

例：

```text
目標Bで必要な精度：80
本人が要求する精度：98
実際の出力可能精度：85
```

この場合、目的には十分なのに、本人の内部基準では不合格になる。

すると、

```text
作る
↓
検査する
↓
破れを見つける
↓
直す
↓
また破れを見つける
↓
出さない
```

になる。

これは **Cの破壊検査が出力固定を阻害している状態** である。

---

## Cの過剰破断

```text
Cの過剰破断：
壊れない形にしようとして、形にする前に壊し続ける。
```

構造破壊学的には、検査の目的は「あらゆる境界で壊れないもの」を作ることではない。

目的は、

```text
どの境界Bで
どの用途なら
どこまで耐えるか
```

を知ることである。

成熟したCは、

```text
十分に検査する
ただし、目的Bに対して十分なら出す
```

となる。

破断したCは、

```text
あらゆるBで壊れないことを求める
↓
そんなものは作れない
↓
出力できない
```

となる。

# C項目分解

以下の項目は、Big Five 系質問紙に頻出する C 項目を、SFO用に一般化して整理したものである。

実際の質問紙によって文言は異なるため、ここでは特定尺度の逐語項目ではなく、代表的な意味単位として扱う。

---

## 1. 整理整頓が得意である

```text
項目：整理整頓が得意である
表層意味：物理・情報環境を秩序化しやすい
空間タグ：物理環境空間、情報空間、生活空間
階層タグ：物理構造 + 中核構造
入力タイプ：物、書類、予定、情報、配置
反応タイプ：分類、配置、整列、秩序維持
測っているSFO成分：C_order、C_environment、M_B_environment_maintenance
```

### 高スコア時の読み

環境を整えることで、安定した実行流路を保ちやすい。

### 低スコア時の読み

環境が未整理でも動ける、または整理の優先度が低い。
ただし、散らかりがHや実行コストになる場合もある。

### 破断条件

```text
整理整頓 = 有能、と読む。
散らかり = 無能、と読む。
探索中の混沌と、単なる管理不能を混同する。
```

### 相談翻訳

> 整理できるかどうかは、性格の良し悪しではなく、物や情報の流れをどれくらい固定しておきたいかに近いです。

---

## 2. 計画的である

```text
項目：計画的である
表層意味：未来の流れを事前に線形化しやすい
空間タグ：未来空間、タスク空間、制度空間
階層タグ：中核構造 + 上層構造
入力タイプ：予定、期限、目標、リスク
反応タイプ：手順化、順序化、資源配分、事前固定
測っているSFO成分：C_plan、C_verify、future_M_B_projection
```

### 高スコア時の読み

未来の不確定性を、実行可能な線形流路として事前に閉じやすい。

### 低スコア時の読み

未来を開いたまま扱いやすい。
即興や状況対応に向くが、締切や段取りが不安定になりやすい。

### 破断条件

```text
計画的 = 優秀、と読む。
非計画的 = だらしない、と読む。
計画による硬直と、柔軟な即応性を見落とす。
```

### 相談翻訳

> 計画性は、未来をどれくらい先に線形化して安心したいかの違いとして見られます。

---

## 3. 怠けずに取り組む

```text
項目：怠けずに取り組む
表層意味：短期的な楽さより、維持した流路を優先する
空間タグ：タスク空間、達成空間、自己統制空間
階層タグ：基層構造 + 中核構造
入力タイプ：誘惑、疲労、退屈、課題
反応タイプ：自己統制、流路維持、遅延報酬選択
測っているSFO成分：C_self_control、C_continue、C_achievement
```

### 高スコア時の読み

気分や誘惑に流されず、決めた流れを保ちやすい。

### 低スコア時の読み

短期勾配に流れやすい。
ただし、疲労・意味不足・環境不適合を「怠け」と誤読しないこと。

### 破断条件

```text
怠けない = 健全、と読む。
怠ける = 意志が弱い、と読む。
身体疲労・意味喪失・報酬設計不全を性格化する。
```

### 相談翻訳

> 続かない時は、意志の弱さだけでなく、その流れを支える意味・報酬・環境足場が足りていない可能性があります。

---

## 4. 始めたことを最後までやる

```text
項目：始めたことを最後までやる
表層意味：開始した流路を完了まで閉じる
空間タグ：タスク空間、制作空間、責任空間
階層タグ：中核構造
入力タイプ：未完了物、締切、途中成果、飽き
反応タイプ：継続、完了、出力固定
測っているSFO成分：C_continue、C_finish、C_commit
```

### 高スコア時の読み

未完了状態を残しにくく、完了へ流れを閉じやすい。

### 低スコア時の読み

探索はできるが、完了・提出・固定が難しい場合がある。
別の足場や締切設計が必要。

### 破断条件

```text
完了する = 良い、とだけ読む。
完了できない = 欠陥、と読む。
途中で開いた可能性を捨てられない状態を見落とす。
```

### 相談翻訳

> 始める力と、最後まで閉じる力は別です。途中までは動けるのに出せない場合、完了の境界設定が合っていないのかもしれません。

---

## 5. 責任感がある

```text
項目：責任感がある
表層意味：約束・役割・期待を時間をまたいで保持する
空間タグ：役割空間、制度空間、人間関係空間
階層タグ：中核構造
入力タイプ：約束、義務、期待、信頼、依頼
反応タイプ：役割保持、自己拘束、信頼維持
測っているSFO成分：C_commit、role_M_B_maintenance、Aとの混合
```

### 高スコア時の読み

約束や役割を自分のM_Bとして保持しやすい。
人から信頼されやすい。

### 低スコア時の読み

役割や義務を自己側に固定しにくい。
自由度は高いが、信頼構築に課題が出やすい。

### 破断条件

```text
責任感 = 善、と読む。
責任感の低さ = 悪、と読む。
背負いすぎや、他者責任の取り込みを見落とす。
```

### 相談翻訳

> 責任感が強い場合、約束を守る力になりますが、自分が背負わなくていいものまで保持してしまうことがあります。

# C項目の空間タグ集計

現時点の仮説では、Big Five の C 項目は以下の空間に偏りやすい。

```text
物理・環境空間：整理整頓、片づけ、秩序
タスク・実行空間：開始、継続、完了、生産性
役割・制度空間：責任、義務、約束、信頼
達成・評価空間：高目標、努力、成果、自己基準
判断・リスク空間：慎重さ、衝動抑制、検査
自己統制空間：誘惑抑制、遅延報酬、習慣維持
```

したがって、Big Five の C は、単なる「誠実性」ではなく、主に以下の合成スコアとして読める。

```text
C_bigfive ≒
C_order
+ C_productiveness
+ C_responsibility
+ C_self_control
+ C_achievement
+ C_cautiousness
+ C_efficacy
```

ただし、創作空間・概念空間・関係空間・身体空間・遊び空間などでは、Cの出方が大きく異なる可能性がある。

# Cの破断条件

```text
C高 = 有能・善、と読む
C低 = 怠惰・無責任、と読む

計画性と硬直を区別しない
C_totalで見て空間別Cを見落とす
```

Cは人格の優秀さではない。

# 相談プロトコル用翻訳

## C高を伝える場合

悪い例：

> あなたは誠実性が高いです。

良い例：

> 一度決めた流れや約束をかなり保とうとするタイプに見えます。信頼されやすい強みですが、崩れたときに自分を責めやすかったり、必要以上の精度を求めて出せなくなることもありそうです。

## C低を伝える場合

悪い例：

> あなたは誠実性が低いです。

良い例：

> 流れを固定して続けるより、その場の変化や興味に反応しやすいタイプかもしれません。柔軟さはありますが、締切や習慣は外部足場を作った方が安定しそうです。

## 空間別に伝える場合

```text
仕事の約束はかなり守れる一方で、生活リズムや部屋の整理は同じようには維持できないのかもしれません。
```

```text
創作では毎日続けられるのに、事務処理になると急に流れが切れるタイプかもしれません。
```

```text
概念体系を保つCは高いですが、身体リズムを保つCは低めに見えます。
```

## 過剰検査を扱う場合

```text
今は品質意識が高いというより、目的に必要な精度を超えて検査し続けて、出す前に止まっている可能性があります。
```

```text
完璧に壊れない形を作るより、「この用途なら十分耐える」という境界を決めた方が出力しやすそうです。
```

# 観測プローブ

Cを見るには、単に「真面目ですか？」ではなく、どの空間で慣性を維持できるかを聞く。

```text
どの作業なら自然に続きますか？
どの作業はすぐ流れが切れますか？
始めるのが苦手ですか、続けるのが苦手ですか、終わらせるのが苦手ですか？
約束や責任はどの範囲まで自分の中に残りますか？
部屋・予定・情報・人間関係・仕事で、どれが一番整えやすいですか？
出す前にどれくらい検査しますか？
目的には十分でも、自分の中で不十分に感じて出せないことがありますか？
崩れた習慣を戻すのは得意ですか？
計画が崩れたとき、修復しますか、放棄しますか、別ルートに切り替えますか？
```

見るべきは、

```text
どの空間で流れを保てるか
どの空間で流れが切れるか

どの精度まで検査するか
いつ出力できるか
どの責任を背負いすぎるか
```

である。

# SFOへの移植

Cは、SFOでは以下のように扱う。

```text
C = 空間別慣性維持性マップ
```

### 空間軸

どの空間のM_Bを維持できるか。
どの空間のM_Bは維持しにくいか。

### 階層軸

主に中核構造。
ただし、物理整理は物理構造、自己統制は基層構造、達成志向や理想基準は上層構造にも出る。

### 操作軸

高Cは以下に流れやすい。

```text
守る
続ける
閉じる
整える
検査する
修復する
責任化する
```

低Cまたは未固定状態では以下に流れやすい。

```text
開く
流す
即興する
逸れる
放置する
別ルートに移る
```

成熟したCでは以下が重要。

```text
目的Bに対して十分なら出す
```

破断したCでは以下が問題になる。

```text
あらゆるBで壊れないことを求めて出せない
```

# 暫定結論

```text
Big Five の C は、単なる誠実性ではない。

それは、質問紙が選んだ制度・実行・責任・秩序空間群における、
開始・継続・完了・秩序維持・責任保持・自己統制・慎重検査・達成志向の合成スコアである。
```

さらにSFO的に言えば、

```text
C = 空間別慣性維持性マップ
```

である。

最終的には、C高／C低ではなく、

```text
どの空間の慣性を維持できるか
どの空間の慣性は維持できないか
開始・継続・完了のどこで止まるか
どの精度まで耐久検査するか
目的Bに対して十分なら出せるか
責任をどこまで保持するか
崩れた流れを修復できるか
```

を読む。

Cは人格の優秀さではなく、

```text
成立した流れを保つ／閉じる／反復する／責任化する
```

力学である。
