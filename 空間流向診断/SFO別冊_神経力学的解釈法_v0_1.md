## ■ 2. 流用しない概念

```text
神経力学系には強力な原理があるが、SFO の限界空間前提と
相性が悪いため、以下は借用しない。

借用しない：
    ・Free Energy Principle の単一原理化
      （生命が自由エネルギーを最小化するという単一基底）
      → SFO の「絶対前提を持たない」原則と矛盾
      → 部分概念（生成モデル・予測誤差）は借りるが、
        最上位原理としては採用しない

    ・完全可観測前提
      （状態が外部から完全に測れるという仮定）
      → SFO の「観測者依存」と矛盾

    ・パラメータの真値存在前提
      （θ_i の正解値が存在するという仮定）
      → SFO の「真理ではなく有効範囲」原則と矛盾
      → パラメータは観測者ごとの仮設値として扱う

    ・平衡状態の特権化
      （定常解を特別扱いする姿勢）
      → SFO は動態そのものを記述対象とする
      → 平衡もメタステーブルもカオスも並列に扱う

    ・「真の力学方程式」の存在仮定
      → 現在の関数形は局所線形近似に過ぎない
      → 別の関数形での同等記述が並立しうる
```

## ■ 4. アトラクター動態

### ● M_act = 安定固定点

```text
M_act ⊂ {x* | f(x*, u, 0) = 0, J(x*) の固有値全て負の実部}

つまり M_act は、外部入力 u のもとで、
現在の状態が引き込まれる安定固定点群である。

通常運用では x_i(t) はこれら固定点群の周辺を巡回する。
```

### ● M_lat = 鞍点（弱安定）

```text
M_lat ⊂ {x* | f(x*, u, 0) = 0, J(x*) の固有値に正の実部あり}

M_lat は鞍点。一部の方向には安定だが、別の方向には不安定。
通常は通過するだけ。
ξ や u の揺動で M_lat 近傍に長く留まると、
そこから新しい固定点（M_B'）への分岐が起きうる。
```

### ● M_Δ = 遷移相

```text
M_Δ = M_act から M_act' へ移動する軌道が
      鞍点 M_lat 近傍を通過する不安定相状態

数学的には、不安定多様体に乗った軌道として記述。
時間幅は ξ レベルと固有値の比で決まる。
```

### ● 引込領域（basin of attraction）

```text
basin(M_act_k) = { x(0) | lim_{t→∞} x(t) ∈ neighborhood(M_act_k) }

これが M_B の「有効範囲」の力学的対応物。
basin の境界 = separatrix。
EFP がこれを越えると別の M_act_k' に引き込まれる。
```

### ● 重要な特殊ケース

```text
極限環（limit cycle）：
    x(t) が周期軌道を巡る
    → 強迫的反芻、ルーチン依存、周期的気分変動

カオス（strange attractor）：
    決定論的だが非周期、初期値鋭敏性
    → 内部生成 F が予測困難に分岐し続ける状態
    → 創造的混乱、アイデア爆発、または病的混乱

メタステーブル状態：
    弱安定で揺動により遷移
    → O3（未線形保持力）の対応物
    → 「決めずに保ち続けている」状態
```

## ■ 5. 分岐解析と構造破壊学

### ● 跳躍 M_B → M_Δ → M_B' の分岐解析的記述

```text
v2.3 の跳躍条件：
    H(t) > θ + ξ(t)  →  M_B → M_Δ → M_B'

神経力学系での読み替え：
    H_vec の蓄積によって、ベクトル場 f が
    パラメータ μ = H_vec の関数として構造変化する：
    
    f = f(x, u, ξ; μ(H_vec))
    
    H_vec が閾値を越えると、固定点が消失・出現・統合・分裂する。
    これが分岐 (bifurcation)。
```

### ● 主要な分岐種類とSFO的解釈

```text
saddle-node 分岐：
    安定固定点と鞍点が衝突して消滅
    → 既存 M_act が突然消える
    → 「何かを大事にしていたが、急に意味が消えた」
    → 燃え尽き、価値観崩壊

pitchfork 分岐：
    一つの安定点が二つに分裂（対称性破れ）
    → 一つの M_B が二つの M_B' に分裂
    → 価値観の二極化、自己分裂感

Hopf 分岐：
    安定固定点が極限環に変わる
    → 静かだった状態が周期振動を始める
    → 強迫反芻の開始、躁うつ周期の出現
    → 慢性高NA 環境で頻発

transcritical 分岐：
    安定点と不安定点が入れ替わる
    → これまで主役だった M_act が脇役になり、
      隠れていた M_lat が表に出る
    → 「自分が変わった気がする」体験

crisis（カオス分岐）：
    分岐の連続でカオス領域に入る
    → 過剰拡張状態、変性意識、急性精神病
    → 構造耐久証明書の極限破壊試験領域
```

## ■ 10. シミュレート実装の指針

### ● 最小実装

```text
コア層のみで動態シミュレートを試す。

状態変数（最小セット）：
    x = [DA, 5-HT, NA, OXT,
         M_act_1, M_act_2,    # 2つの仮設空間
         H_1, H_2,
         Attention_DMN, Attention_TPN]

実装言語：
    Python + scipy.integrate.solve_ivp
    （numpy + matplotlib で十分）

入力：
    EFP(t) = 階段関数または周期関数で外部刺激を表現
    W_ij = 単一個体ならゼロ、二者なら 1 ペア

雑音：
    ξ(t) = ガウス雑音（連続）+ 稀な大跳躍（Lévy 雑音）

時間ステップ：
    Δt ≈ 0.01 〜 0.1 秒相当
    観察時間：数日相当（10^5 ステップ程度）
```

### ● 検証ターゲット

```text
既知の力学（v2.2 基層構造文書に記述済み）が再現できるかを検査。

検証 1：ストーカー力学
    OXT_i_to_j 高 × OXT_j_to_i 不在 × NA 高
    → j のアトラクターが i の引込領域に吸収される
    → j の独立変数自由度が消失
    → H 蓄積 → 跳躍

検証 2：DV 力学
    H_external 蓄積 × OXT 結合先（家族）への持ち込み
    → 内側結合相手で H 散逸が起きると誤認
    → 慢性的 H 蓄積 → Hopf 分岐 → 周期的暴発

検証 3：自己犠牲の力学
    OXT 結合先（個人/概念/コミュニティ）の M_act 強度が
    自己の M_act 強度を超える
    → basin(他者) ⊃ basin(自己)
    → 自己アトラクター消失
    → 命を厭わない行動

検証 4：道具の習熟
    特定 EFP に対する反応の繰り返し
    → 道具との結合 W_self,tool が強化
    → tool が拡張 M の一部として処理される
    → 「自分の体の一部のように扱える」
```

### ● 観測対象

```text
シミュレートで観測すべきもの：

1. 状態軌跡 x(t)
2. アトラクター構造（固定点の位置と固有値）
3. 引込領域の境界
4. 跳躍タイミングと跳躍前後の M_act 変化
5. H_vec の蓄積・散逸プロファイル
6. メタステーブル状態の停留時間分布
7. パラメータ感度（θ_i のどこが効くか）
8. 二者結合系での同期・非同期遷移
```

### ● シミュレート結果の扱い方（重要）

```text
シミュレート結果 ≠ 真理

シミュレートで再現できることは、
「そのパラメータセットでその動態が出る」だけであり、
「その個体が真にそのパラメータを持つ」を意味しない。

複数のパラメータセットが同じ動態を生成しうる
（パラメータ識別不可能性）。

したがって：
    ・「あなたはこのパラメータです」とは言わない
    ・「このパラメータセットの一つで、あなたの観測動態が
      再現できます」と言う
    ・別のパラメータセットでも同じ動態が出る可能性を残す
```
