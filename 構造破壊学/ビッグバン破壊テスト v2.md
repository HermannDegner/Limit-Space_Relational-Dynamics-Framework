```text

```

SILN分解では、境界Bを仮定し、整合慣性M_Bを仮設する操作が基本なので、ここでは「宇宙全体」ではなく **観測可能領域B_obs** を明示する必要があります。

# 1. ラベル分解

「ビッグバン」というラベルは、少なくとも4層に割れる。

| 層       | 内容                      | 判定       |
| ------- | ----------------------- | -------- |
| 神話層     | 宇宙全体が一点から始まった           | 破壊       |
| 起源層     | t=0を説明する理論              | 破壊／未到達   |
| 観測可能領域層 | B_obs内の熱履歴・膨張履歴         | 高耐久      |
| 計算層     | ΛCDM、FLRW、CMB解析、BBN、BAO | 高耐久だが熱あり |

特に「一点から爆発した」というイメージは壊すべきです。NASA WMAPは、ビッグバンは空間内の一点で起きた爆発ではなく、宇宙には膨張の中心点がないと説明しています。NASAのJWST Q&Aでも、Big Bangという名前は誤解を招きやすく、宇宙は中心を持つ火薬爆発ではなく「どこでも同時に起きた時間的プロセス」と説明されています。([NASA Science][1])

# 3. M_B_obs：観測可能領域SILN

ビッグバン破壊後に残る中核はこれ。

```text
M_B_obs =
観測可能領域は、過去に高温・高密度であり、
膨張と冷却を通じて、CMB・軽元素・銀河構造を残した、
という整合慣性。
```

このM_B_obsを支える観測は強いです。

CMBは特に強い柱です。ESAはCMBをビッグバンの残光として説明し、宇宙が約38万年後に透明化して光が自由に進めるようになったと述べています。NASAも、再結合期に宇宙が透明になり、その光が現在もCMBとして観測されると説明しています。([欧州宇宙機関][2])

NASAのBig Bang解説では、CMBはビッグバン後約38万年に由来し、現在は宇宙膨張で引き伸ばされて約2.7Kの背景放射として観測されると説明されています。([NASA Science][3])

## テストB：局所現象テスト

### 対象

```text
膨張率H0は大域的な宇宙膨張をそのまま表す
```

### 破壊圧

```text
局所ボイド
局所速度場
観測者位置
距離梯子の補正
赤方偏移—距離変換の局所汚染
```

### 判定

```text
部分破壊。
```

局所構造がH0測定を歪める可能性はかなり重要です。近年のlocal supervoid / KBC void系の研究では、局所の数百Mpc領域が低密度であれば、外向き流れが局所的な膨張率を高く見せ、Hubble tensionに関係しうると検査されています。ただし、そのような局所ボイドだけで標準ΛCDM内のHubble tension全体を解くのは難しい、という制約も同時に出ています。([OUP Academic][4])

ここで壊れるのは、

```text
宇宙全体の膨張率 = 近傍で測ったH0
```

という短絡です。

残るのは、

```text
大域膨張近似 + 局所構造補正
```

です。

## テストC：CMB耐久テスト

### 対象

```text

```

### 破壊圧

```text

別の局所現象で説明できるか？
```

### 判定

```text
高耐久。
```

CMBはビッグバン残存SILNの最強部です。ESAはPlanckの主目的を、ビッグバンの残光であるCMBを全天で高感度・高解像度に測ることだと説明しており、CMBは宇宙が約38万年だった時代の状態を保存しているとしています。([欧州宇宙機関][2])

局所現象説がビッグバンを完全に置換するには、CMBの全天性・黒体性・微小ゆらぎ・大規模構造との接続まで同時に説明する必要があります。これは重い。

## テストD：元素合成テスト

### 対象

```text
初期宇宙で軽元素が形成された
```

### 破壊圧

```text
観測元素比と理論予測のズレ
特にリチウム問題
```

### 判定

```text
中核は生存、局所ヒビあり。
```

NASAは、ビッグバン後の数分間に陽子と中性子が衝突し、水素・ヘリウム・微量のリチウムやベリリウムが形成されたと説明しています。([NASA Science][5])

ただし、標準ビッグバン元素合成にはリチウム7の予測量が観測より多いという未解決問題があります。2024年のレビューでも、軽元素存在比は広い範囲でよく合う一方、リチウム7は理論値が観測より約2.5倍高い問題が残るとされています。([サイエンスダイレクト][6])

構造破壊学的には、

```text
BBN全体は生存。
リチウム部分にEが残存。
```

です。

さらにDESIの2025年結果では、暗黒エネルギーが宇宙定数ではなく時間変化している可能性の兆候が強まったと報告されています。ただし、これは「確定破壊」ではなく、標準モデル更新候補です。([DESI][8])

したがって壊れるのは、

```text
ΛCDM = 最終真理
```

であって、

```text

```

は残ります。
