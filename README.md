# Limit-Space Relational Dynamics Language
# 限界空間・関係力学言語 (RDL)

このリポジトリは、  
**限界空間** と **関係力学言語（RDL）** を中心に、  

存在・認識・構造・変化・跳躍を扱うための思考実験群／理論断片群／応用モデル群です。

ここにある文書は、完成された単一理論というよりも、  
**SILN を扱うための言語と、その運用記録、その自己観察記録** です。

## What is this? / これは何か

このリポジトリは、世界を固定的な実体の集合としてではなく、  
**関係の中で一時的に安定している構造（SILN）** として捉える試みです。

中心にあるのは次の命題です：

> **世界は SILN（M_B）の分解と再構成と追従の連続である。**

そしてこのリポジトリは、それを扱うための言語を整備し続ける場所です：

> **RDL（関係力学言語）は SILN を扱うための言語である。**

中心にある問いは、たとえば次のようなものです。

- なぜ構造は安定するのか
- なぜ構造は壊れるのか
- なぜ認識は偏るのか
- なぜ思考・感情・社会・物理現象に似た流れが見えるのか
- なぜ限界や誤差が、単なる失敗ではなく新しい構造の入口になるのか
- なぜ「同じもの」が時間を超えて同じであり続けられるのか（あるいは、続けられないのか）

## RDL への名称移行と位置づけ / Transition to RDL

歴史的には「Relational Dynamics Framework（RDF）」という名称でスタートしましたが、理論が固定的な「枠組み（Framework）」から「世界を認識・記述するための動的な言語（Language）」へと自己進化したことに伴い、現在は **RDL（Relational Dynamics Language：関係力学言語）** へと名称を移行しています。

RDL は以下の階層全体を指します：

```
[最広] 関係力学（Relational Dynamics）
        分野・思想・基本姿勢

[中広] 関係力学運用系(RDL System)
        言語＋操作系＋応用＋道具特性 を含む全体

[中核] 関係力学言語(RDL)
        SILN を扱うための語彙・文法・意味論

[深層] 弱い暫定普遍文法（限界空間版）
        SILN 動態の深層構造として広く有効に見える
        ただし絶対普遍ではない

[存在様式] 生物的構造を持つ言語
        代謝・恒常性・自己食・適応・有限性を備える

[動態構造] AI 的構造
        分散表現・注意・訓練ループ・創発を備える
```

現在は移行期であるため、過去の文書群やファイル名には「RDF」という表記が多数残っていますが、これらはすべて「RDL」と同義として（あるいはかつての Framework の名残を持つ独立ラベルとして）扱ってください。
「RDL」と書くときは系統全体を指す通称として、  
「関係力学言語」と書くときは中核を指す精密表記として使い分けます。

## 中核命題 / Core Propositions

このリポジトリで確定している命題群：

### 1. RDL は SILN を扱うための言語である

世界側にあるのが SILN、観測者側にあるのが言語。  
RDL は世界そのものではなく、SILN を扱うための道具です。  
ただし RDL 自身も SILN として動的に運用されます（自己食的構造）。

### 2. 絶対基底は存在しない

限界空間前提により、観測・認識・記述には常に限界があります。  
その限界は欠陥ではなく、構造が変化し新しい可能性が生まれるための条件です。  
RDL は絶対解の生成器ではなく、**ある B（境界）の範囲で有効な近似** を提供します。

### 3. 関係力学言語は弱い暫定普遍文法である

これまで適用した SILN 担体（物理・生体・認知・社会・AI など）で  
破綻なく機能しており、SILN 動態の深層文法として広く有効に見えます。  
ただし絶対普遍ではなく、反例が出れば言語の方を更新します。

### 4. 関係力学言語は生物的構造を持つ

代謝（語彙更新）、恒常性（中核保持）、境界（借用での物質交換）、  
繁殖（応用層への分岐）、自己食（自己食的構造）、適応（反例対応）、  
有限性（破綻条件）── すべてを備えています。  
これは比喩ではなく構造的相同です。

### 5. LSRDL 文書系は AI 的構造を持つ

分散表現（概念は単一文書に局在しない）、注意（文脈で文書が呼び出される）、  
訓練ループ（会話→更新→次の会話）、創発（命題が会話の中で立ち上がる）、  
自己教師（自己食的構造）── これらを備えています。  
人間 + 文書系 + AI の三者で一つの認知系を構成しています。

## Core Orientation / 基本姿勢

### 1. 絶対的な確定性を前提にしない

観測・認識・言語化・モデル化には、常に限界があります。  
その限界は欠陥ではなく、構造が変化し、新しい可能性が生まれるための条件です。

### 2. 構造を「関係の安定」として見る

物体、概念、習慣、制度、人格、物語、理論などは、  
固定された実体ではなく、関係の中で一時的に維持される安定状態（SILN）として扱います。

### 3. 変化を「誤差・熱・跳躍」として見る

構造に対して外部または内部から作用が加わると、ズレや蓄積が生じます。  
それが一定の閾値を超えると、構造は維持ではなく再編成へ向かいます。

### 4. 動性は欠陥ではなく構造的帰結である

ラベル M_B ネットワーク、B、M_B、観測者 ── すべて常時揺れています。  
SILN分解は一回操作ではなく、揺れに対する追従運用です。

### 5. 理論自体も変化する構造として扱う

このフレームワークは、完成形を固定することを目的にしていません。  
むしろ、更新・分岐・再編成されること自体を前提にしています。  
**完成しないことが言語の生物的・AI 的性質の証拠です。**

| 関係力学言語（RDL） | SILN を扱うための言語 |
| SILN | Survival-biased Integration of Local Linear approximations in a Nonlinear world。関係の中で一時的に安定しているパターン |
| ξ | 揺らぎ。確定しきれない余白。SILN化を拒む方向 |
| B | 境界。観測者が引く irreducible な操作 |
| M_B | 境界 B における整合慣性。SILN の本体 |
| EFP | 素流圧。M_B の外側から流入する未解釈の作用束（直接観測不能） |
| F | 作用解釈。M_B が EFP を解釈した結果（観測者が実際に持つ唯一の情報） |

## Important Note / 注意

このリポジトリに含まれる文書は、必ずしも相互に完全整合していません。

**理論が変化し続ける過程そのもの** です。

新しい文書には、以前の考え方を修正・拡張・反転しているものがあります。

読むときは、各文書を「最終回答」ではなく、  
ある時点での観測記録・構造断面・思考ログとして扱ってください。
