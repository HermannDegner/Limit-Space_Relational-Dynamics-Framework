# Limit-Space Relational Dynamics Framework  
# 限界空間・関係力学フレームワーク

このリポジトリは、  
**限界空間** と **関係力学** を中心に、存在・認識・構造・変化・跳躍を扱うための思考実験群／理論断片群／応用モデル群です。

ここにある文書は、完成された単一理論というよりも、  
変化し続ける関係構造を観測・記述・再構成するための作業場です。

---

## What is this? / これは何か

このフレームワークは、世界を固定的な実体の集合としてではなく、  
**関係の中で一時的に安定している構造** として捉えます。

中心にある問いは、たとえば次のようなものです。

- なぜ構造は安定するのか
- なぜ構造は壊れるのか
- なぜ認識は偏るのか
- なぜ思考・感情・社会・物理現象に似た流れが見えるのか
- なぜ限界や誤差が、単なる失敗ではなく新しい構造の入口になるのか

---

## Core Orientation / 基本姿勢

このリポジトリでは、以下の前提を重視します。

### 1. 絶対的な確定性を前提にしない

観測・認識・言語化・モデル化には、常に限界があります。  
その限界は欠陥ではなく、構造が変化し、新しい可能性が生まれるための条件です。

### 2. 構造を「関係の安定」として見る

物体、概念、習慣、制度、人格、物語、理論などは、  
固定された実体ではなく、関係の中で一時的に維持される安定状態として扱います。

### 3. 変化を「誤差・熱・跳躍」として見る

構造に対して外部または内部から作用が加わると、ズレや蓄積が生じます。  
それが一定の閾値を超えると、構造は維持ではなく再編成へ向かいます。

### 4. 理論自体も変化する構造として扱う

このフレームワークは、完成形を固定することを目的にしていません。  
むしろ、更新・分岐・再編成されること自体を前提にしています。

## Key Terms / 主要語彙

このリポジトリでは、文書ごとに語彙の揺れや発展があります。  
以下は厳密な最終定義ではなく、読むための入口です。

| Term | Rough Meaning |
|---|---|
| 限界空間 | 絶対的確定性が成立しない場。可能性・誤差・揺らぎを含む前提空間 |
| 関係力学 | 構造と作用の関係から、安定・変化・跳躍を記述する考え方 |
| 構造 | 関係の中で一時的に安定しているパターン |
| 素流圧 | 構造に作用する流れ・圧力・影響 |
| 整合慣性 | 構造が自分を保とうとする傾向 |
| 揺らぎ | 確定しきれない余白。変化可能性の源泉 |
| 熱 | 未整合・誤差・負荷の蓄積 |
| 跳躍 | 既存構造の維持では処理できなくなったときの再編成 |

## How to Read / 読み方

このリポジトリに、唯一の正しい読む順番はありません。

ただし、初めて読む場合は次の順序が比較的入りやすいです。

1. **限界空間系の文書**  
   絶対性を前提にしない基本姿勢を掴む。

2. **関係力学・RDF 系の文書**  
   構造、作用、整合、跳躍などの基本語彙を掴む。

3. **語句定義・ラベル系の文書**  
   用語の使い方や、既存語の再定義を確認する。

4. **考察系・応用系の文書**  
   認知、感情、AI、社会、創作、教育、都市、構造破壊などへの展開を見る。

5. **数理・モデル系の文書**  
   形式化、シミュレーション、エージェント化、診断モデルなどを見る。
