## ■ 0. 位置づけ

本文書は、**SILN とは何か**を正面から扱う文書である。

SILN という概念は RDL 系の至る所に登場するが、
「SILN とは何か」それ自体を掘った単独文書はこれまで存在しなかった。

SILN分解の数理フレームワーク v2.2 は「分解操作」を扱う文書であり、
限界空間的語句定義 v3 は辞書的な定義を与えるが、
どちらも SILN の概念的射程・操作可能性の構造・自己食的性質を正面から論じない。

本文書はその空白を埋める。

## ■ 1. SILN とは何か

SILN は以下の頭字語である：

```
Survival-biased Integration of Local Linear approximations in a Nonlinear world
非線形世界における局所線形近似の生存バイアス的積分
```

しかしこの定義は長すぎて直感を与えない。

一文で言えば：

> **SILN とは、関係の中で一時的に安定しているパターンである。**

物体・概念・習慣・制度・理論・感情・人格・生態系・言語──
我々が「何かが存在する」「何かが続いている」と感じるものは、
すべて関係の中で一時的に安定を保っているパターンとして読める。

固定された実体として存在しているのではない。
関係と作用の中で、ある程度壊れずにいるから「そこにある」ように見える。
それが SILN である。

---

## ■ 2. SILN の射程

SILN の射程は、原理的にすべてである。

```
物体       ：炭素・水・岩石・星
生命       ：細胞・個体・種・生態系
認知       ：習慣・記憶・性格・信念
社会       ：制度・文化・言語・国家・貨幣
理論       ：ニュートン力学・RDL・数学
創作       ：物語・世界観・キャラクター
操作       ：SILN分解という操作そのもの
概念       ：「SILN」というラベル自体
```

「SILN化できないもの」という概念を立てても、
その概念自体がラベルとして扱われた瞬間に SILN化される。

だから SILN の射程は「すべて」ではなく、より正確には：

> **SILN化を免れるものは、ラベル化される前の ξ だけである。**
> **だし、ξ と呼んだ瞬間にすでに SILN化されている。**

これは限界空間の自己参照的性質の直接の帰結である。

---

## ■ 3. 操作可能性は連続量である

射程はすべてだが、すべての SILN が同じように扱えるわけではない。

SILN の重要な性質は、**操作可能性が連続量である**ことだ。

「SILN かどうか」は二値ではない。
「どの程度操作できる SILN か」というグラデーションの問いになる。

```
高  操作可能 SILN   ：分解・追従・借用・設計ができる
                      例：個体の習慣・制度・ニュートン力学
↕
↕  操作困難 SILN   ：粗い近似なら扱える。精密化は難しい
                      例：生態系全体・文明・集合的無意識
↕
低  操作不能 SILN   ：仮設すると便利だが、操作はできない
                      例：EFP・ξ・𝒦（関係保存則の不変量）
```

操作不能 SILN については次節で詳しく扱う。

---

## ■ 4. 操作不能 SILN：「仮設すると便利」

操作不能 SILN は、「存在を認める」という言い方が強すぎる。

より正確な言い方は：

> **仮設すると記述が便利になる。**
> **仮設しなくても世界は困らない。**
> **ただし仮設すると操作が成立する。**

典型例：

**EFP（素流圧）**
```
「EFP が存在する」ではなく、
「EFP を仮設すると、F の逆算が書ける」
EFP_estimated = interp⁻¹(M_B, F)
仮設しないと、F がどこから来たかを記述できない。
```

**ξ（揺らぎ）**
```
「ξ が存在する」ではなく、
「ξ を仮設すると、閉じなさが記述できる」
B を引いてもなぜ M_B が完全に安定しないかを説明できる。
仮設しないと、動態の起源が記述できない。
```

**𝒦（関係保存則の不変量）**
```
「𝒦 が存在する」ではなく、
「𝒦 を仮設すると、||M_B|| · D[ξ] = 𝒦 という保存則が書ける」
仮設しないと、慣性と揺らぎの交換関係が記述できない。
```

これらは形まず上空間（限界空間の補完文書参照）に属する概念であり、
限界空間の内側に持ち込むと体系が破綻する。
しかし限界空間の記述式の中に「接続ラベル」として置くことで、
操作性を確保する装置として機能する。

**操作不能 SILN は存在論的主張ではなく、操作上の仮設である。**

## ■ 5. SILN の四つの性質

すべての SILN に共通する性質を四つ挙げる。

### 性質 1：境界依存性

SILN は境界 B なしには存在しない。

```
M_B = P_B(整合慣性場)
```

同じ対象でも B が変われば別の SILN として現れる。

```
炭素という SILN：
  B = 化学スケール  → M_B = sp2/sp3 結合パターン
  B = 結晶スケール  → M_B = ダイヤ／黒鉛／ナノチューブ
  B = 社会スケール  → M_B = 「ダイヤモンド」というラベル運用
```

「真の SILN」「本来の SILN」は存在しない。
あるのは「境界 B における SILN」だけである。

### 性質 2：仮設性

SILN（M_B）は直接観測できない。

行動・出力・反応のパターンから推定した仮説に過ぎない。
「M_B を発見した」ではなく「M_B を仮設した」が正確である。

仮設であることは弱点ではない。
仮設だからこそ、目的に合わせて引き直せる。

### 性質 3：ξ 残存性

B を引いても、ξ は M_B の中・縁・外に残り続ける。

```
D[ξ] は ||M_B|| をいくら大きくしてもゼロにはならない
```

これが SILN の「閉じなさ」の根拠であり、
動態（誤差・跳躍・更新）が存在する理由である。

完全に安定した SILN は存在しない。
どんな SILN にも揺らぎが染み込んでいる。

### 性質 4：分解しないと意味を持たない

SILN はそのままでは抽象度が高すぎて使えない。

```
「知性は SILN である」
→ それだけでは何も操作できない

「知性というラベルに対して B を仮定し、M_B を仮設する」
→ 操作が始まる
```

SILN はラベルであり、分解操作の入り口である。
分解なしの SILN はラベルの空洞化を招く。

## ■ 6. SILN の動態

SILN は静止しない。

```
M_act  ：活性断面（現在強く機能している）
M_lat  ：潜在断面（まだ発現していない）
M_Δ    ：再編相（変化の途中・不安定）
M_B'   ：再編後の安定状態
```

これらは別の SILN ではなく、**同じ SILN の異なる相状態**である。

整合（M_act が安定して機能する状態）と
跳躍（H が閾値を超えて M_Δ を経由して M_B' へ移る遷移）は、
別々の現象ではなく同一ダイナミクスの異なる断面である。

## ■ 7. 自己食的構造

SILN という概念自体も SILN である。

```
「SILN」というラベルに対して B を引くと、
M_B = 「関係の中で一時的に安定しているパターン」というラベル運用の整合慣性
が現れる。

この M_B 自体が揺れており、
文書が更新されるたびに M_B ネットワークは変化する。
```

本文書も SILN である。
本文書を SILN として分解することができる。
その分解にも別の M_B が必要であり、この再帰は閉じない。

これは欠陥ではなく、限界空間の前提と整合する構造的帰結である。

## ■ 8. SILN と他の中核概念の関係

```
ξ（揺らぎ）
  ：SILN化を拒む方向の極限
  ：しかし「ξ」と呼んだ瞬間に SILN化される
  ：SILN の「閉じなさ」の源泉

B（境界）
  ：SILN を切り出す操作
  ：B なしに SILN は定まらない
  ：B 自体も SILN（観測者の M_B から派生する）

M_B（整合慣性）
  ：SILN の本体
  ：すべての M は M_B である
  ：独立した「M そのもの」は存在しない

EFP（素流圧）
  ：SILN に作用する未解釈の束
  ：操作不能 SILN の典型例

跳躍
  ：SILN の再編成
  ：M_B → M_Δ → M_B' の流れ
  ：整合と同一ダイナミクスの別断面
```
