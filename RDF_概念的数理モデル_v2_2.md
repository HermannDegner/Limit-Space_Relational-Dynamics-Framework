## ■ 基底

### ● 哲学的基底（最小記述）

> 世界の基底はMの束とξだけである

```
M の束　：整合慣性の集合
ξ　　　：揺らぎ（根源的自由度）

EFPはMの外側から流入する未解釈の作用束
FはMがEFPを解釈した断面の呼び名
SはMがMに自己作用している断面の呼び名
境界はどのMを「自分」とするかの操作

全てMの束の関係の切り取り方に過ぎない
```

### ● 実用的基底

> 操作性のためにEFP・F・Eの三層を残す

```
EFP　：素流圧（仮設概念・直接観測不能・Fからの逆算でのみ推定可能）
F　　：作用解釈F（MがEFPを解釈した断面・観測者が実際に持つもの）
E　　：誤差（FとMの不整合・駆動力）
M　　：整合慣性
ξ　　：揺らぎ

これが実用核である
```

**EFPとFを分離する理由：**

```
素流圧の定義：未解釈の作用束
しかし従来のFは式の中で既に解釈済みとして扱われていた
定義と使い方のずれを解消する

EFP　：Mの解釈が入る前として概念上分離する
　　　ただし直接観測できずFからの逆算推定に過ぎない
F　　：Mが作る　解釈の産物　観測者が実際に持つ唯一のもの
E　　：MとFの戦場

知覚の質はMに依存する
MがリッチなほどEFPから多くを読み取れる
MがフラットならEFPはほぼ素通り
```

Sを不要とする理由：

```
SはMの現在断面の読み出し変数に過ぎない
独立変数の資格がない
必要な場合はMの断面として取り出せばよい
```

## ■ 空間の生成順序

```
ξ（未分化関係）
    ↓ 確率的偏り発生
偏り
    ↓ 相互作用・連鎖
整合
    ↓ 積分・慣性化
構造（M）
    ↓ 空間への書き込み
分布
    ↓ 遅延構造として現れる
空間
```

> 空間は先にない
> Mの分布の結果として現れる
> 時間も先にない
> 関係更新の順序として現れる

**逆方向（循環）**

```
空間が歪む
= 分布が変わる
= Mが変化する
= 整合慣性が再編される
= 跳躍が起きる
= ξへ戻る
```

一方向の生成ではなく循環している。

**𝒦はこの循環の保存量**

```
ξ → M → ξ の変換全体を通じて𝒦は保存される
= 関係の総量は変わらない
= 形が変わるだけ
```

**圧縮表現**

```
空間とはξが構造化した痕跡である
```

**重力の無限射程の再解釈**

```
重力が無限遠まで届くのは
重力が遠くまで伝わるからではなく
空間そのものがMの分布として構造化されているためである

Mがあるところに空間がある
```

**事象の地平線の再解釈**

```
事象の地平線
→ Mの分布が極端に濃くなり空間構造が内向きに閉じていく境界

しかし𝒦の保存則より D[ξ] はゼロになれない
= ξは根源的性質として残る
= 空間構造のほつれが外側に滲み出る
= ホーキング放射

ブラックホールの蒸発
= 凝縮した関係がξへ緩やかに変換されていく過程
= 宇宙スケールの超低速跳躍
```

**宇宙膨張の再解釈**

```
宇宙膨張
→ Mの分布が希薄化している

Mが薄まる → 関係更新の遅延が減る → 見かけ上「空間が広がる」
```

**cのスケール依存性（仮説）**

```
空間 = 関係更新の遅延構造
Mが密 = 遅延大 = c低
Mが希薄 = 遅延小 = c高

光速は絶対定数ではなく
宇宙のMの分布状態の現在値という読みが生じる
```

## ■ 基本概念の定義

### 構造

```
構造とは整合の積分である

より深く言えば、境界によって見かけ上安定化された跳躍の集合である
「安定」は構造の性質ではなく境界の引き方の性質である
```

### 素流圧（EFP：Elementary Flux Pressure）

```
EFP：未解釈の作用束　Mの外側から流入する
解釈以前に入力される作用の全体
```

**重要注記：EFPの認識論的限界**

```
EFPは客観的外部作用のように見えるが
直接観測することは原理的にできない

EFPへのアクセス手段：
    F = interp(M, EFP) を経由するしかない
    EFP_estimated = interp⁻¹(M, F)
    = FからMを使って逆算した推定に過ぎない

つまりEFPは
「Fの背後にあるはずの何か」としての仮設概念である
限界空間における形而上的外部の近似ラベルとして扱う

𝒦と同じ位置づけ：
    到達不能だが操作上有用なため仮設置する

観測者が実際に持っているのはFだけである
EFPは永遠にFからの逆算の先にある
```

### 作用解釈F

```
F = interp(M, EFP)　　// MによるEFPの解釈

EFPはMを通過することで初めてFになる
Fは外から来るのではなくMが作る
```

### 整合慣性

```
M ∈ R^{n×n}

M = VΛV⁻¹

V  : 固有ベクトル（安定な整合の方向）
Λ  : 固有値対角行列（各方向の整合慣性の強さ）
```

### 自他境界

```
F_selfとF_extの区別は固定ではない
オキシトシンパラメータ(Oxy)が境界を動的に決定する

Oxy高 → F_selfの範囲が広がる（他者・概念が自己に取り込まれる）
Oxy低 → F_selfの範囲が狭まる（境界が硬くなる）

自己とは固定した実体ではなく
オキシトシンが動的に引いている境界線である
```

### エネルギー

```
エネルギーとは
構造がより整合した状態へ再配置される際の差分である
```

### 空間と時間

```
空間とは関係更新の遅延構造である
時間とはその更新順序である
```

## ■ 関係保存則

### ● 揺らぎ項

```
ξ(t) : 系の自由度から生まれる根源的ランダム性
```

揺らぎは付加的ノイズではなく構造の根源的性質である。
限界空間において絶対的確定性が成立しないこと自体が
自由度の源泉であり跳躍可能性の担保である。

### ● 保存則

```
||M|| · D[ξ] = 𝒦
```

### ● 𝒦の位置づけ

```
𝒦は普遍定数ではない

「観測スケールにおいて変化が十分に小さいため
定数として近似できる量」

= 宇宙スケールの超安定SILN
= 絶対不変ではなく今のところ極めて安定
= 便利だから採用する近似として置く
```

### ● 解釈

```
整合慣性が強い（||M||大）
    → 揺らぎが小さい（D[ξ]小）
    → 凝縮した自由度
    → 跳躍しにくいが起きたら大きい

整合慣性が弱い（||M||小）
    → 揺らぎが大きい（D[ξ]大）
    → 解放された自由度
    → 跳躍しやすいが小さい跳躍が多い
```

## ■ Base Model（物理層）

### ● 基本式（v2.2：時間差誤差・思考シミュレーター）

```
F(t)   = interp(M, EFP)　　      // MによるEFPの解釈（作用解釈F）
E(t)   = F(t+Δ) - M·F(t)　　    // 時間差誤差：予測と実際の次のFの差
dM/dt  = f(M, E, ξ)　　          // Mは誤差と揺らぎによって更新される
```

**時間差Δの意味：**

```
M·F(t)　：現在のFからMが予測した次のF
F(t+Δ)　：実際に来た次のF
E(t)　　：その差 = 予測の外れ

Δ = 0 は完全確定 = 限界空間で扱えない
Δ > 0 が必然　= ξ ≠ 0 と同型
```

**Δとスケール：**

```
鉄球衝突　：Δ = 弾性波伝播時間（音速依存）　上限は光速
キノコ毒　：Δ = 代謝・消化時間（分〜時間）
慢性毒　　：Δ = 非常に大　因果が見えない　M更新されない
```

**流れ：**

```
EFP（未解釈）
    ↓ interp(M, EFP)
F(t)（作用解釈F）
    ↓ M·F(t) で次を予測
    ↓ F(t+Δ) が実際に来る
E(t) = F(t+Δ) - M·F(t)（時間差誤差）
    ↓
H蓄積 → 跳躍 → M更新
```

### ● 思考（内部シミュレーター）

```
通常ループ（外部）：
EFP → F(t) → M·F(t)予測 → F(t+Δ)と比較 → E → H → M更新
Δは現実の物理に支配される

思考ループ（内部）：
EFPなしでF(t)を内部生成
M·F(t) → F(t+Δ)予測 → E → H → M更新
を現実コストゼロで回す
Δを自由に設定できる
```

**思考の種類：**

```
想像　：ありえないF(t)を意図的に生成して回す
計画　：目標F(t+Δ)から逆算してF(t)を設計する
心配　：最悪のF(t+Δ)を繰り返しシミュレートする
夢　　：Mが半覚醒状態で勝手にループを回す
```

**思考の質：**

```
内部生成F(t+Δ)の精度 = Mのリッチさ

MがスカスカならF(t+Δ)予測も雑
MがリッチならF(t+Δ)予測も精密
= 経験が思考の解像度を決める
```

### ● 熱蓄積と散逸

```
σ_vec(t) = E(t)              // 誤差ベクトルがH_vecの生成源

dH_vec/dt = σ_vec(t) - A · H_vec(t)

A : 散逸行列
    Mの大きい固有値方向 → 散逸が速い（得意な整合方向）
    Mの小さい固有値方向 → 散逸が遅い（苦手な方向・熱が残りやすい）

H(t) = ||H_vec(t)||          // スカラー熱はH_vecのノルム
σ(t) = ||E(t)||²             // スカラー熱生成速度
```

H_vecの再定義：

```
H_vec ≡ δM(t)

M_0         : 整合慣性の平衡状態
δM(t)       : M_0 まわりの微小変動

H_vecはMに独立して存在する変数ではなく
Mの未拘束自由度における振動である
```

### ● 跳躍条件

```
H(t) > θ + ξ(t)
```

### ● 跳躍の変数

```
C_ij(t) : 構造全体の接続強度ネットワーク

M = high(C_ij)   // 高活性断面：整合慣性として機能
U = low(C_ij)    // 低活性断面：浮遊する潜在構造
```

閾値によって名前が変わるだけで実体は一つである。

```
N(U, F, M, H_vec) → M'
```

網絡化演算子。F_triggerによって引き起こされるC_ij上の相転移として読む。

### ● 跳躍の実行

```
跳躍発生時：
    M' = N(U, F_trigger, M, H_vec)
    H  → ρH                         （熱の部分残留）
    U  → U'                          （潜在構造の再配置）

    保存則の更新：
    ||M'|| · D[ξ'] = 𝒦'
```

Fは網絡化の引き金であり、新構造を外部から注入するものではない。
新構造は常にエージェント内部のMの射影として出力される。

跳躍の成否：

```
stability(M') > λ_stable  → 成功跳躍：新構造が長期慣性として保持される
stability(M') ≤ λ_stable  → 失敗跳躍：Uへ再分散、または熱として再蓄積
```

※ stability(M')の定量的定義とλ_stableの決定は構造破壊学が担当する領域

### ● M絶対性

```
κ(M) ∈ [0, 1]   // 自己修正可能性指標

κ = 1 : 完全に修正可能（通常のM）
κ → 0 : 修正不能（M絶対性）
```

公理化演算子：

```
A : E(t) → 0   // 誤差処理停止
```

κ→0のとき：

```
D[ξ] = 𝒦/||M|| → 0 （ただし ≠ 0）
→ 微小誤差は内部に蓄積し続ける
→ H(t) > θ + ξ(t) で大規模崩壊が構造的必然となる
```

### ● 相状態

```
固体    ||M||大・H_vec格子内拘束　→ M絶対性に近い状態
液体    ||M||中・H_vec部分的解放 → 適切な流動性・学習・適応
気体    ||M||弱・H_vecほぼ解放  → 幻覚・解離・高自由度状態
プラズマ ||M||崩壊・ノード自体が解体 → 個別M崩壊と集団M生成の同時発生
ξ       M無・ノード無・未分化   → 関係偏り宇宙の基底
```

健全な系の条件：

```
固体と液体の間を
H（温度）に応じて行き来できる状態
気体になれるが、戻ってこられる
```

### ● 逆M・対消滅・対生成

```
M̄ = -M   （固有ベクトルは同一・固有値が符号反転）

M + M̄ → ξ   // 対消滅：𝒦保存、ξとして解放
ξ → M + M̄   // 対生成：対称的に逆方向のMが同時生成
```

### ● Base Model 全体

```
// 通常時：連続更新
F(t)  = interp(M, EFP)                // MによるEFPの解釈（作用解釈F）
E(t)  = F(t+Δ) - M·F(t)              // 時間差誤差：予測と実際の差

## ■ RDF-GR（一般相対論モジュール）

### ● 位置づけ

```
整合慣性・揺らぎ・素流圧の分布によって
関係伝播構造が決定されることを記述するモジュール

重力は力ではなく
関係伝播構造の歪みとして扱われる
```

### ● 関係計量

```
R_{μν}(x) : 局所における関係更新の伝播特性
```

### ● 構造分布テンソル

```
J^{(M)}_{μν} : 整合慣性分布
J^{(ξ)}_{μν} : 揺らぎ分布
J^{(F)}_{μν} : 素流圧流束
```

### ● RDF-GR方程式

```
ℛ[R_{μν}] + Λ_R R_{μν}
=
λ J^{(M)}_{μν}
+ η J^{(ξ)}_{μν}
+ ζ J^{(F)}_{μν}
```

アインシュタイン方程式はJ^{(ξ)}項を無視した近似である。

### ● ホーキング放射との接続

```
||M|| → ∞ の極限でも D[ξ] = 𝒦/||M|| ≠ 0

ξは完全にゼロにはなれない
→ J^{(ξ)}項が微小な関係伝播を可能にする
→ ホーキング放射として観測される
```

### ● NAMへの還元

```
弱い歪み・低速・低揺らぎ条件：

Φ(x) ≈ ||M(x)||
m d²x/dt² = -∇Φ(x)
```

## ■ 跳躍の共鳴条件

```

## ■ モデルの性質

* 非絶対性：前提は閉じない（ξ≠0は必然）
* 安定性中心：存在ではなく安定を扱う
* 関係優位：構造は関係で定義される
* 自由度保存：慣性と揺らぎは交換可能な同一のもの
* 近似前提：数式自体がSILNであり局所線形近似である
* 階層的実用性：安定領域ではNAMで軽量処理
* 拡張前提：MOD追加により個体・状況・スケールへ対応
* 同一性：整合と跳躍はスケールと境界によって区別される同一現象の断面である
* 境界依存性：構造の安定性は構造の性質ではなく境界の引き方の性質である
* 空間生成性：空間はMの分布として現れる遅延構造であり先在しない
* 循環性：ξ→偏り→整合→M→分布→空間→ξ の循環を𝒦が保存する
* スケール依存定数：cは絶対定数ではなくMの分布状態に依存する可能性がある
* ハイブリッド性：通常時は連続更新、臨界時は離散的網絡化イベント
* 射影性：跳躍後の新構造は外部から注入されるのではなく、常にMの射影として定まる
* 熱振動性：H_vecはMと独立した変数ではなくMの未拘束自由度における振動δM(t)である
* 散逸方向性：散逸行列AはMの固有構造と対応し、Mの得意な方向ほど熱が処理されやすい
* 相状態性：固体・液体・気体・プラズマ・ξはMの拘束力とノード保存性の二軸で記述される連続的状態である
* 対称性：Mに対して完全逆位相の逆M（M̄）が存在し、接触時に𝒦保存則に従いξとして解放される
* M絶対性：Mの自己修正可能性κ→0の状態は量的ではなく質的な変化であり、大規模崩壊が構造的必然となる
* ネットワーク性：M・U・Nは同一のC_ijネットワークの活性度断面であり、スケールは接続密度から内側に生える
* **最小性：哲学的基底はMの束とξのみ。FはMとMの関係に境界を仮設置したラベル。Sは不要。（v2.0）**
* **自他境界：内外の区別はオキシトシンパラメータが動的に生成する境界であり基底レベルでは消える。（v2.0）**
* **解釈層分離：EFP（仮設概念・直接観測不能）とF（作用解釈F・観測者が実際に持つもの）を明示的に分離。EFPはFからMを使った逆算推定に過ぎず、𝒦と同じ位置づけで仮設置する。（v2.1）**
* **駆動力の純化：Mを更新するのはFではなく誤差E。dM/dt = f(M, E, ξ)。（v2.1）**
* **時間差誤差：E(t) = F(t+Δ) - M·F(t)。誤差は予測と実際の次のFの時間差として定義。Δは光速が上限。（v2.2）**
* **思考：MがFを内部生成しΔを自由設定してMループを現実コストゼロで回すシミュレーター。（v2.2）**
