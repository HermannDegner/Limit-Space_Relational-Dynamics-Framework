## ■ 位置づけ

エージェント設計の最小構成として、兎を選ぶ。

理由：

- 基層構造が支配的でシンプル
- 上層構造のノイズが少ない
- 跳躍モジュールの動態が観測しやすい
- 現実の行動との照合が容易

兎エージェントはRDFの各モジュールが
最もクリーンに機能する検証環境として機能する。

## ■ 1. 兎の世界構造

### 物理構造

草原・天敵・気温・地形・光量

これは所与の制約として扱う。

### 基層構造（40億年SILN）

```
優先順位：
1. 捕食者回避
2. 食べる
3. 繁殖
4. 巣への帰還
```

この優先順位自体が整合慣性 M_base として固定されている。
人生スケールでは書き換えられない。

### 基層パラメータの傾向

```
ノルアドレナリン系：支配的
　→ 捕食者への反応速度が生存に直結
　→ α1（非常時切替）の閾値が低い

ドーパミン系：中程度
　→ D3（再現性固定）が学習を担う
　→ D4（退屈）が探索を促す

セロトニン系：低め
　→ 誤差許容度が低い
　→ 敏感・繊細・素早い反応
```

## ■ 2. 状態変数（最小構成）

```
S(t)    : 現在の状態ベクトル（位置・覚醒度・空腹度）
M_core  : 整合慣性（行動傾向の重み）← 土台基準・原則不変
w[x]    : 各素流圧への感度重み（符号を含む）← 学習で更新される
H       : 誤差熱
θ       : 跳躍閾値 ← 反復で更新される
F(t)    : 素流圧（外部入力の束）
ξ(t)    : 揺らぎ
```

状態方程式（Base Model）：

```
dS/dt = -M_core · S + F(t) + ξ(t)
```

誤差と熱：

```
E(t)  = F(t) - M_core · S(t)
σ(t)  = ||E(t)||²
dH/dt = σ(t) - α·H(t)
```

跳躍条件：

```
H(t) > θ + ξ(t)
```

## ■ 3. 未知の人間との遭遇

### フェーズ1：遭遇（素流圧入力）

未知の大型物体が視野に入る。

```
F(t) = {大きい, 動いている, 未知, 匂い未分類}

→ ノルアドレナリンα1発火
→ 平時モード → 非常時モード
→ H が急上昇

H_encounter >> θ_normal
```

この時点では敵意の有無は関係ない。
**未知であること自体が熱源**である。

既存の捕食者パターンとの誤差：

```
E_predator(t) = F(t) - M_predator · S(t)

人間の素流圧は捕食者SILNと整合しない
→ ||E_predator|| が大きい
→ σ(t) が上昇
→ H が維持される
```

### フェーズ2：距離を保ちながら観測

逃げきらず、観測を続ける。

入力される素流圧の束：

```
F_obs(t) = {
  移動速度：遅い・急激でない,
  音：低周波・一定,
  匂い：捕食者パターンと不一致,
  接近速度：ほぼゼロ
}
```

整合試行：

```
捕食者SILNとの誤差：大　→ 敵カテゴリに入れられない
無害SILNとの整合：未成立 → 安全カテゴリにも入れられない
→ 「未知カテゴリ」のまま保留
→ H は高いまま維持
```

### フェーズ3：反復観測による感度重みの変化（v0.2 修正）

時間が経過しても攻撃が来ない。

```
「危険信号なし」の整合が反復される
→ D3 が再現性の確認を開始
```

v0.1では「M_safe が形成される」と記述していたが、
正確には **w[人間刺激] の符号と強度が変化する** 過程である。

```
初期：w[人間] << 0   （強い反発）
反復後：w[人間] → 0 に近づく（反発が弱まる）
```

熱の低下：

```
dH/dt = σ(t) - α·H(t)

σ(t) が低下 → H が徐々に減衰
H < θ_background に達すると
非常時モードが解除される
```

### フェーズ4：無視

```
H < θ_normal

→ ノルアドレナリン系が平時モードへ
→ 人間が「処理対象」から外れる
→ 食べるという基層欲求が前面に戻る
```

**無視とは信頼ではない。**
処理優先度が下がった状態の記述である。

```
人間の内部カテゴリ：
「今のところ攻撃してこない未知カテゴリ」

「無害カテゴリ」ではない
```

## ■ 4. 近づかれたら逃げる

### 近づくという素流圧の性質

```
F_approach(t) = {
  網膜上のサイズ変化率：↑
  匂い濃度変化率      ：↑
  音の大きさ変化率    ：↑
  距離変化速度        ：負（縮まる）
}
```

重要なのは**絶対値ではなく変化率**である。

```
||dF/dt|| > δ_threshold
→ ノルアドレナリンα1 再発火
→ H が急上昇
```

### 局所整合の崩壊

```
成立していた整合：
「今のところ無害」= 距離が保たれている条件付き整合

距離が縮まる
→ 条件が崩れる
→ w[人間・接近] に誤差が入る
→ H が再上昇

H > θ → 跳躍候補が生成される
```

### 跳躍候補の選別

```
候補1：逃げる（40億年SILN・耐久度最高）
候補2：さらに観測する（コスト高）
候補3：固まる（特定条件下のみ）
```

D3が反復確認してきた実績：

```
「近づかれて観測継続した個体」の生存率
＜
「近づかれて逃げた個体」の生存率

→ M_escape が圧倒的な整合慣性を持つ
→ 短期の「無害観測」では上書き不能
```

**近づく＝逃げる**は自動応答レベルに固定されている。

## ■ 5. 構造形成の二つのルート

### ルート1：害なし反復

```
「近づかれても何も起きない」の繰り返し

n回目の観測後の θ 実効値：
θ_eff(n) = θ_base + Δθ · log(1 + n_safe/n_total)

n_safe  ：安全確認回数
n_total ：総遭遇回数
```

ただし収束が遅い。

```
逃走コスト < 観測継続コスト

→ D3が「逃げる」を毎回再確認する
→ 害なしの証明は弱い素流圧
→ 何も起きないという情報は誤差を生みにくい
```

### ルート2：報酬（食べ物）がある

報酬は基層構造に直接接続される。

```
F_reward(t) = {食べ物の匂い・視覚・近接}

→ ドーパミン系（D1）が発火
→ ノルアドレナリン系と拮抗状態が発生
```

熱の競合：

```
H_escape ：逃げろ系の熱
H_reward ：行けドーパミン系の熱

H_total = H_escape + H_reward

拮抗状態 → 跳躍候補の多様性が増す
```

逃げた場合の誤差：

```
E_opportunity(t) = reward_expected - reward_obtained

逃げた → 食べ物を得られなかった
→ これが誤差として記録される
→ D3が「近づいたら報酬が来た」の再現性確認を開始
```

報酬ルートが速い理由：

```
害なし反復：消極的情報（誤差が小さい）
報酬あり  ：積極的誤差（逃げると損失が発生する）

損失由来の誤差の方がHを上げやすい
→ 跳躍候補の生成が促進される
```

### 両ルートの組み合わせ

```
害なし反復：θを徐々に上げる（逃げにくくする）
報酬      ：近づく行動に正の w を形成する

同時進行：
θ_eff(n)      ↑
w[人間・近接]  ↑（負から0方向へ、報酬で正へ）

→ 中核構造レベルへの定着が現実的になる
```

## ■ 6. 「人という構造」の収束過程（v0.2 修正）

### v0.1からの変更点

v0.1では「M_human が形成される」と記述していた。
正確には **w[人間関連刺激] の符号と強度が変化する** 過程である。

M（整合慣性）は土台基準であり、跳躍単位では書き換わらない。
変わるのは w と θ である。

### 初期状態

```
人間というラベルは存在しない

F_initial = {巨大・動く・未知}

w[この刺激] << 0  （強い反発・デフォルト）
```

### 揺らぎながら収束する軌跡

```
遭遇1：未知・逃げる　　　　　　　　 w[人間] << 0
遭遇2：また来た・逃げる　　　　　　 w[人間] << 0
遭遇3：また同じやつ・少し観測　　　 w[人間] ↑ 微小
遭遇4：報酬あり・逃げたが惜しかった w[人間] ↑ 小
遭遇5：近づいたら報酬・でも怖かった w[人間] ↑ 中（0に近づく）
遭遇N：w[人間・報酬文脈] > 0 になる → 近づく行動が安定
```

各遭遇での w の更新：

```
w[人間](n+1) = w[人間](n) + η · Δw_n · D3_weight(n)

η           ：学習率
Δw_n        ：n回目の整合の寄与
D3_weight(n)：再現性確認の重み（反復で増加）
```

揺らぎの変化：

```
初期：D[ξ] 大・w[人間] << 0 → 毎回異なる反応
中期：D[ξ] 中・w[人間] ≈ 0 → 傾向が出てくる
後期：D[ξ] 小・w[人間] > 0 → 安定した応答パターン
```

関係保存則より：

```
||w[人間]|| · D[ξ] = 𝒦_local

w[人間] の絶対値が大きくなるほど揺らぎは小さくなる
→ 行動が予測可能になる
```

### 収束先は「人間」ではない

```
兎が形成する構造：

w[巨大・動く・報酬文脈] > 0
w[巨大・動く・接近急速] << 0  （こちらは変わらない）

「あの大きくて動くものが来ると
 草よりうまいものが現れる」

という具体的・局所的な w の配置
```

人間側が「仲良くなった」と認識していても、
兎側は「報酬付き巨大物体への w が正転した」だけかもしれない。

```
ラベルの抽象度ではなく
w の安定した配置が構造の実在性を決める
```

## ■ 7. 野生と飼育の構造的差異

```
野生の兎：
w[逃げる] >> w[近づく]
　→ 近づく w が形成される前に回避が優先
　→ 「人間＝今のところ無害な未知」のまま

飼育された兎：
反復接触・安定した報酬供給
　→ D3が「人間＝安全・報酬」の w を中核まで落とし込む
　→ w[近づく] が w[逃げる] を上回る領域が形成される
```

同じ基層構造（40億年SILN）を持ちながら、
反復接触によって θ の実質的な高さと w の配置が変わっている。

```
θ_wild    >> θ_domestic（同じ個体でも環境で変化する）
w_wild[人間] << 0
w_domestic[人間・安全文脈] > 0
```

## ■ 8. 赤ちゃんとの構造的同型性

兎の「人という構造の形成過程」は、
人間の乳児が「お母さんという構造を形成する過程」と
本質的に同型の可能性がある。

```
乳児の初期状態：
F = {大きい・温かい・音を発する・未知}
w[この刺激] ≈ 0 または弱い反発

反復接触と報酬（授乳・接触・安心）：
→ w[お母さん的刺激] が揺らぎながら正方向へ収束していく
→ 「お母さん」という w の配置が形成される
```

```
抽象化レベルは異なる
しかし w の形成メカニズムは同型

兎：局所具体的な w の配置
乳児：より抽象的な w へと拡張されていく
```

この違いは上層構造の発達によるものと解釈できる。

## ■ 総括

```
兎エージェントにおける「人という構造」の形成は：

1. 未知カテゴリとして入力される（ラベルなし）
2. 捕食者SILNとの不一致が保留状態を生む
3. 害なし反復が θ を緩やかに上げ w[反発] を弱める
4. 報酬が w[近づく] を正方向へ引き上げる
5. D3が再現性を確認しながら w の配置を安定させる
6. 揺らぎながら「報酬付き巨大物体への正の w」に収束する
```

収束先は「人間」という抽象概念ではない。
しかしその w の配置は実在し、行動を規定する。

```
ラベルの精度よりも w の安定した配置が構造を作る
```

人生はカオスの海を安定を手掛かりに航行する旅である。
兎も、その海を泳いでいる。
