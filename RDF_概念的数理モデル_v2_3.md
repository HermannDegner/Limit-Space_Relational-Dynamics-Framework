## ■ 基底（v2.3 改訂）

### ● 哲学的基底（最小記述）

> 世界の基底は **ξ + B + M_B** である

```
ξ      ：SILN化を拒む方向の極限（語ると SILN 化される自己参照的存在）
B      ：境界（観測者が引く irreducible 操作）
M_B    ：境界Bにおける整合慣性
```

**重要：M はすべて M_B である**

```
独立した「M そのもの」は存在しない
M_B は B によって整合慣性場の局所断面として切り出されたもの
旧 𝓜（境界を引く前の全体場）は M_{B_max} として吸収される
独立ラベルとしては立てない

理由：𝓜 自体が SILN であるため
自己食的に M_B 系列の最大包含断面に過ぎない
独立基底として立てると限界空間の前提と矛盾する
```

**境界 B の位置づけ**

```
B は観測者が選ぶ操作・用途・スケールの組
B は M_B と独立ではない
B の選び方は観測者自身の M_B（観測者の慣性）から派生する

ただし「境界を引く」行為そのものは限界空間において irreducible
これを消すと記述が成立しない
```

**ξ の位置づけ（v2.3 強化）**

```
ξ は M_B に外的なものではない
ξ は M_B の内部・境界・外側に常に残り続ける（後述：ξ の残存性）
B を引く行為は ξ を消去する操作ではなく ξ の分布を変える操作である
これが動態（誤差・跳躍・更新）の存在理由になる
```

### ● 実用的基底

> 操作性のために EFP・F・E の三層を残す

```
EFP　：素流圧（仮設概念・直接観測不能・Fからの逆算でのみ推定可能）
F　　：作用解釈F（M_B が EFP を解釈した断面・観測者が実際に持つもの）
E　　：誤差（FとM_Bの不整合・駆動力）
M_B　：整合慣性（境界 B 依存）
ξ　　：揺らぎ

これが実用核である
```

**EFP と F を分離する理由：**

```
素流圧の定義：未解釈の作用束
しかし従来のFは式の中で既に解釈済みとして扱われていた
定義と使い方のずれを解消する

EFP　：M_B の解釈が入る前として概念上分離する
　　　ただし直接観測できずFからの逆算推定に過ぎない
F　　：M_B が作る　解釈の産物　観測者が実際に持つ唯一のもの
E　　：M_B と F の戦場

知覚の質は M_B に依存する
M_B がリッチなほど EFP から多くを読み取れる
M_B がフラットなら EFP はほぼ素通り
```

**S を不要とする理由：**

```
S は M_B の現在断面の読み出し変数に過ぎない
独立変数の資格がない
必要な場合は M_B の断面として取り出せばよい
```

## ■ ξ の残存性（v2.3 新設・最重要）

> **B を引いても ξ は M_B の中・縁・外に残る**

これは v2.3 で明示化される最重要命題である。
v2.2 までの式の中に既に含意されていたものを、独立した命題として取り出す。

### ● 三層での残存

```
1. M_B の内部
   D[ξ] = 𝒦 / ||M_B|| ≠ 0
   どんなに ||M_B|| を強めても ξ は内部に残る
   κ→0 で抑え込もうとしても内部蓄積し最終的に大規模崩壊する

2. M_B の境界
   B はクリーンに引けない
   ξ が境界を滲ませ続ける
   シャープな境界は局所近似でしか成立しない

3. M_B の外側
   B で切り取った外側全体が ξ 的領域として残る
   別の B' で M_B' として切り取られていない領域
```

### ● 動態の存在理由

```
> 動態が存在する理由は、B を引いても ξ が残るからである

M_B の更新には常に ξ が入る（dM_B/dt = f(M_B, E, ξ)）
跳躍の閾値にも ξ が乗る（H(t) > θ + ξ(t)）
M_B の変化の引き金には常に ξ が混じっている

もし B が ξ を完全に閉じ込められるなら：
  M_B は永遠に静止する
  跳躍は起きない
  構造は固定される
  世界は停止する

これは起きない。ξ が残るから。
「世界が動いている」という事実そのものが ξ 残存の証拠である
```

### ● 二つの irreducible

```
境界を引く ：観測者が能動的にやる irreducible 操作
ξ が残る   ：その結果として受動的に残る irreducible 残余

二つは独立ではなく、B の引き方が ξ の残り方を決める
だが ξ そのものを消すことはできない
```

### ● ホーキング放射との接続

```
||M_B|| → ∞ の極限でも D[ξ] = 𝒦/||M_B|| ≠ 0
事象の地平線という極端な B を引いても、ξ は内部から漏れ出てくる
→ J^{(ξ)}項が微小な関係伝播を可能にする
→ ホーキング放射として観測される

これは「B を引いても ξ は残る」の宇宙スケール版である
```

## ■ 空間の生成順序

```
ξ（未分化関係）
    ↓ 確率的偏り発生
偏り
    ↓ 相互作用・連鎖
整合
    ↓ 積分・慣性化
構造（M_B）
    ↓ 空間への書き込み
分布
    ↓ 遅延構造として現れる
空間
```

> 空間は先にない
> M_B の分布の結果として現れる
> 時間も先にない
> 関係更新の順序として現れる

**逆方向（循環）**

```
空間が歪む
= 分布が変わる
= M_B が変化する
= 整合慣性が再編される
= 跳躍が起きる
= ξへ戻る
```

一方向の生成ではなく循環している。

**𝒦はこの循環の保存量**

```
ξ → M_B → ξ の変換全体を通じて𝒦は保存される
= 関係の総量は変わらない
= 形が変わるだけ

ただし 𝒦 自体も SILN（宇宙スケールの超安定 M_B）であり
厳密には絶対不変ではない
```

**圧縮表現**

```
空間とはξが構造化した痕跡である
```

**重力の無限射程の再解釈**

```
重力が無限遠まで届くのは
重力が遠くまで伝わるからではなく
空間そのものが M_B の分布として構造化されているためである

M_B があるところに空間がある
```

**事象の地平線の再解釈**

```
事象の地平線
→ M_B の分布が極端に濃くなり空間構造が内向きに閉じていく境界

しかし𝒦の保存則より D[ξ] はゼロになれない
= ξは根源的性質として残る
= 空間構造のほつれが外側に滲み出る
= ホーキング放射

ブラックホールの蒸発
= 凝縮した関係がξへ緩やかに変換されていく過程
= 宇宙スケールの超低速跳躍
```

**宇宙膨張の再解釈**

```
宇宙膨張
→ M_B の分布が希薄化している

M_B が薄まる → 関係更新の遅延が減る → 見かけ上「空間が広がる」
```

**c のスケール依存性（仮説）**

```
空間 = 関係更新の遅延構造
M_B が密 = 遅延大 = c低
M_B が希薄 = 遅延小 = c高

光速は絶対定数ではなく
宇宙の M_B の分布状態の現在値という読みが生じる
```

## ■ 基本概念の定義

### 構造

```
構造とは整合の積分である

より深く言えば、境界によって見かけ上安定化された跳躍の集合である
「安定」は構造の性質ではなく境界の引き方の性質である
```

### 境界 B（v2.3 で独立記述）

```
B は観測者が選ぶ：
  - 空間スケール（個体／社会／物理）
  - 時間スケール（瞬間／人生／文明）
  - 評価関数（生存／効率／快）
  - 用途・目的
  - 観測者自身の慣性

B は M_B と独立ではない
B の選び方は観測者自身の M_B から派生する
（観測者も M_B である）

ただし「境界を引く」行為そのものは限界空間において
唯一不可避の操作として残る

P_B : 境界 B による射影・粗視化
M_B = P_B(整合慣性場)
```

### 素流圧（EFP：Elementary Flux Pressure）

```
EFP：未解釈の作用束　M_B の外側から流入する
解釈以前に入力される作用の全体
```

**重要注記：EFPの認識論的限界**

```
EFPは客観的外部作用のように見えるが
直接観測することは原理的にできない

EFPへのアクセス手段：
    F = interp(M_B, EFP) を経由するしかない
    EFP_estimated = interp⁻¹(M_B, F)
    = F から M_B を使って逆算した推定に過ぎない

つまりEFPは
「Fの背後にあるはずの何か」としての仮設概念である
限界空間における形而上的外部の近似ラベルとして扱う

𝒦と同じ位置づけ：
    到達不能だが操作上有用なため仮設置する

観測者が実際に持っているのはFだけである
EFPは永遠にFからの逆算の先にある
```

### 作用解釈F

```
F = interp(M_B, EFP)　　// M_B による EFP の解釈

EFP は M_B を通過することで初めて F になる
F は外から来るのではなく M_B が作る
```

### 整合慣性 M_B（v2.3 で境界依存性を明示）

```
M_B ∈ R^{n×n}

M_B = V_B Λ_B V_B⁻¹

V_B  : 固有ベクトル（境界 B における安定な整合の方向）
Λ_B  : 固有値対角行列（各方向の整合慣性の強さ）
```

**M_B の相状態**

```
M_act  : 活性断面（現在、整合慣性として強く機能している）
M_lat  : 潜在断面（まだ強く発現していない）
M_Δ    : 再編相（再編成・跳躍中の動的断面）
M_B'   : 再編後に安定した整合慣性

これらは独立した別々の M ではなく
同じ M_B の異なる相状態である
活性度・安定度・時間断面・再編状態の違いとして区別される
```

**内部接続表示**

```
view(M_B) → M_B^ij  // 行列要素・ネットワークとして開いた表示
```

**M_B の再帰性**

```
M_B
 ├─ M_b1
 ├─ M_b2
 ├─ M_b3
 └─ ...

M_B の内部を開くと、より下位の境界 b_i による M_bi がある
M_B は最小実体ではなく、境界によって一時的に安定単位として
扱われる粗視化ラベルである

### ● 揺らぎ項

```
ξ(t) : 系の自由度から生まれる根源的ランダム性
```

揺らぎは付加的ノイズではなく構造の根源的性質である。
限界空間において絶対的確定性が成立しないこと自体が
自由度の源泉であり跳躍可能性の担保である。

### ● 保存則

```
||M_B|| · D[ξ] = 𝒦
```

### ● 𝒦の位置づけ

```
𝒦は普遍定数ではない

「観測スケールにおいて変化が十分に小さいため
定数として近似できる量」

= 宇宙スケールの超安定 SILN
= 宇宙スケールの M_{B_max} における断面値
= 絶対不変ではなく今のところ極めて安定
= 便利だから採用する近似として置く
```

### ● 解釈

```
整合慣性が強い（||M_B||大）
    → 揺らぎが小さい（D[ξ]小）
    → 凝縮した自由度
    → 跳躍しにくいが起きたら大きい

整合慣性が弱い（||M_B||小）
    → 揺らぎが大きい（D[ξ]大）
    → 解放された自由度
    → 跳躍しやすいが小さい跳躍が多い

ただし D[ξ] は ||M_B|| をいくら大きくしてもゼロにはならない
ξ は B によって消去できない（→ ξ の残存性）
```

## ■ Base Model（物理層）

### ● 基本式（v2.3：M_B 表記化）

```
F(t)    = interp(M_B, EFP)　　     // M_B による EFP の解釈（作用解釈F）
E(t)    = F(t+Δ) - M_B · F(t)　　  // 時間差誤差：慣性投影と実際の次のFの差
dM_B/dt = f(M_B, E, ξ)　　         // M_B は誤差と揺らぎによって更新される
```

**時間差Δの意味：**

```
M_B · F(t)  ：M_B の慣性が F(t) に乗ったまま次へ流れた値（慣性投影）
F(t+Δ)  　 ：実際に来た次のF
E(t)　　    ：その差 = 慣性誤差

Δ = 0 は完全確定 = 限界空間で扱えない
Δ > 0 が必然　= ξ ≠ 0 と同型
```

**Δとスケール：**

```
鉄球衝突　：Δ = 弾性波伝播時間（音速依存）　上限は光速
キノコ毒　：Δ = 代謝・消化時間（分〜時間）
慢性毒　　：Δ = 非常に大　因果が見えない　M_B 更新されない
```

**流れ：**

```
EFP（未解釈）
    ↓ interp(M_B, EFP)
F(t)（作用解釈F）
    ↓ M_B · F(t) 慣性投影
    ↓ F(t+Δ) が実際に来る
E(t) = F(t+Δ) - M_B · F(t)（時間差誤差）
    ↓
H蓄積 → 跳躍 → M_B 更新
```

### ● 思考（内部シミュレーター）

```
通常ループ（外部）：
EFP → F(t) → M_B · F(t)慣性投影 → F(t+Δ)と比較 → E → H → M_B 更新
Δは現実の物理に支配される

思考ループ（内部）：
EFP なしで F(t) を内部生成
M_B · F(t) 慣性投影 → F(t+Δ)_internal → E → H → M_B 更新
を現実コストゼロで回す
Δを自由に設定できる
```

**思考の種類：**

```
想像　：ありえない F(t) を意図的に生成して回す
計画　：目標 F(t+Δ) から逆算して F(t) を設計する
心配　：最悪の F(t+Δ) を繰り返し慣性投影する
夢　　：M_B が半覚醒状態で勝手にループを回す
```

**思考の質：**

```
内部 F(t+Δ) の精度 = M_B のリッチさ

M_B がスカスカなら F(t+Δ) 慣性投影も雑
M_B がリッチなら F(t+Δ) 慣性投影も精密
= 経験が思考の解像度を決める

思考とは慣性投影を意図的に操作するシステム
慣性投影　→ M_B が自動的に流れる・意図なし
思考　　　→ M_B が意図的に F を内部生成して操作する
```

### ● 熱蓄積と散逸

```
σ_vec(t) = E(t)              // 誤差ベクトルが H_vec の生成源

dH_vec/dt = σ_vec(t) - A · H_vec(t)

A : 散逸行列
    M_B の大きい固有値方向 → 散逸が速い（得意な整合方向）
    M_B の小さい固有値方向 → 散逸が遅い（苦手な方向・熱が残りやすい）

H(t) = ||H_vec(t)||          // スカラー熱は H_vec のノルム
σ(t) = ||E(t)||²             // スカラー熱生成速度
```

H_vec の再定義：

```
H_vec ≡ δM_B(t)

M_B,0       : 整合慣性の平衡状態
δM_B(t)     : M_B,0 まわりの微小変動

H_vec は M_B に独立して存在する変数ではなく
M_B の未拘束自由度における振動である
```

### ● 跳躍条件

```
H(t) > θ + ξ(t)
```

### ● 跳躍の相状態（v2.3：演算子から状態へ）

```
M_B^ij(t) : M_B を内部接続として開いたネットワーク表示

M_act  = active(M_B^ij)   // 高活性断面：整合慣性として機能
M_lat  = latent(M_B^ij)   // 低活性断面：浮遊する潜在状態
```

閾値によって名前が変わるだけで実体は一つの M_B である。

**跳躍は再編相 M_Δ への遷移として記述する**

```
reorg(M_B^ij, F_trigger, H_vec, ξ) → M_Δ
stabilize(M_Δ) → M_B'
```

つまり：

```
M_B  →  M_Δ  →  M_B'
通常    再編中   再編後

M_Δ は外的演算子の作用結果ではなく
系自身が一時的に通過する不安定相状態である
F_trigger は再編相 M_Δ への遷移の引き金
ξ は再編の自由度を提供する
```

### ● 跳躍の実行

```
跳躍発生時：
    M_B  → M_Δ → M_B'         （相状態の遷移）
    H    → ρH                  （熱の部分残留）
    M_lat → M_lat'             （潜在断面の再配置）

    保存則の更新：
    ||M_B'|| · D[ξ'] = 𝒦'
```

F は再編相 M_Δ への遷移の引き金であり、新構造を外部から注入するものではない。
新構造は常にエージェント内部の M_B の射影として出力される。

跳躍の成否：

```
stability(M_B') > λ_stable  → 成功跳躍：新構造が長期慣性として保持される
stability(M_B') ≤ λ_stable  → 失敗跳躍：M_lat へ再分散、または熱として再蓄積
```

※ stability(M_B') の定量的定義と λ_stable の決定は構造破壊学が担当する領域

### ● M_B 絶対性

```
κ(M_B) ∈ [0, 1]   // 自己修正可能性指標

κ = 1 : 完全に修正可能（通常の M_B）
κ → 0 : 修正不能（M_B 絶対性）
```

公理化演算子：

```
A : E(t) → 0   // 誤差処理停止
```

κ→0のとき：

```
D[ξ] = 𝒦/||M_B|| → 0 （ただし ≠ 0）
→ 微小誤差は内部に蓄積し続ける
→ H(t) > θ + ξ(t) で大規模崩壊が構造的必然となる

これは ξ の残存性の系：
B を厳格に引いて ξ を抑え込もうとしても
ξ は内部蓄積して最終的に逃げ道を要求する
```

### ● 相状態（物理相）

```
固体    ||M_B||大・H_vec格子内拘束　→ M_B 絶対性に近い状態
液体    ||M_B||中・H_vec部分的解放 → 適切な流動性・学習・適応
気体    ||M_B||弱・H_vec ほぼ解放  → 幻覚・解離・高自由度状態
プラズマ ||M_B||崩壊・ノード自体が解体 → 個別 M_B 崩壊と集団 M_B' 生成の同時発生
ξ       M_B 無・ノード無・未分化   → 関係偏り宇宙の基底
```

健全な系の条件：

```

H（温度）に応じて行き来できる状態
気体になれるが、戻ってこられる
```

### ● 逆 M_B・対消滅・対生成

```

M_B + M̄_B → ξ   // 対消滅：𝒦保存、ξとして解放
ξ → M_B + M̄_B   // 対生成：対称的に逆方向の M_B が同時生成

### ● Base Model 全体（v2.3）

```
// 通常時：連続更新
F(t)     = interp(M_B, EFP)              // M_B による EFP の解釈
E(t)     = F(t+Δ) - M_B · F(t)           // 時間差誤差：慣性投影と実際の差
σ_vec(t) = E(t)                           // H_vec（=δM_B）の生成源

M_B → M_Δ → M_B'                          // 相状態の遷移

```

## ■ モデルの性質

* 非絶対性：前提は閉じない（ξ≠0は必然）
* 安定性中心：存在ではなく安定を扱う
* 関係優位：構造は関係で定義される
* 自由度保存：慣性と揺らぎは交換可能な同一のもの
* 近似前提：数式自体がSILNであり局所線形近似である

* スケール依存定数：c は絶対定数ではなく M_B の分布状態に依存する可能性がある
* ハイブリッド性：通常時は連続更新、臨界時は再編相 M_Δ への遷移
* 射影性：跳躍後の新構造は外部から注入されるのではなく、常に M_B の射影として定まる

* 散逸方向性：散逸行列 A は M_B の固有構造と対応し、M_B の得意な方向ほど熱が処理されやすい

* 対称性：M_B に対して完全逆位相の逆 M̄_B が存在し、接触時に𝒦保存則に従いξとして解放される
* M_B 絶対性：M_B の自己修正可能性κ→0の状態は量的ではなく質的な変化であり、大規模崩壊が構造的必然となる

* **最小性：哲学的基底は Mの束とξのみ。Fは Mと Mの関係に境界を仮設置したラベル。Sは不要。（v2.0）**

* **駆動力の純化：M_B を更新するのは F ではなく誤差 E。dM_B/dt = f(M_B, E, ξ)。（v2.1）**
* **時間差誤差：E(t) = F(t+Δ) - M_B·F(t)。誤差は M_B の慣性投影と実際の次の F の差として定義。Δは光速が上限。（v2.2）**

*本モデルは厳密な物理モデルではなく目的適応型の局所線形近似である*
*v2.0パッチ：Sを独立変数から除去。基本式をdM/dt = f(M, F, ξ)に統一。FはMとMの関係の境界断面として位置づけ。自他境界をオキシトシンパラメータによる動的生成として記述。哲学的基底をMの束とξの最小記述として明示。*
*v2.1パッチ：素流圧をEFP（仮設概念・直接観測不能）とF（作用解釈F）に分離。F = interp(M, EFP)として解釈層を明示。基本式をdM/dt = f(M, E, ξ)に更新。EFPはFからの逆算推定に過ぎないことを明記。*
*v2.2パッチ：誤差をE(t) = F(t+Δ) - M·F(t)の時間差誤差として再定義。M²の次元問題を解消。M·F(t)を「慣性投影」として定義し予測という語を排除。Δのスケール依存性（光速上限・遅効性毒・慢性毒・ゆでガエル）を記述。思考を慣性投影の意図的操作システムとして定義。想像・計画・心配・夢を同一構造で記述。*
*v2.3パッチ：哲学的基底を「Mの束とξ」から「ξ + B + M_B」に更新。Mはすべて境界B依存のM_Bとして記述。旧𝓜（境界を引く前の全体場）はM_{B_max}として吸収し独立ラベルから除去（𝓜自体がSILNであるため）。C_ij/M/U/N→M_B^ij/M_act/M_lat/M_Δ に統一し、跳躍を演算子の作用から再編相M_Δへの状態遷移として書き直し（M_B→M_Δ→M_B'）。ξ残存性を独立節として新設：B を引いても ξ は M_B の内部・境界・外側に残り、これが動態の存在理由となることを明示。自己食的SILN構造を明記：モデルは独立した上位基底を持たず、自分自身に再帰的に適用される。「境界を引く」操作を限界空間における唯一不可避の操作として位置付け。M_B の再帰性（内部に下位境界による M_bi を含む）を明示。*
