# SILN分解の数理フレームワーク v1.1

---

## ■ 概要

SILN（Survival-biased Integration of Local Linear approximations in a Nonlinear world）は
そのままでは抽象度が高すぎるため、分解なしでは意味を持たない。

したがって、実用上は

**「境界を仮定し、整合慣性 M を仮設する操作」**

として扱う。

※ Mは「抽出」ではなく「仮設」である。直接観測できないため、
　行動・出力・反応のパターンから推定した仮説に過ぎない。

---

## ■ 全体プロセス

```
① ラベル設定（対象定義）
② 境界仮定（仮説）
③ 状態展開（S, M, F, ξ, H）
④ 構造仮設（Mの推定）
```

---

対象となるSILNをラベルとして定義する。

例：

- 国家

- テセウスの船

※ この段階ではまだ中身は存在しない

## ■ ② 境界仮定（仮説）

SILN分解における最重要ステップ。

以下を仮定する：

- 空間スケール（個体／社会／物理）
- 時間スケール（瞬間／人生／文明）
- 評価関数（生存／効率／快）

```
仮説 = 観測者が引いた境界
```

※ 限界空間では絶対前提は存在しないため必須

## ■ テセウスの船への適用

```
SILN = (M, S_phys, L, W)
```

- S_phys：物理構造
- L：ラベル
- W：社会的関係

本質：

```
船 = M(t) の軌跡
```

観測者ごとにMは微妙に異なる。
造船士のM、乗組員のM、港の人のM、それぞれ別物である。
それでも「テセウスの船」というラベルが共有できるのは、
Mが完全一致しなくても機能する共有装置としてラベルが働くからである。

---

## ■ ラベルの二面性

ラベルは以下の二面性を持つ：

```
① 境界を作り、思考の自由度を下げる
② 観測者間のMのズレを束ね、共有を可能にする
```

Mが完全に一致していればラベルは不要。
完全にバラバラならば共有できない。
**微妙にズレているちょうどその状態**でラベルが意味を持つ。

ラベルとは観測者間の整合誤差を吸収するバッファである。

## ■ SILN分解の限界性

SILN分解には構造的な限界がある。

**Mの検証問題**

ラベルのMを強固にするには、複数の観測者から収集する操作が有効である。
しかしその操作自体が素流圧としてMに作用する。
したがってラベルのMは観測プロセスと不可分であり、
観測前の純粋なMには到達できない。

**絶対解は存在しない**

```
M_観測者A ≠ M_観測者B ≠ M_観測者C ...
```

複数観測によってラベルの輪郭は統計的に近似できるが、
真のMへの収束は保証されない。

SILN分解は絶対解を出すツールではなく、
**目的に応じた有効な近似を仮設するツール**である。

限界空間において確定は局所的確定に過ぎない。
SILN分解もその例外ではない。
