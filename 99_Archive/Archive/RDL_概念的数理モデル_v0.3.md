# RDL 概念的数理モデル v0.3

---

## ■ 理論的基盤

本モデルは以下を統合した近似記述である：

```
限界空間        → 認識論的基盤（絶対的構造を前提しない）
関係偏り宇宙    → 宇宙スケールの記述（偏りと関係のみを基底とする）
非平衡熱力学    → 物理的骨格（散逸構造・分岐点）
RDL／SILN      → 中間スケールの言語
```

> 本モデルは真理の記述ではなく、
> **整合の動態を操作可能な形で表現するための近似モデル**である

---

## ■ 基底

> 宇宙の基底は未分化な相互作用（偏り）である

* 存在を前提としない
* 力を前提としない
* 関係の成立と持続のみを扱う
* 「内部」「外部」の絶対的境界は成立しない

---

## ■ 基本概念の定義

### 構造

```
構造とは整合の積分である
S(t) ∈ R^n
```

ある時点における関係の安定パターンの表現。
物体・概念・個体・社会いずれも同じ枠組みで扱う。

### 素流圧

```
F(t) ∈ R^n
```

構造に作用するあらゆる相互作用。
内部外部の区別なく、作用してくる関係の全体。

### 整合慣性

```
M ∈ R^{n×n}
```

固有値分解：

```
M = VΛV⁻¹

V  : 固有ベクトル（安定な整合の方向）
Λ  : 固有値対角行列（各方向の整合慣性の強さ）
```

使われる経路ほど慣性が強まる（Ohm's law的）。
慣性の強い方向に処理が向かう。

### エネルギー

```
エネルギーとは
構造がより整合した状態へ再配置される際の差分である
```

## ■ Base Model（物理層）

### ● 状態方程式

```
dS/dt = -MS + F(t)
```

* MS  : 現構造による整合処理（慣性の強い方向へ流れる）
* F(t): 素流圧（あらゆる作用との関係性）

### ● 誤差（未整合残差）

```
E(t) = F(t) - MS(t)
```

現構造で処理しきれなかった素流圧の残差。熱源となる。

### ● 熱蓄積と散逸

```
σ(t)  = ||E(t)||²        // エントロピー生成率（誤差熱の発生速度）
dH/dt = σ(t) - αH(t)     // 熱の蓄積と散逸
```

* σ : 誤差から生じる熱生成
* α : 散逸係数（構造が熱を逃がす速度）

**動態**

```
σ > αH が続く → 熱が蓄積 → 跳躍へ向かう
σ < αH が続く → 熱が減衰 → 安定へ向かう
```

### ● 跳躍条件

```
H(t) > θ(t)
```

θ(t) は固定値ではなく内部状態によって動的に変化する。

### ● 跳躍の実行

```
跳躍とは探索機構である

跳躍発生時：
    H  → 0      （熱の解放）
    S  → S'     （新しい状態への遷移）
    M  → M'     （固有構造の再編）
```

M' はその時点での素流圧環境全体との
新しい整合パターンとして決まる。
予測不能だが、より高い σ を散逸できる構造へ向かう傾向を持つ。
多くはゴミである（生き残った構造のみが観測される）。

## ■ Base Model 全体

```
dS/dt = -MS + F(t)           // 状態方程式
E(t)  = F(t) - MS(t)         // 誤差（未整合残差）
σ(t)  = ||E(t)||²            // 熱生成
dH/dt = σ(t) - αH(t)         // 熱蓄積
跳躍  ← H(t) > θ(t)          // 分岐条件
M'    ← F環境との新整合       // 跳躍後の構造再編
```

## ■ 次元削減（局所近似）

高次元のMは文脈ごとに支配的な固有値が少数に限られる。

```
固有値でランク付け：λ₁ >> λ₂ >> ... >> λₙ

支配的なk個のみ保持
→ 実効次元 k << n の低次元空間で近似
```

低次元近似：

```
S(t) ≈ Vₖ a(t)           // k次元座標へ投影

da/dt = -Λₖ a + Vₖᵀ F(t)

Λₖ ∈ R^{k×k}（対角行列）
```

これはSILNにおける「局所線形近似」に対応する。
次元数 k は文脈依存であり Fluctuation MOD 側から渡される。

---

## ■ 跳躍の共鳴条件

跳躍は外部きっかけ単体でも内部蓄積単体でも起きにくい。

```
共鳴条件：
F(t) · H(t) > θ_leap

素流圧と蓄積熱の内積的共鳴が閾値を超えた時に跳躍
```

過冷却状態への小さな振動（ニュートンのリンゴ）が
蓄積熱と共鳴して相転移を起こす構造。

---

## ■ SILNネットワーク拡張

単一構造だけでなく複数構造の相互作用を扱う。

```
S^(i)_{t+1} = -M^(i) S^(i) + Σⱼ W_{ij} S^(j) + F^(i)(t)
```

* W_{ij} : 構造間の相互作用強度
* 複数の整合が同時に存在し相互作用する

**跳躍の定着条件**

```
個体iの跳躍後M'_i が
他の構造jの整合慣性と共鳴した時
→ M'_i の固有構造が
　 より大きなスケールで安定パターンとして定着する
```

ニュートンの跳躍がハレーという素流圧と共鳴し
社会SILNとして定着したプロセス。

## ■ 宇宙基本サイクルとの対応

```
偏り発生          → F(t) の初期揺らぎ
相互作用連鎖      → dS/dt = -MS + F(t)
整合による構造形成 → Mの固有構造の蓄積
密度上昇          → H(t) の蓄積
限界到達          → H(t) > θ(t)
跳躍（破壊）      → M → M'
再配置            → 新しい固有構造の形成
新構造生成        → より高い σ を散逸できる M'
```

## ■ Fluctuation MOD 接続点

Base Model への介入は以下の点のみ。
Base Model の構造自体には触れない。

```
M（またはΛₖ）への注入
    → 整合慣性の方向と強度
    → 特定チャンネルの固有値を選択的に操作

α への注入
    → 散逸速度
    → 高い場合：熱が溜まりにくい・安定志向
    → 低い場合：熱が溜まりやすい・跳躍しやすい

θ への注入
    → 跳躍閾値の動的変化
    → 極限状態では θ が急低下

W_{ij} への注入
    → 構造間結合強度（オキシトシン的境界拡張など）
```

**式として**

```
α_eff   = α  × g_α(mod_params)
θ_eff   = θ  × g_θ(mod_params)
Λ_eff   = Λ  × G_Λ(mod_params)
W_eff   = W  × G_W(mod_params)
```

Base Model はスケールを問わず同一の式で動く。

| スケール | 構造S | 素流圧F | 熱H | 跳躍 |
|---|---|---|---|---|
| 宇宙 | 恒星 | 重力・核融合 | 内部エネルギー | 超新星爆発 |
| 生命 | 種 | 環境変化 | 遺伝的誤差蓄積 | 種の分岐 |
| 社会 | 制度 | 思想・経済 | 不満・矛盾蓄積 | 革命 |
| 個人 | 世界観 | 経験・情報 | 誤差熱蓄積 | 気づき・転換 |
| 認知 | 整合慣性 | 素流圧 | 接続候補蓄積 | ミーム定着 |

## ■ モデルの性質

* 非絶対性：前提は閉じない
* 安定性中心：存在ではなく安定を扱う
* 関係優位：構造は関係で定義される（内部外部の境界なし）
* 拡張前提：MOD追加により個体・状況・スケールへ対応

## ■ 残課題

* M' 生成の具体的メカニズム（跳躍後の固有構造再編）
* MOD接続関数 g_α, g_θ, G_Λ, G_W の関数形
* W_{ij} の学習則（共鳴による結合強化）
* 微小跳躍の積み重ねによるM変化の記述

## ■ 総括

```
dS/dt = -MS + F(t)           // 構造は関係の中で更新される
E(t)  = F(t) - MS(t)         // 整合しきれない残差が熱になる
σ(t)  = ||E(t)||²            // 熱の生成速度
dH/dt = σ(t) - αH(t)         // 熱は蓄積し散逸する
跳躍  ← H(t) > θ(t)          // 限界を超えると構造が再編される
M'    ← F環境との新整合       // 新構造は関係の中で決まる
```

> 構造とは整合の積分である
> 跳躍とは探索であり、構造生成の源である
> 宇宙は偏りの相互作用連鎖の中で
> 整合によって構造を形成し
> 跳躍によってそれを更新し続ける動的系である
