# RDL 概念的数理モデル v0.2

---

## ■ 概要

本モデルは、関係力学言語（RDL）における
「整合」「整合慣性」「素流圧」「跳躍」を
散逸構造論を基盤とした数理構造として記述するものである。

**設計方針**

```
Base Model（物理層）
    普遍・固定・スケール不問
    非平衡熱力学・散逸構造の基礎式

        ↓ インターフェース（三接続点）

Fluctuation MOD（揺らぎ層）
    個体・状況・スケール依存
    Base Modelには触れず係数を渡すのみ
```

本モデルは真理の記述ではなく、

> **整合の動態を操作可能な形で表現するための近似モデル**

として位置付けられる。

---

## ■ Base Model（物理層）

### ● 状態（構造）

```
S(t) ∈ R^n
```

ある時点における認識・内部状態の表現。

---

### ● 整合慣性行列

```
M ∈ R^{n×n}
```

固有値分解：

```
M = VΛV⁻¹

V：固有ベクトル（安定な整合の方向）
Λ：固有値対角行列（各方向の整合慣性の強さ）
```

---

### ● 状態方程式

```
dS/dt = -MS + F(t)
```

- MS：現構造による予測・維持
- F(t)：素流圧（外部および内部からの作用）

---

### ● 誤差

```
E(t) = F(t) - MS(t)
```

現構造で処理できなかった残差。熱源となる。

---

### ● 熱蓄積と散逸

```
σ(t)  = ||E(t)||²       // エントロピー生成率（誤差熱の発生速度）
dH/dt = σ(t) - αH(t)    // 熱の蓄積と散逸
```

- σ：誤差から生じる熱生成
- α：散逸係数（系が熱を逃がす速度）

**動態の解釈**

```
σ > αH が続く → 熱が蓄積 → 跳躍へ向かう
σ < αH が続く → 熱が減衰 → 安定へ向かう
```

---

### ● 跳躍条件と跳躍後

```
跳躍条件：H(t) > θ

跳躍後：
    M  → M'   （固有構造の再編）
    H  → 0    （熱の解放）
    S  → S'   （新しい状態への遷移）
```

跳躍後の M' は素流圧環境に依存して決まる。
予測不能だが、より高い σ を散逸できる構造へ向かう傾向を持つ。

---

### ● Base Model 全体

```
dS/dt = -MS + F(t)          // 状態方程式
E(t)  = F(t) - MS(t)        // 誤差
σ(t)  = ||E(t)||²           // 熱生成
dH/dt = σ(t) - αH(t)        // 熱蓄積
跳躍  ← H(t) > θ            // 分岐条件
```

## ■ 次元削減（局所近似）

完全版 M は高次元だが、文脈ごとに支配的な固有値は少数に限られる。

```
固有値でランク付け：λ₁ >> λ₂ >> ... >> λₙ

支配的なk個の固有値・固有ベクトルのみ保持
    → 実効次元 k << n の低次元空間で近似
```

低次元近似：

```
S(t) ≈ Vₖ a(t)          // k次元座標へ投影

da/dt = -Λₖ a + Vₖᵀ F(t)

Λₖ ∈ R^{k×k}（対角行列）
```

Λₖ が対角になることで計算が大幅に軽くなる。
これはSILNにおける「局所線形近似」に対応する。

次元数 k は文脈依存であり、Fluctuation MOD 側から渡される。

---

## ■ Fluctuation MOD 接続点

Base Model への介入は以下の三点のみ。
Base Model の構造自体には触れない。

```
M（またはΛₖ）への注入
    → 整合慣性の方向と強度を変える
    → 特定チャンネルの固有値を選択的に操作

α への注入
    → 散逸速度を変える
    → 高い場合：熱が溜まりにくい・安定志向
    → 低い場合：熱が溜まりやすい・跳躍しやすい

θ への注入
    → 跳躍閾値を動的に変える
    → 極限状態では θ が急低下し通常では起きない跳躍が発生
```

**式として**

```
α_eff = α × g_α(mod_params)
θ_eff = θ × g_θ(mod_params)
Λ_eff = Λ × G_Λ(mod_params)
```

## ■ MOD 実装例（神経物質 MOD）

```
ドーパミン   → F(t) の入力感度フィルタ
             　探索軸の α を下げ熱が溜まりやすくする

セロトニン   → α への注入
             　高い場合：散逸速度↑ → 熱が溜まりにくい
             　低い場合：散逸速度↓ → 誤差に敏感

ノルアドレナリン → θ への注入
             　生存関連誤差検知時に θ を急低下
             　生存軸の Λ を選択的に増幅

オキシトシン → M の境界拡張
             　自己以外の対象を整合慣性の内部に組み込む
```

## ■ スケール横断性

Base Model はスケールを問わず同一の式で動く。
Fluctuation MOD の設定のみが異なる。

| スケール | 蓄積 H | 跳躍 | 生成 |
|---|---|---|---|
| 宇宙 | 核融合限界 | 超新星爆発 | 重元素・新星系 |
| 生命 | 遺伝的誤差 | 種の分岐 | 生物多様性 |
| 社会 | 不満・思想蓄積 | 革命 | 新制度・文化 |
| 個人 | 誤差熱蓄積 | 気づき・転換 | 新整合構造 |
| 認知 | 接続候補 | ミーム定着 | 概念の拡張 |

## ■ モデルの性質

- 非絶対性：前提は閉じない
- 安定性中心：存在ではなく安定を扱う
- 関係優位：構造は関係で定義される
- 拡張前提：MOD追加により個体・状況・スケールへ対応

---

## ■ 残課題

- MOD 接続関数 g_α, g_θ, G_Λ の具体的な関数形
- 跳躍後の M' 生成メカニズムの記述
- 複数構造間のネットワーク拡張（エージェント間伝播）
- 共鳴条件 F(t) · H_internal の操作的定義

---

## ■ 総括

```
dS/dt = -MS + F(t)
E(t)  = F(t) - MS(t)
σ(t)  = ||E(t)||²
dH/dt = σ(t) - αH(t)
跳躍  ← H(t) > θ
```

> 整合とは、素流圧を既存の固有構造へ投影し誤差を散逸する過程である
> 跳躍とは、散逸しきれない熱が閾値を超えたときに起きる固有構造の再編である
> 可能性空間の拡張は、この跳躍によってのみ起きる

---

*本モデルは厳密な物理モデルではなく目的適応型の近似である*
*Base Model は普遍層として固定し、個別性は全て Fluctuation MOD が担う*
