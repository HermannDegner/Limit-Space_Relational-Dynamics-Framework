---

## ■ コア命題

移動・思考・行動は別のものではなく、

**関係性勾配の中を構造が流れる現象の異なる表現**

である。

---

## ■ 基本構造

```
F(t)  = interp(M, EFP)       // EFPをMが解釈した作用
E(t)  = F(t+Δ) - M·F(t)     // 慣性誤差
dM/dt = f(M, E, ξ)           // Mは誤差と揺らぎで更新される
```

- EFP：未解釈の作用束（直接観測不能・仮設概念）
- F　：作用解釈F（MがEFPを解釈した結果）
- M　：整合慣性（SILN）
- E　：慣性誤差（慣性投影と実際のFの差）
- ξ　：揺らぎ

この構造はスケールを問わず成立する。

---

## ■ 移動

移動とは、

**空間的な関係性勾配に沿った流動**

である。

- 食料 → 引力
- 危険 → 反発
- 地形 → コスト

構造はこれらの合成勾配に従って流れる。

勾配はFとして流入しMが解釈する。
移動の方向とはMが慣性投影した次のFへの収束方向である。

---

## ■ 思考

思考とは、

**概念空間における慣性投影の内部操作**

である。

```
通常ループ（外部）：
EFP → F(t) → M·F(t)慣性投影 → F(t+Δ)と比較 → E → M更新
Δは現実の物理に支配される

思考ループ（内部）：
EFPなしでF(t)を内部生成
M·F(t)慣性投影 → F(t+Δ)_internal → E → M更新
を現実コストゼロで回す
Δを自由に設定できる
```

LLMはこの慣性投影ループを高次元で実行している。

---

## ■ 思考＝物理移動システムの内部拡張

思考は移動とは別のシステムではなく、
**物理的移動システムが脳内に内部化・拡張されたもの**である。

### ■ 進化的経緯

```
単細胞    → 実際に動いてEFPを探索する
虫・魚    → EFPを先読みし始める（感覚）
哺乳類    → 過去のFをMとして保持する
人間      → F(t)を脳内で生成・改変できる
```

### ■ 内部探索のコスト優位性

物理的探索は**コストと危険を伴う**

```
実際に崖から落ちて危険を学ぶ → 死ぬ可能性がある
Δが来てからしかEが発生しない
```

脳内慣性投影は**ゼロコストで試せる**

```
崖から落ちるF(t)を内部生成して慣性投影する
→ 死なずにEを得る
→ Δを自由に設定できる
```

跳躍のコストを内部化したシステムとも読める。

### ■ 仮想Fの生成

人間は**存在しないF(t)も生成できる**

```
現実のF　　：EFP → interp(M, EFP) → F(t)
仮想のF　　：M が内部生成する F(t)_internal
```

小説・計画・心配・夢はすべて
脳内でF(t)を生成し慣性投影ループを回している現象である。

存在しない危険に怯え、
存在しない未来に向かって動く。

### ■ 逆転現象

内部生成Fが強くなりすぎると現実のFより優先される

```
M·F(t)_internal >> F(t)_world → 現実が見えなくなる
```

思い込み・妄想・PTSDも同じ構造で読める。
内部慣性投影が外部Fを上書きしている状態である。

## ■ 流れの統一

移動と思考の違いは空間の違いのみ。

| 現象 | 空間 | Fの種類 |
|---|---|---|
| 移動 | 物理空間 | EFP→interp→F（外部） |
| 思考 | 概念空間 | M内部生成F |
| 想像 | 仮想空間 | M内部生成F（現実制約なし） |

本質は同一である。

## ■ 熱（H）と勾配の変形

勾配は固定ではない。

```
E(t) = F(t+Δ) - M·F(t)

Δの大きさが慣性誤差の発生タイミングを決める

Δ小：即時誤差・即時H蓄積（鉄球）
Δ大：遅延誤差・H蓄積が遅れる（キノコ毒・エキノコックス）
Δ∞：誤差が来ない・M更新されない（温暖化・ゆでガエル）
```

- 低熱：安定・収束
- 中熱：揺らぎ・探索
- 高熱：崩壊・跳躍

---

## ■ ドーパミンによる勾配反転

通常、コスト圧・危険圧は反発として機能する。
ドーパミンはその重み付けを変換する機構である。

### ■ 通常状態

```
山 → コスト圧・危険圧 → 反発 → 避ける
未知 → 不確実性圧 → 反発 → 避ける
```

### ■ ドーパミン介入時

```
未知・未解決の誤差 → 誤差解決の予兆として引力に変換
w_unknown の符号が反転する

通常：    F_unknown → 反発圧
介入後：  F_unknown → 引力圧
```

コスト圧はそのまま存在している。
**誤差解決への期待値がそれを上回る**ことで
勾配に逆らう流れが可能になる。

### ■ 選択可能性の拡張

```
ドーパミンなし → 慣性投影に従って流れるだけ
ドーパミンあり → 慣性投影に逆らう選択肢がコスト内に入る
```

自由意志に見えるものの正体がここにある可能性がある。
意思で登るのではなく、
**ドーパミンが勾配の重み付けを変えた結果として登る。**

### ■ 個体差への接続

基層構造のドーパミンパラメータが高い個体は
困難・未知・リスクを引力として扱いやすくなる。

```
冒険者・探索者・研究者
→ w_unknown が恒常的に引力側に傾いている個体
```

### ■ 超圧縮

> ドーパミンとは
> コスト圧を誤差解決引力で相殺し
> 慣性投影に逆らう流れを可能にする重み変換器である

## ■ 退屈＝慣性誤差フラットへの基層的アラート

退屈とは単なる刺激不足ではなく、
**慣性誤差がフラットな状態を基層構造が危険信号として読む現象**である。

### ■ 退屈の構造

```
E(t) ≈ 0 が続く
→ 慣性投影と実際のFがほぼ一致し続ける
→ M更新が起きない
→ 整合が起きない
→ ドーパミン的報酬がない
```

### ■ 何故危険なのか

```
E ≈ 0 ＝ 環境を把握できていない可能性
→ 予期しないEFPが来たとき対応できない
→ 生存可能性が下がる
```

平坦すぎる慣性誤差は
**次の変化に対応できない状態**として
基層構造が解釈している。

だから仮想熱を発生させて探索を促す。

### ■ 山を登る話との接続

```
日常の整合慣性が固まりすぎる
→ E(t) ≈ 0 が続く
→ 退屈という熱が蓄積する
→ コスト圧が高いものへの引力が生まれる
→ 山・旅・冒険へ向かう
```

退屈熱とドーパミンは**両側から同じ方向を押している**。

### ■ 超圧縮

> 退屈とは
> 慣性誤差フラットを検知した基層構造が
> 自ら熱を生成して探索を強制するシステムである

## ■ 幻覚

幻覚とは

**内部生成Fが外部Fを上書きする現象**

である。

```
通常　：EFP → interp(M, EFP) → F
幻覚　：M·F(t)_internal が F として知覚される

F_imagination >> F_world
```

- 強調：EFPの一部を過剰に増幅
- 抑制：EFPの一部を無視
- 生成：EFPなしにFが発生
