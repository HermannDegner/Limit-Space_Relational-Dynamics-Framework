# 合成型非ユークリッド移動アーキテクチャ

---

## ■ コアコンセプト

> **線形合成で方向を決め、非ユークリッドで軌道を歪める**

---

## ■ 基本構造

```
F_total = Σ F_i
v ← v + F_total
v ← Warp(v, M(x))
x ← x + v
```

---

## ■ 役割分担

### ● F（素流圧）
- 行きたい方向
- 目的・欲求・回避

### ● M（整合慣性）
- 空間の歪み
- 通りやすさ
- 曲がりやすさ

---

## ■ 直感

| 要素 | 意味 |
|------|------|
| F | 意志のベクトル |
| M | 地形・空間 |
| Warp | 地形による変形 |

---

## ■ 動作フロー

```
1. 力の計算（線形合成）
2. 速度更新
3. 空間歪み適用
4. 位置更新
```

---

## ■ 数理対応

```
dS/dt = -M S + F + ξ
```

- F → 線形合成
- M → 空間歪み
