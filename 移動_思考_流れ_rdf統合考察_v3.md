# 移動・思考・流れ：RDF統合考察 v3
*（RDF概念的数理モデルv2.2対応／EFP-F分離・慣性投影・時間差誤差・肉体制御・感情統合）*

---

## ■ コア命題

肉体制御・移動・思考・感情は別の機能ではなく、

**EFPをMが解釈しF(t)を生成し
慣性投影との誤差がHを蓄積しMを更新する
動的ループの異なる表現**

である。

---

## ■ 基本構造

```
F(t)  = interp(M, EFP)       // EFPをMが解釈した作用
E(t)  = F(t+Δ) - M·F(t)     // 慣性誤差
dM/dt = f(M, E, ξ)           // Mは誤差と揺らぎで更新される
```

- EFP：未解釈の作用束（直接観測不能・仮設概念）
- F　：作用解釈F（MがEFPを解釈した結果）
- M　：整合慣性（SILN）
- E　：慣性誤差（慣性投影と実際のFの差）
- ξ　：揺らぎ

この構造はスケールを問わず成立する。

## ■ 4領域の統一

### ● 肉体制御

```
空間　：身体
EFP　：重力・接触・内受容感覚
F(t)　：姿勢・筋状態・張力として解釈
慣性投影：次の身体状態の予測流動
出力　：動き
```

**定義**：
> 身体空間における勾配の流動
> 運動とは慣性投影と実際の身体状態の誤差を処理し続けること

---

### ● 移動

```
空間　：外界
EFP　：地形・食料・危険・他者
F(t)　：引力・反発・コストとして解釈
慣性投影：次の位置・状態の流動
出力　：位置変化
```

**定義**：
> 空間的勾配に沿った流動

---

### ● 思考

```
空間　：概念空間
EFP　：（なし・または外部言語・感覚）
F(t)　：M内部生成　または外部EFP解釈
慣性投影：M·F(t) → 次の概念状態
出力　：次の思考・判断
```

**定義**：
> 慣性投影の内部操作

```
通常ループ（外部）：
EFP → F(t) → M·F(t)慣性投影 → F(t+Δ)と比較 → E → M更新
Δは現実の物理に支配される

思考ループ（内部）：
EFPなしでF(t)を内部生成
M·F(t)慣性投影 → F(t+Δ)_internal → E → M更新
現実コストゼロ・Δを自由に設定できる
```

思考の種類：

```
想像　：ありえないF(t)を意図的に生成して回す
計画　：目標F(t+Δ)から逆算してF(t)を設計する
心配　：最悪のF(t+Δ)を繰り返し慣性投影する
夢　　：Mが半覚醒状態で勝手にループを回す
```

---

### ● 感情

```
空間　：評価空間
EFP　：状況・記憶・身体状態
F(t)　：評価・重み付けとして解釈
出力　：勾配の重み変化・探索制御・行動誘導
```

**定義**：
> 流れに対する評価・重み付け・探索制御機構

感情の分解：

```
感情 = 評価 + 重み付け + 探索制御

評価　　　：E(t)の方向（良い/悪い）
重み付け　：w_i の強度変化（何を引力/反発とするか）
探索制御　：跳躍の促進・抑制（Hの閾値操作）
```

感情とMの関係：

```
感情は外部にある状態ではなく
MがEFPを解釈した際の重み変化として現れる

恐怖　：危険EFPへの重みが急増・反発強化
喜び　：整合成功時の重み安定・慣性強化
怒り　：慣性誤差Eの急増・H蓄積の加速
悲しみ：重要なMの損失・慣性投影の空振り
```

## ■ 思考＝物理移動システムの内部拡張

思考は移動とは別のシステムではなく、
**物理的移動システムが脳内に内部化・拡張されたもの**である。

### ■ 進化的経緯

```
単細胞    → 実際に動いてEFPを探索する
虫・魚    → EFPを先読みし始める（感覚）
哺乳類    → 過去のFをMとして保持する
人間      → F(t)を脳内で生成・改変できる
```

### ■ 内部探索のコスト優位性

```
実際に崖から落ちる　：Δが来てからしかEが発生しない・死ぬ可能性
崖を慣性投影する　　：Δを自由設定・コストゼロ・死なない
```

跳躍のコストを内部化したシステムとも読める。

### ■ 逆転現象

内部生成Fが強くなりすぎると現実のFより優先される

```
M·F(t)_internal >> F(t)_world → 現実が見えなくなる
```

思い込み・妄想・PTSDも同じ構造で読める。
内部慣性投影が外部Fを上書きしている状態である。

## ■ 熱（H）と時間差

```
E(t) = F(t+Δ) - M·F(t)

Δの大きさが慣性誤差の発生タイミングを決める

Δ小：即時誤差・即時H蓄積（鉄球・痛み）
Δ大：遅延誤差・H蓄積が遅れる（キノコ毒・エキノコックス）
Δ∞：誤差が来ない・M更新されない（温暖化・ゆでガエル）
```

- 低熱：安定・収束
- 中熱：揺らぎ・探索
- 高熱：崩壊・跳躍

## ■ ドーパミンによる勾配反転

通常、コスト圧・危険圧は反発として機能する。
ドーパミンはその重み付けを変換する機構である。

```
通常：    F_unknown → 反発圧
介入後：  F_unknown → 引力圧（誤差解決への期待値が上回る）
```

```
ドーパミンなし → 慣性投影に従って流れるだけ
ドーパミンあり → 慣性投影に逆らう選択肢がコスト内に入る
```

> ドーパミンとは
> コスト圧を誤差解決引力で相殺し
> 慣性投影に逆らう流れを可能にする重み変換器である

## ■ 退屈＝慣性誤差フラットへの基層的アラート

```
E(t) ≈ 0 が続く
→ 慣性投影と実際のFがほぼ一致し続ける
→ M更新が起きない・ドーパミン的報酬がない
→ 基層構造が危険信号として仮想熱を発生
→ 探索を促す
```

> 退屈とは
> 慣性誤差フラットを検知した基層構造が
> 自ら熱を生成して探索を強制するシステムである

## ■ 知覚と歪み

```
通常　：F(t) = interp(M, EFP)
歪みあり：F(t) = interp(g(H, A, M), EFP)
```

- H：熱（高いほどinterpが歪む）
- A：注意（EFPの取り込み方が変わる）

幻覚とは内部生成Fが外部Fを上書きする現象である。

```
F_internal >> F_world
```

## ■ 動的ループ

```
EFP → F(t) → 慣性投影 → E(t) → H → 跳躍 → M更新 → 新たなF
```

Mが変われば同じEFPから異なるFが生まれ
異なる地形が形成される。
