# 水のSILN係数算出フロー

## 0. 目的

水を基準物理SILNとして扱い、RDFにおける相転移・拘束・散逸・振動の参照係数を抽出する。

これは「水の真の本質」を定義する操作ではなく、RDFを物理構造へ接地するための局所近似である。

---

## 1. 基本ラベル設定

```text
対象SILN：水
スケール：人間スケール・地球環境
標準外部拘束：P = 1 atm
基準相：固体・液体・気体
```

RDF展開：

```text
S_water：水の状態
M_water：水分子・水素結合ネットワークの整合慣性
H_water：構造振動強度
P：外部拘束的素流圧
ξ_water：分子揺らぎ・自由度
θ：相転移閾値
A_water：散逸係数・散逸行列
ρ_water：相転移後の熱残留率
```

---

## 2. 標準基準点の設置

水SILNの最初の基準点として、1気圧における融点・沸点を置く。

```text
θ_melt^water(P=1atm) = 0℃
θ_boil^water(P=1atm) = 100℃
```

RDF解釈：

```text
1気圧0℃
= 標準外部拘束下における固体水Mと液体水Mの境界

1気圧100℃
= 標準外部拘束下における液体水Mと気体水Mの境界
```

---

## 3. 温度のRDF翻訳

温度は、構造振動を観測可能なスカラーへ圧縮した量として扱う。

```text
T ≈ scalar(H_water)
H_vec ≡ δM_water
```

圧縮表現：

```text
温度 = 水分子ネットワークの振動強度の統計的スカラー化
```

したがって：

```text
T < 0℃       → 固体M優勢
0℃ < T < 100℃ → 液体M優勢
T > 100℃      → 気体M優勢
```

ただしこれは `P = 1 atm` の条件下に限る。

---

## 4. 圧力のRDF翻訳

圧力は、外部から水構造を拘束する素流圧として扱う。

```text
P ≈ density(F_constraint)
```

圧縮表現：

```text
圧力 = 外部から構造を押し込む拘束的素流圧の密度
```

温度と圧力の役割：

```text
温度H：内側からMを揺らす
圧力P：外側からMを拘束する
```

---

## 5. 相状態の基本判定

暫定的に、相の解放度を次のように置く。

```text
Φ_water ≈ H_water / (M_water + P)
```

解釈：

```text
Φが低い  → 固体
Φが中程度 → 液体
Φが高い  → 気体
Φが極高  → 分子構造崩壊・プラズマ的状態
```

より慎重には、実際の相状態は次の関数として扱う。

```text
Phase_water = f(H_water, M_water, P, ξ_water)
```

---

## 6. 圧力依存閾値の導入

相転移閾値は固定値ではなく、圧力Pの関数として扱う。

```text
θ_melt^water(P)
θ_boil^water(P)
```

基準点：

```text
θ_melt^water(1atm) = 0℃
θ_boil^water(1atm) = 100℃
```

圧力変化：

```text
P < 1 atm → θ_boil(P) < 100℃
P > 1 atm → θ_boil(P) > 100℃
```

RDF解釈：

```text
低圧：外部拘束Pが弱く、液体Mから気体Mへ抜けやすい
高圧：外部拘束Pが強く、液体Mが維持されやすい
```

水の融点側は特殊性を持つ：

```text
P ↑ → θ_melt(P) はわずかに低下
```

RDF解釈：

```text
氷Mは液体水Mより体積が大きいため、圧力Pが固体Mを不利にする
```

---

## 7. 相転移のRDF表現

### 融解

```text
H_water(T) > θ_melt(P)
→ 固体M → 液体M
```

意味：

```text
氷の格子Mが崩れ、水素結合ネットワークが流動化する
```

### 沸騰

```text
H_water(T) > θ_boil(P)
→ 液体M → 気体M
```

意味：

```text
液体水としての接続Mを維持できなくなり、分子がネットワークから離脱する
```

### 凝固

```text
H_water(T) < θ_melt(P)
→ 液体M → 固体M
```

意味：

```text
振動が不足し、格子状Mが再形成される
```

### 凝縮

```text
H_water(T) < θ_boil(P)
→ 気体M → 液体M
```

意味：

```text
自由化していた水分子が再び相互整合し、液体ネットワークを形成する
```

---

## 8. 潜熱の扱い

相転移中は温度が大きく変化しないまま、エネルギーが吸収・放出される。

RDFでは、これは次のように読む。

```text
潜熱 = Hが温度上昇ではなくM再編へ吸収される量
```

融解・沸騰時：

```text
入力H → Mの拘束解除・再編に使用
```

凝固・凝縮時：

```text
Mの再拘束 → Hとして放出
```

---

## 9. 抽出したいSILN係数

### 9.1 相転移閾値

```text
θ_melt^water(P)
θ_boil^water(P)
θ_sublime^water(P)
```

### 9.2 拘束係数

```text
M_solid_water
M_liquid_water
M_gas_water
```

意味：

```text
各相における水分子ネットワークの整合慣性
```

### 9.3 散逸係数

```text
A_water
```

候補対応：

```text
比熱
熱伝導率
熱拡散率
粘性
```

### 9.4 熱残留率

```text
ρ_water
```

意味：

```text
相転移後にどれだけHが残るか
```

### 9.5 自由度係数

```text
D[ξ_water]
```

意味：

```text
水分子ネットワーク内で許容される揺らぎ・自由度
```

関係保存則：

```text
||M_water|| · D[ξ_water] = 𝒦_water
```

---

## 10. 算出手順

### Step 1：基準点を置く

```text
P = 1 atm
θ_melt = 0℃
θ_boil = 100℃
```

### Step 2：圧力を変えた相転移点を集める

```text
(P_i, θ_melt_i)
(P_i, θ_boil_i)
```

### Step 3：閾値関数を近似する

```text
θ_melt(P)
θ_boil(P)
```

### Step 4：各相の安定範囲を定義する

```text
Solid_range(P)
Liquid_range(P)
Gas_range(P)
```

### Step 5：潜熱からM再編コストを推定する

```text
L_melt → 固体Mから液体Mへの再編コスト
L_boil → 液体Mから気体Mへの再編コスト
```

### Step 6：比熱・熱伝導率・粘性からAを推定する

```text
A_water ≈ 散逸・伝達・変形抵抗の合成近似
```

### Step 7：密度・粘性・表面張力からMの拘束性を推定する

```text
M_water ≈ f(密度, 粘性, 表面張力, 圧縮率)
```

### Step 8：相ごとのMとD[ξ]を比較する

```text
固体：M大、D[ξ]小
液体：M中、D[ξ]中
気体：M小、D[ξ]大
```

---

## 11. 暫定まとめ式

```text
Phase_water = f(H_water, P, M_water, ξ_water)
```

```text
H_water(T) > θ_phase(P)
→ M_water が相転移する
```

```text
||M_water|| · D[ξ_water] = 𝒦_water
```

---

## 12. 一文圧縮

水のSILN係数算出とは、1気圧0℃・100℃を基準点として、水分子ネットワークMが温度Hと圧力Pによってどの条件で固体・液体・気体へ相転移するかを、閾値θ・散逸A・拘束M・自由度ξとして抽出する操作である。
