センサー系
視覚（ほぼ360度・変化量感度）

ウサギ側は全周検知、識別なし、変化量に比例した素流圧
ハンター側は前方コーン、精度高め、継続追跡
両者非対称な設計

音（全方位・遮蔽なし）

現状維持ベースで視覚と明示的に分離
検知した場合の恐怖熱の上がり方を視覚と差別化

匂い（時間遅延・履歴）

糞として空間に痕跡を残す
恐怖フェロモン混入で経験が環境に書き込まれる
個体が死んでも情報が残る外部記憶


行動系
フリーズ状態

音検知→止まって情報収集
視覚確定→逃走判断の段階的閾値
恐怖熱を蓄積しつつ速度抑制
アンカーとは目的が違う情報収集のための停止

緊張構造

両者静止で睨み合い
熱が先に閾値超えた方が動く


経験M系
個体内M更新

同じ草パッチで整合が繰り返される→引力重みが積み上がる
ハンター遭遇経験→恐怖感度の恒常的変化
個体ごとに異なる勾配場の見え方が生まれる

環境への書き込み

糞による外部記憶
個体Mではなく空間側に履歴を刻むアプローチ
社会的SILNの原始的な形


実装順序案
1. 視覚・音センサーの明示的分離
2. フリーズ状態 + 変化量感度
3. 糞（匂いセンサー）
4. 経験Mの更新則
